Reconciliation: all-rental alternative would have been $1,040 (Look 2 $500 + Look 3 plain rental $540); all-build alternative would have been $2,337 (Look 2 build $1,201 + Look 3 build $1,136). The mixed decision costs $853 more than all-rental and $444 less than all-build — the split is driven by which garment actually failed the timed test, not by an average price comparison.

Rule of thumb applied here: any change under roughly 20 seconds gets a minimum of three timed runs with the actual assigned dresser before the source decision (build vs. rent) is locked — a single run can be an outlier in either direction.
