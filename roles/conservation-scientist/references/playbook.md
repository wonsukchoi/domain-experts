# Conservation Scientist — Playbook

## Stocking-rate worksheet (filled example)

| Step | Input | Value |
|---|---|---|
| 1. Pasture acreage | Surveyed/deeded | 640 ac |
| 2. Current-year forage production | Clipped, 3-transect double-sample, air-dry | 1,200 lbs/ac |
| 3. Range condition class (from ESD comparison) | Mid-seral, fair-good | — |
| 4. Proper-use factor (condition-adjusted) | Per SRM/NRCS guidance for mid-seral | 45% |
| 5. Available forage/acre | (2) × (4) | 540 lbs/ac |
| 6. Total available forage | (5) × (1) | 345,600 lbs |
| 7. AUM forage demand (1,000-lb AU) | NRCS standard, ~2.6% body wt/day × 30 days | 780 lbs/AUM |
| 8. Season capacity (AUMs) | (6) ÷ (7) | 443.1 AUMs |
| 9. Proposed/actual use (AUMs) | Herd size × AU factor × months | 780 AUMs |
| 10. Utilization ratio | (9) ÷ (8) | 1.76 (176% of capacity) |

**Animal-unit conversion factors (common reference set, verify against current NRCS table):**

| Class | AU factor |
|---|---|
| Mature cow, no calf | 1.00 |
| Cow with calf < 6 mo | 1.30 |
| Bull | 1.35 |
| Yearling cattle | 0.70 |
| Mature horse | 1.25–2.00 (verify per breed/weight) |
| Ewe with lamb | 0.20 |
| Doe with kid | 0.15 |

If the utilization ratio exceeds 1.0, present both remediation paths (herd reduction, season shortening) with the resulting AUM math for each — don't pick one unilaterally without the lessee's operational constraints.

## Prescribed-fire prescription window (filled example)

| Parameter | Prescribed range | Field reading (day-of) | In prescription? |
|---|---|---|---|
| 10-hr fuel moisture | 8–12% | 9.5% | Yes |
| Relative humidity | 30–55% | 42% | Yes |
| Wind speed (20 ft) | 5–15 mph | 11 mph | Yes |
| Wind direction | SW–W (away from residential boundary) | WSW | Yes |
| Mixing height | > 1,700 ft | 2,100 ft | Yes |
| Air temperature | 45–75°F | 61°F | Yes |
| Days since > 0.25 in. rain | 3–10 days | 5 days | Yes |

**Go/no-go rule:** every parameter must fall inside its prescribed range simultaneously. One out-of-range parameter (commonly wind direction toward a smoke-sensitive receptor, or fuel moisture below 6% risking control loss) is a no-go regardless of how favorable the others read — there is no partial-credit average across parameters.

**Season-of-burn effect note:** the same fuel type burned in dormant-season (late winter) vs. growing-season (early summer) produces different top-kill rates on woody encroachment — growing-season burns generally set back resprouting brush harder, at the cost of higher control risk from more volatile fuel moisture. State which season the objective requires before setting the calendar window.

## Multi-use tradeoff matrix (filled example — 3,200-acre mixed-use allotment)

| Allocation option | Grazing AUMs/yr | Allowable timber cut (bd ft/yr) | Elk winter-range habitat quality (1–5 scale) | Recreation trail-days/yr capacity |
|---|---|---|---|---|
| A: Grazing-priority | 2,850 | 180,000 | 2 | 4,500 |
| B: Balanced (recommended) | 2,100 | 140,000 | 3 | 6,000 |
| C: Habitat-priority | 1,400 | 90,000 | 4 | 6,000 |
| D: Timber-priority | 1,900 | 240,000 | 2 | 3,800 |

Present the full matrix to all stakeholder groups in one meeting, not sequential one-on-one negotiations — sequential negotiation lets each party assume it's giving up less than it is relative to what the others received. State explicitly which option is recommended and why (here, Option B trades roughly 26% of the grazing-priority AUM level for a habitat-quality gain that keeps the allotment off a state sensitive-species watch list, while retaining recreation capacity at its ceiling).
