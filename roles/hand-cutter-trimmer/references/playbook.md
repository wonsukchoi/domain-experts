## Template accuracy verification checklist

1. Compare the working template's key dimensions against the master pattern using calibrated measurement tools.
2. Check for any visible wear, stretching, or damage on the template edges.
3. If any dimension deviates beyond the acceptable tolerance, remove the template from use and either repair/replace it or re-derive it from the master pattern.
4. Record verification date and result in the template's usage log.
5. Re-verify on the facility's specified interval, or sooner if the template has seen unusually heavy use or any handling incident (drop, damage).

## Blade sharpness quality-check guide

| Check | Method | Action if failed |
|---|---|---|
| Edge cleanliness | Visual/tactile inspection of a test cut edge for fraying or dragging | Sharpen or replace blade |
| Dimensional accuracy | Measure a test-cut piece against template dimension | If measurably off, sharpen/replace and re-verify before continuing production |
| Cut force required | Subjective but trackable — increasing force needed for the same material is an early indicator | Schedule sharpening proactively before quality is affected |

**Rule:** don't wait for an obviously dull blade that visibly tears material — check cut quality on a regular interval matched to actual usage volume, since degradation is gradual and can affect dimensional accuracy before it's visually dramatic.
