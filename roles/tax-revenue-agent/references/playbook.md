# Tax Examiner / Revenue Agent — Playbook

## Statute of limitations trigger checklist

| Scenario | Applicable period |
|---|---|
| Return filed, unreported income ≤ 25% of reported gross income | Standard 3 years from filing |
| Return filed, unreported income > 25% of reported gross income | Extended 6 years from filing |
| Fraudulent return | Unlimited |
| No return filed | Unlimited (period never begins to run) |

**Use:** Always calculate the actual ratio of unreported income to reported gross income before selecting a period — don't assume the standard 3-year period without checking, and don't assume the 6-year period applies without confirming the 25% threshold is actually crossed.

## Indirect income reconstruction worksheet (filled example)

| Component | Value |
|---|---|
| Total bank deposits for the year | $230,000 |
| Less: identified non-taxable deposits (loan proceeds, gift) | −$20,000 |
| Identified business-related deposits | $210,000 |
| Reported gross receipts on return | $180,000 |
| **Unreported income** | $210,000 − $180,000 = **$30,000** |
| Reported gross income (for SOL threshold check) | $185,000 |
| 25% threshold | $185,000 × 25% = $46,250 |
| **SOL determination** | $30,000 < $46,250 → **standard 3-year period applies** |

## Cohan estimate derivation (filled example)

| Component | Value |
|---|---|
| Claimed deduction (no mileage log) | $15,000 |
| Substantiating evidence available | Fuel receipts, comparable industry mileage data for similar contractors |
| **Independently derived reasonable estimate** | **$9,000 allowed** |
| **Disallowed amount** | $15,000 − $9,000 = **$6,000** |

**Use:** Never resolve a Cohan situation as "$0 allowed" (full disallowance) or "full amount allowed as claimed" without independent derivation — the estimate should be bounded by whatever objective evidence (partial receipts, comparable data) is actually available.

## Deficiency and penalty calculation (filled example)

| Component | Value |
|---|---|
| Unreported income | $30,000 |
| Disallowed deductions | $6,000 |
| **Total additional taxable income** | $36,000 |
| Marginal tax rate | 24% |
| **Additional tax (deficiency)** | $36,000 × 24% = **$8,640** |
| Substantial understatement threshold check | Greater of $5,000 or 10% of correct tax — $36,000 exceeds this |
| Badges of fraud present? | No — cooperative taxpayer, consistent explanations, no concealment |
| **Applicable penalty** | Accuracy-related, 20% |
| **Penalty amount** | $8,640 × 20% = **$1,728** |
| **Total proposed assessment** | $8,640 + $1,728 = **$10,368** |

## Badges of fraud checklist (for penalty-type determination)

1. Concealment of income or assets (e.g., unreported accounts, false statements to examiner)
2. Destroyed, altered, or deliberately withheld records
3. Consistent pattern of substantial underreporting across multiple years
4. Implausible or repeatedly shifting explanations for discrepancies
5. Use of false documents or fictitious deductions

**Use:** If none of these are present, the case should proceed as an accuracy-related (negligence/substantial understatement) matter, not civil fraud — the evidentiary bar for fraud (clear and convincing) is meaningfully higher and shouldn't be pursued absent specific indicators like these.
