# Curator artifacts

Filled templates for the four documents a curator produces most often: acquisition proposal, deaccession recommendation, condition report, and loan facility-report checklist, plus label-copy examples at the correct length.

## 1. Acquisition proposal memo

```
TO: Acquisitions Committee
FROM: [Curator], [Department]
RE: Proposed acquisition — [Object/Collection name]

Object(s): [count, medium, maker, date]
Source: [donor name / dealer / auction house]
Method: [gift / purchase / bequest]
Appraised or asking value: $[X] ([appraiser/source], dated [date])

Collecting-plan fit: [which documented gap/strength this fills; cite the plan section]
Comparable holdings: [what's already in the collection this duplicates or complements]
Condition: [conservator's summary — see condition report if attached]
Provenance summary: [ownership chain, any 1933–1945 continental-Europe gap flagged
  and portal status, any Indigenous-cultural-property screening result]

Total cost of ownership (15-year horizon):
  Cataloging/rehousing (one-time): $[per object] x [count] = $[subtotal]
  Storage/insurance (annual): $[per object/year] x [count] x 15 = $[subtotal]
  Projected conservation: $[estimate, if known]
  TOTAL: $[sum]

Attached conditions: [naming / permanent display / resale restriction — none, or
  specify and state the negotiated counter-terms]

Recommendation: [Accept in full / Accept partial — specify subset / Decline]
Rationale: [1–3 sentences tying fit + cost to the recommendation, not the
  appraised value alone]
```

## 2. Deaccession recommendation memo

```
TO: Collections Committee
FROM: [Curator]
RE: Deaccession recommendation — [Object], accession #[XXXX]

Reason: [duplicate of accession #YYYY / outside current collecting scope /
  condition beyond feasible conservation / donor-restriction conflict]
Clear title confirmed: [Y/N — deed of gift or purchase record on file]
Restrictions on file: [none / specify — must be honored in disposal method]

Disposal preference order applied:
  1. Transfer to [named public institution] — [pursued: Y/N, outcome]
  2. Public auction — [house, estimate range, if reached this step]
  3. Private sale — [only if 1 and 2 documented as not viable; buyer, price]

Proceeds use, if sale: proceeds will be restricted to [conservation / storage
  materials / other direct collections-care line item] per AAMD "direct care"
  policy; will NOT fund [salaries / exhibition costs / operating budget].

Committee vote required before any object leaves custody.
```

## 3. Condition report

```
Accession #: [XXXX]         Object: [title, medium, date]
Examined by: [conservator/registrar]     Date: [YYYY-MM-DD]
Purpose: [incoming loan / outgoing loan / periodic survey / pre-treatment]

Overall condition grade: [Excellent / Good / Fair / Poor / Critical]

Structural: [e.g., "stretcher bars sound, no warping"]
Surface: [e.g., "stable craquelure overall; 2cm area of flaking paint, upper
  left quadrant, marked on diagram"]
Prior repairs/interventions: [documented or observed]
Active deterioration flagged: [none / specify — e.g., "active flaking,
  isolate from vibration and light exposure pending conservator review"]

Diagram: [attach marked photograph/line drawing with numbered condition notes]

Recommendation: [safe to travel as-is / requires stabilization before travel /
  requires immediate conservation attention regardless of loan status]
```

**Grading scale in use** (adapt to institutional standard, e.g. Canadian Conservation Institute or in-house equivalent):
- **Excellent** — no condition issues affecting display or handling.
- **Good** — minor, stable issues noted; no action required before display.
- **Fair** — stable but visible issues; monitor, no urgent action.
- **Poor** — active or progressing issues; conservation review required before further handling or loan.
- **Critical** — active deterioration; isolate immediately regardless of queue position.

## 4. Loan facility-report checklist (outgoing or incoming)

```
[ ] Facility report received and reviewed (climate control, security, fire
    suppression, staffing, exhibit case specifications)
[ ] Insurance certificate or indemnity commitment on file, fixed value: $[X]
[ ] Courier required: [Y/N] — if Y, courier fee and travel terms confirmed
[ ] Condition report completed and signed by both parties before object ships
[ ] Loan agreement signed, specifying: loan period, credit line wording,
    photography/reproduction rights, insurance responsibility, return
    shipping terms, and any display-condition restrictions (light levels,
    case type, minimum security)
[ ] Object return date confirmed and calendared; renewal requires a new
    signed agreement, not an email extension
```

## 5. Label copy — length and register

**Object label (75-word ceiling, per Serrell benchmark):**

> **Rising Star Quilt**
> Mary Higgins Thompson, c. 1890. Cotton, hand-pieced and quilted.
> Thompson pieced this pattern from her daughters' outgrown dresses — the
> mismatched blues and reds were fabric she had on hand, not a chosen
> palette. Quilts like this one were made for use, not display; the wear
> along the binding shows decades on a bed, not in a frame.
> Gift of the Thompson family, 2019. 1890.4.2

(62 words — under the ceiling; leads with one retained idea: the fabric was leftover, not selected, which reframes "mismatched" as evidence rather than flaw.)

**Orientation panel (for an exhibition's central argument — no fixed ceiling, but built around one thesis sentence, not a survey):**

> **Pieced from What Was on Hand**
> The forty quilts in this gallery were not made from fabric bought for the
> purpose. Every one was pieced from clothing, feed sacks, or dress
> remnants already in the house. That constraint — using what existed
> rather than choosing what was ideal — produced the visual language this
> exhibition traces: asymmetry read not as error but as record of a
> household's actual materials.
