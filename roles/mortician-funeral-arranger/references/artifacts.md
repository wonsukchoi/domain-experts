## Death-certificate worksheet — verification checklist

| Field | Source to verify against | Verified? |
|---|---|---|
| Full legal name (incl. spelling) | Driver's license / state ID / passport | ☐ |
| Date of birth | Driver's license / birth certificate / SSN card | ☐ |
| Social Security number | SSN card or documented from a prior tax/benefits record | ☐ |
| Place of birth (city, state/country) | Birth certificate if available; family verbal if not | ☐ |
| Marital status at death | Family verbal, cross-check against will/estate documents if discrepancy | ☐ |
| Informant name/relationship | Confirmed against the person present and authorized | ☐ |
| Cause/manner of death | Physician/medical examiner pronouncement — never family-supplied | ☐ |

**Rule:** every row must show a primary-source verification, not just what was said in the room — a field with no checkmark isn't filed until it's checked.

**Reconciliation rule:** the total on this statement must match the sum of line items exactly, and every selected item must trace back to something the family actually confirmed during the conference — not a default or assumed selection.
