## PD plan tied to a named practice

| Element | Entry |
|---|---|
| Named practice | Writing and scoring a DOK-3 (non-routine, multi-step) task for a Geometry standard |
| Teachers assigned | 4 (grade 7 team, sections A–D) |
| Session 1 | Model task-writing against Webb's DOK criteria; each teacher drafts one task for an assigned standard (7.G.1, 7.G.4, or 7.G.6) |
| Session 2 | Peer review of drafted tasks against a DOK-3 rubric; revise before use |
| Walkthrough checkpoint | Week 3 of fall unit — observer confirms the drafted task (or an equivalent) is in use, not just that sessions were attended |
| Practice-implementation rate (the metric that matters) | # teachers observed using a DOK-3 task at checkpoint / 4 assigned |
| PD-attendance rate (the metric that doesn't, alone) | # teachers who attended both sessions / 4 assigned |

Reporting attendance rate alone (e.g., "4/4 attended, 100%") without the walkthrough-confirmed implementation rate is the single most common way a PD plan looks successful and changes nothing in the classroom.
