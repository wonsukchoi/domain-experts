## Fee proposal (percentage-of-construction-cost basis)

**Project:** 18,000 SF mixed-use building, $6.54M construction cost (post-reconciliation).

| Phase | % of total fee | Basis |
|---|---|---|
| Schematic Design (SD) | 15% | Deliverable: concept plans, massing, code/zoning summary |
| Design Development (DD) | 25% | Deliverable: coordinated consultant sets, outline specs, cost reconciliation |
| Construction Documents (CD) | 40% | Deliverable: permit set, full specifications |
| Bidding/Negotiation | 5% | Deliverable: addenda, bid analysis |
| Construction Administration (CA) | 15% | Deliverable: RFI/submittal review, site visits (biweekly), punch list |

Total fee basis: 8% of $6.54M = **$523,200**. Reimbursables (printing, travel, renderings beyond 2 included) billed separately, not absorbed into the percentage.

**Fallback ladder if owner pushes for lump sum before scope is set:** (1) percentage-of-construction-cost with a not-to-exceed cap once DD-level scope is fixed → (2) hourly not-to-exceed for SD only, percentage for DD forward → (3) decline the lump-sum-before-scope request and quote SD as a standalone fixed-fee study phase (typically $35,000-55,000 for a project this size) to define scope before committing to a total fee.

>

Construction Type V-B, sprinklered (NFPA 13). >

Required egress width 12 in (0.3 in/occupant, stairways); provided 88 in (two 44-in stairs) — 7.3× the code minimum, no redesign required. Floor 1 (Group M) occupant load at 60 SF/occupant gross = 100 occupants; required width 30 in; provided 88 in — passes with 2.9× margin.

## Change order (excerpt, cost/schedule impact)

>

> **Schedule impact:** 4 additional working days for excavation/inspection re-sequencing at this column line; float analysis shows this column line was not on the critical path (6 days of float remaining) — **net schedule impact: 0 days to substantial completion.**
>
