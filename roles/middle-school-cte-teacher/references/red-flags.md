# Red flags — what an experienced middle school CTE teacher notices instantly

Smell tests with thresholds. Any one can be innocent; two or three together is a pattern worth acting on.

### A new rotation cohort about to start hands-on machine work with no equipment-specific safety-test record for this roster

Usually means: the prior cohort's signed-off test or the building's fall orientation is being assumed to transfer to a different group of students.
First question: "Does this specific roster have a dated, equipment-specific safety-test record for these specific machines?"
Data to pull: the safety-test log filtered by this cohort's roster and this cycle's machine list.

### A parent or counselor conversation implying the wheel "starts" a pathway or concentrator count

Usually means: course participation is being conflated with Perkins V concentrator credit, which only begins at a high-school-credit-bearing CTE course.
First question: "Which specific high school course code, if any, are we asking whether this counts toward?"
Data to pull: the district's Perkins V articulation/course-equivalency list.

### A 6-9 week rotation unit plan listing 4 or more sub-objectives

Usually means: the unit was scaled down from the year-long high school course's scope instead of redesigned around 1-2 achievable sub-objectives sized to the actual periods available.
First question: "Which single deliverable proves these sub-objectives, and does the period count remaining after the safety day actually support building it?"
Data to pull: periods remaining after the safety day(s); a prior cohort's actual completion time for the same or a comparable deliverable.

### Career-interest inventory answers recorded in the gradebook as a scored assignment

Usually means: the interest signal is being treated as skill-mastery data, which gives students a reason to answer strategically instead of honestly.
First question: "Is this graded for completion only, or for the content of the answers?"
Data to pull: the gradebook category weighting attached to the inventory.

### A student assigned to a high-prestige CTSO competitive event with no rotation record of the underlying skill

Usually means: roster-filling took priority over skill match, setting the student up for a public poor showing.
First question: "What rotation artifact or station log shows this student has actually practiced this event's specific skill?"
Data to pull: the student's rotation history for this cycle and the event's published skill requirements.

### Two consecutive rotation cohorts using the identical tool-checkout/workstation-pairing chart, unmodified

Usually means: the chart is a copy-paste rather than a rebuild against the new roster's body sizes, handedness mix, and prior-exposure spread.
First question: "When was this chart last regenerated, and against which roster?"
Data to pull: the chart's revision timestamp versus the current cohort's start date.

### A safety near-miss with no same-day log entry tagged to station and cohort

Usually means: the incident risks being forgotten, or repeated at the same station with the next cohort because root cause was never checked.
First question: "Has this exact station produced an incident with any prior cohort this year?"
Data to pull: the per-station incident log across all cohorts this year, not just the current one.

### A rotation's culminating deliverable can't realistically be finished inside the periods remaining after the safety day(s)

Usually means: the unit plan under-budgeted for the mandatory safety certification time at the start of the block.
First question: "How many periods remain after safety certification, and what did a prior cohort's actual build time look like for this deliverable?"
Data to pull: the block's total period count, the safety-day count, and a prior cohort's completion-time record.

### A district or administrator request to shorten or skip the day-one safety pass-off "to save time" in an already-short block

Usually means: schedule pressure is being prioritized over equipment-specific certification, which is riskiest precisely in the rotation model where no prior familiarity can be assumed.
First question: "What specifically would replace the equipment-specific pass-off, and does it hold the same 100%-mastery bar?"
Data to pull: the district's shop-safety policy minimum certification requirement.

### A single rotation cohort with a wide readiness spread on Day 1 (some students have prior equipment exposure, most don't)

Usually means: default table-order station groupings will pair two first-time students on the highest-risk equipment unless the chart is deliberately redistributed.
First question: "Who in this cohort has documented prior equipment exposure, and are they spread across stations rather than clustered together?"
Data to pull: prior-elective or prior-rotation history for this specific roster.

### A CTSO chapter roster built primarily from which students volunteered, with no skill cross-check

Usually means: enthusiasm is being used as the sole selection filter, which produces mismatches the event-matching worksheet exists to catch.
First question: "For each volunteer, does the rotation record show the specific skill this event tests?"
Data to pull: the CTSO event-matching worksheet for the current roster.
