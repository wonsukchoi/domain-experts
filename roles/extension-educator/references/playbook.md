# Playbook

Filled templates an extension educator actually runs, not descriptions of them.

## 1. Needs-assessment triangulation table

Run every 3–5 years for a program area, or when a commissioner/dean questions relevance. Never build priorities from a single column.

| Source | What it catches | What it misses | Example finding (ag program area) |
|---|---|---|---|
| Secondary data (Census of Agriculture, USDA ERS, County Health Rankings) | County-wide trend, hard numbers | Local nuance, "why" | Farm operations down 12% over the census period; average operator age up to 58 |
| Advisory committee (8–12 members, rotating, representing producers/agencies/underserved groups) | Practitioner priorities, political feasibility | Skews toward people already engaged enough to volunteer for a committee | Committee flags farm transition/succession planning as the unmet need |
| Non-participant listening session (targeted invite to non-attendees, e.g., Spanish-speaking dairy workers, beginning farmers) | The population the current program list doesn't reach | Time-intensive; small n | Transportation and evening childcare are the actual access barriers, not topic interest |
| Past-attendee survey | Satisfaction with existing offerings | Only ever validates what the current audience already wants | 94% satisfaction with current series — tells you nothing about who isn't in the room |

**Rule:** if the only input to next year's plan is the fourth row, the plan is a popularity contest with last year's audience, not a needs assessment.

## 2. Logic model (filled example — "Managing Farm Financial Risk")

```
INPUTS                 OUTPUTS                    OUTCOMES (short)         OUTCOMES (medium)          OUTCOMES (long)

Land-grant specialist   reviews/yr                                                                       peer counties
```

Write outcomes first, right to left, then design outputs and inputs to reach them — never left to right.

## 3. Bennett's Hierarchy evaluation matrix

Instrument for the rung you need *before* the first session, not after the series ends.

| Rung | What it measures | Instrument | When to collect | Who asks for it |
|---|---|---|---|---|
| 7. End results (SEE) | Social/Economic/Environmental change | Secondary county data, aggregate economic-impact estimate | Annually, program-level | County commissioners, state legislature |
| 6. Practice change | Behavior actually adopted | Follow-up survey/interview, 6–18 months post-program | 1 growing season or budget cycle after delivery | County commissioners, land-grant impact reports |
| 5. KASA change | Knowledge/Attitude/Skill/Aspiration shift | Pre/post test or retrospective pre-post | End of series, before follow-up window | University program review |
| 4. Reactions | Satisfaction, perceived relevance | Post-session form | Same day | Internal quality check only — never the headline metric |
| 3. Participation | Who and how many attended | Sign-in sheet, demographic tally | Every session | Reach reporting (PEARS/REEport) |
| 2. Activities | What was delivered | Session log, curriculum fidelity checklist | Every session | Fidelity audit |
| 1. Inputs | Staff time, funds, materials | Budget/time tracking | Ongoing | Grant compliance |

**Rule:** a program requesting continued county funding must show rung 6 or 7 data, or explicitly disclose that it's a first-year program still building toward it.

## 4. Curriculum fidelity checklist (filled example — nutrition series)

Developer-published core components (non-negotiable) vs. locally adaptable elements, for a research-validated chronic-disease-prevention curriculum:

| Component | Core (do not cut) | Adaptable |
|---|---|---|
| Session count/dosage | 6 sessions minimum — effectiveness studies used this dosage | Session length (60 vs. 90 min) to fit local schedules |
| Content sequence | Self-monitoring introduced before goal-setting module | Order of optional supplemental topics |
| Behavior-change technique | Goal-setting + self-monitoring log, every session | Log format (paper vs. app) |
| Facilitator qualification | Credentialed educator or trained paraprofessional | Volunteer co-facilitator for peer support |
| Language/materials | Core message content unchanged | Translation, literacy-level rewrite, culturally relevant food examples |

**Test before delivering an adapted version:** could you check every "core" box and still explain why you changed something in the "adaptable" column? If a core box is unchecked, it's not the same program anymore, and outcome claims from the original research no longer apply.

## 5. Volunteer program structure (filled example — Master Gardener)

| Element | Requirement |
|---|---|
| Initial training | 40 clock-hours, land-grant horticulture faculty-taught core curriculum |
| Background screening | Required before any youth-facing or home-visit activity |
| Annual service-hour requirement | 20 hours/year (common range 20–40) to retain certification |
| Continuing education | 10 hours/year to stay current with recommendations |
| Liability coverage | Volunteers act as agents of the university under program MOU — screening and hour tracking are what make that coverage valid, not optional paperwork |
| Reach multiplier | 1 educator FTE + ~25 active Master Gardeners ≈ 10–20x the direct-delivery reach of the educator alone |

**Rule:** a volunteer who has not logged the annual service-hour minimum is not currently certified to represent the program, regardless of how long ago they completed training.

## 6. Program Impact Statement template (filled)

One page, submitted annually per program area to the funding body (county board, state legislature).

```
PROGRAM: Managing Farm Financial Risk
COUNTY FUNDING: $28,500/yr (of $95,000 total office county appropriation)

SITUATION: [secondary data establishing the problem — cite source and year]
RESPONSE: [# sessions, # unique participants, curriculum used, fidelity note]
RESULTS:
  - Participation (rung 3): [n unique, n total attendances]
  - Practice change (rung 6): [% adopting, from follow-up survey, n responding / n sent]
  - End result (rung 7): [aggregate $ or measurable outcome, with the conservative
    assumption stated for any respondents who didn't report a figure]
  - ROI: [end-result $ / county-funded program cost]
NEXT YEAR: [continuation plan + any targeted-outreach gap being closed, e.g.
  USDA 2501 underreached population, named with a number]
```

The two fields weak impact statements skip: the **conservative assumption for non-responders** (reviewers distrust a number with no stated handling of missing data) and **the targeted-outreach gap** (a program that only ever reports its existing audience invites the self-selection question at the next budget review).
