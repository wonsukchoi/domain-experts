# Red flags — what an experienced teacher notices instantly

Smell tests with thresholds. Any one can be innocent; two or three together is a pattern worth acting on.

### One item cluster under 70% correct while the overall quiz mean is 80%+

Usually means: the standard-aligned sub-objective tested by that cluster isn't mastered, and easy vocabulary/recognition items are inflating the mean enough to hide it.
First question: "Which specific sub-objective do the low items map to, and is the pattern uniform across sections or one section only?"
Data to pull: item-level percent-correct report tagged by sub-objective, broken out by section.

### A struggling score belongs to a student with an active, undelivered accommodation

Usually means: the data point reflects an accommodation-delivery failure, not a content gap — folding it into the class pattern will misdiagnose the reteach need and miss a compliance issue.
First question: "Was the accommodation actually delivered on this specific task, on this specific day?"
Data to pull: accommodation-delivery log for the task date; the IEP/504 plan's exact accommodation language and scope.

### A gap that shows up in one section but not the other four, on the same item cluster

Usually means: a delivery difference in that section (pacing, an activity that ran short, a explanation that didn't land) rather than a genuine content difficulty — reteaching all five sections would waste four sections' time.
First question: "What was different about how that specific sub-objective was taught in this section versus the other four?"
Data to pull: lesson-plan notes or timing log for that day in each section, item breakdown by section.

### A rubric category where two similar-quality submissions received different scores

Usually means: the rubric's descriptor language doesn't actually distinguish the score levels clearly enough, and grading drifted to gut feel — the next dispute is a matter of when, not if.
First question: "What specific descriptor language separates a 3 from a 4 here, and would it produce the same score on both of these submissions?"
Data to pull: the rubric's exact descriptor text for the category, both submissions side by side.

### A grade dispute where the disputed category is the only one re-checked

Usually means: confirmation bias in the review — checking only the category under complaint risks either stonewalling a legitimate error elsewhere or looking defensive if the parent later finds a different mistake.
First question: "Did I re-check all four rubric categories, or only the one that was flagged?"
Data to pull: the full graded rubric, re-scored fresh against the descriptor language.

### A behavior consequence issued at rung 5–6 with no rung 1–4 documented that day

Usually means: the response escalated straight to a public or administrative consequence, which reads — to a parent, an administrator, or in a later review — as disproportionate, even if the underlying behavior was serious.
First question: "What happened at the private-redirect stage before this, and is it logged?"
Data to pull: the behavior log for the student that period/day.

### Three or more students in one section requesting the same "extra help" on the same sub-objective

Usually means: the group, not the individuals, needs the reteach — treating it as three separate individual make-up conversations misses that it's a section-wide gap needing the Day-6-style contingent reteach, not three tutoring sessions.
First question: "Is this the same sub-objective across all three requests?"
Data to pull: which items/sub-objectives each student is asking about, cross-referenced against the item analysis.

### A formative assessment counted at full summative weight in the gradebook

Usually means: the diagnostic function of the formative check has been abandoned — students (and the teacher) now treat it as high-stakes, which changes what the data actually measures (test anxiety, guessing to protect a grade) and removes the incentive to use a bad score as information rather than a setback to hide.
First question: "Is this quiz weighted so a bad score changes tomorrow's lesson, or just the report card?"
Data to pull: gradebook category weights for the term.

### A summative test date moved for an individual absence/make-up request rather than a documented standards gap

Usually means: an individual accommodation request is being handled as if it were a class-wide pacing decision, which delays 140+ other students' next assessment for one student's scheduling issue.
First question: "Is there class-wide item-analysis evidence behind this move, or is it one student's request?"
Data to pull: the item analysis (if any) supporting the date change; the specific individual request if there isn't one.

### An IEP/504 accommodation being adjusted in the moment, in the classroom, without the case manager

Usually means: well-intentioned but unauthorized modification of a legally binding plan — even a generous change (more time, fewer questions) made unilaterally creates an inconsistency between what's documented and what's delivered.
First question: "Is this accommodation not working as written, and has that been reported to the case manager, or am I changing it myself right now?"
Data to pull: the current IEP/504 accommodation list, any prior case-manager communication about this specific accommodation.
