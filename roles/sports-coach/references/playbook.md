# Playbook

Filled templates for the four recurring artifacts: recruiting grade rubric, weekly periodized practice plan, acute:chronic workload ratio (ACWR) walkthrough, and the return-to-play stage table.

## 1. Recruiting/transfer grade rubric (20-80 scale)

Score each trait independently before computing a composite — never assign a composite first and back-fill traits to match it.

| Trait | Weight (skill position, e.g. CB) | Prospect A | Prospect B |
|---|---|---|---|
| Ball skills / range | 30% | 65 | 55 |
| Tackling / leverage | 30% | 42 | 55 |
| Long speed (40 time) | 15% | 62 | 50 |
| Football IQ (charted reads) | 15% | 58 | 56 |
| Frame/projection | 10% | 55 | 60 |
| **Weighted composite** | | **55.6** | **54.9** |

Composite math: A = .30(65) + .30(42) + .15(62) + .15(58) + .10(55) = 19.5 + 12.6 + 9.3 + 8.7 + 5.5 = 55.6 (round to 56). B = .30(55) + .30(55) + .15(50) + .15(56) + .10(60) = 16.5 + 16.5 + 7.5 + 8.4 + 6.0 = 54.9.

**Composite alone would favor A by 0.7 points — this is why need-weighting happens after the trait breakdown, not by adjusting the composite.** A run-support need reweights tackling to 45% and ball skills to 15%: A becomes .15(65)+.45(42)+.15(62)+.15(58)+.10(55) = 9.75+18.9+9.3+8.7+5.5 = 52.2; B becomes .15(55)+.45(55)+.15(50)+.15(56)+.10(60) = 8.25+24.75+7.5+8.4+6.0 = 54.9. Reweighted for the actual need, B passes A (54.9 vs 52.2) — the generic composite was hiding the need-specific answer.

**Trajectory flag** (separate from the grade): note age-in-cohort, redshirt-year runway, and any relative-age-effect discount applied. A prospect who is 11 months younger than the position-group median with matching production gets a written flag: "production may understate true talent by ~half a grade; revisit at year-2 checkpoint."

## 2. Weekly periodized practice plan (in-season microcycle, game on Saturday)

| Day | Focus | Volume (total live reps) | Intensity | Notes |
|---|---|---|---|---|
| Sun | Recovery / injury treatment | 0 | — | Film review only, no field work for starters with >70 offensive/defensive snaps prior game |
| Mon | Corrections + install (base) | 40 | Low-moderate | Walkthrough pace; no full-pad contact |
| Tue | Install (situational) + team period | 65 | High | Heaviest practice of the week; full pads, live tempo |
| Wed | Install (opponent-specific) | 55 | High | Second-heaviest; red zone and third-down emphasis |
| Thu | Review + special teams | 30 | Moderate | Shells only, reduce contact volume ahead of game |
| Fri | Walkthrough | 10 | Low | Mental reps, no pads |
| Sat | Game | — | Max | — |

Weekly total live-rep volume for a starter: ~200 reps. This microcycle sits inside a 4-week mesocycle block; the ACWR calculation below uses this week as the "acute" week against the trailing 4-week average.

## 3. Acute:chronic workload ratio (ACWR) — worked calculation

Player's live-rep counts, most recent 4 weeks: 210, 195, 200, 200 (this week).

- **Chronic load** (4-week rolling average) = (210 + 195 + 200 + 200) ÷ 4 = 805 ÷ 4 = 201.25
- **Acute load** (most recent week) = 200
- **ACWR** = 200 ÷ 201.25 = **0.994**

0.994 sits inside the 0.8–1.3 sweet-band — hold volume steady next week. If a game-week spike pushed acute load to 310 (e.g., overtime plus a short week), ACWR = 310 ÷ 201.25 = 1.54 — above the ~1.5 flag threshold, so the following week's practice volume gets cut by roughly 20–25% (to ~150–160 reps) rather than held flat, unless the player has a documented high off-season aerobic base that justifies the spike.

## 4. Graduated return-to-play protocol (head injury, Berlin 2016 framework)

Each stage requires 24 hours symptom-free before advancing; any symptom return means dropping back one stage.

| Stage | Activity | Goal |
|---|---|---|
| 1 | Symptom-limited daily activity | Return to normal life activities |
| 2 | Light aerobic exercise (walk/stationary bike, <70% max heart rate) | Increase heart rate |
| 3 | Sport-specific exercise (skating, running drills — no head-impact risk) | Add movement |
| 4 | Non-contact training drills | Exercise, coordination, cognitive load |
| 5 | Full-contact practice (medical clearance required first) | Restore confidence, assess functional skills |
| 6 | Return to competition | — |

Minimum elapsed time at 24 hours/stage across 6 stages = 5 days from Stage 1 clearance to Stage 6 clearance; in practice most programs run 7–10 days minimum given typical symptom-check scheduling around travel and game days. No stage is skipped for a playoff game — the gate is symptom-based, not calendar-based.
