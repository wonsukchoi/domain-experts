# Playbook

## Fitting diagnosis reference table (filled example)

| Wrinkle/pull signature | Likely diagnosis | Targeted correction |
|---|---|---|
| Horizontal pull line across back at waist, front fits fine | Insufficient ease in back panel specifically | Add ease at back side seams only, not full circumference |
| Diagonal wrinkle from shoulder toward bust point | Bust dart placement or size mismatch | Adjust dart position/size, not overall bodice width |
| Diagonal wrinkle radiating from underarm toward waist | Armhole/side seam balance issue | Rebalance the side seam curve at the armhole, not a uniform waist adjustment |
| Excess fabric bubbling at small of back | Too much ease in the back specifically, or a swayback posture adjustment needed | Take in at center back seam or add a swayback dart |
| Fabric straining across upper back/shoulder blades | Insufficient back width or shoulder ease | Add width across the back shoulder blade area specifically |

## Fitted jacket ease correction calculation (filled example)

Customer waist: 30". Standard fitted-jacket ease: 2". Target finished waist: 32".

| Step | Value |
|---|---|
| Target finished waist | 32" |
| Front panel (already fits at first fitting) | Contributing correct proportional ease — no adjustment |
| Back panel pull observed | Horizontal pull at waist |
| Measured ease needed to relieve back pull | 3/4" total |
| Distribution across 2 back side seams | 3/4" ÷ 2 = 3/8" per seam |

**Correction applied:** Add 3/8" at each back side seam only (3/4" total), leaving the front pattern unchanged since it already measures and fits correctly.

## Seam allowance feasibility check (filled example)

Client requests letting out a dress waist by 1.5" total.

| Measurement | Value |
|---|---|
| Requested letting-out amount (total, both side seams) | 1.5" |
| Per-seam requested amount (2 side seams) | 0.75" per seam |
| Actual measured seam allowance at each side seam | 0.625" (5/8") |
| Feasible? | **No — requested 0.75"/seam exceeds the 0.625" available** |
| Maximum feasible letting-out (both seams combined) | 0.625" × 2 = 1.25" total |

**Result:** The requested 1.5" total letting-out exceeds what the garment's actual 5/8" seam allowance can provide (maximum feasible is 1.25" total). Communicate the 1.25" ceiling to the client before proceeding, rather than promising the full 1.5" and discovering the shortfall mid-alteration.
