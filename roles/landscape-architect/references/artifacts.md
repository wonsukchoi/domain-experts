## Stormwater sizing worksheet (rational method)

| Input | Value |
|---|---|
| Runoff coefficient (C), impervious | 0.85 |
| Runoff coefficient (C), landscape | 0.20 |
| Rainfall intensity (i), 10-yr/24-hr local IDF curve | 2.1 in/hr |
| Impervious drainage area (A) | 1.15 acres |
| Landscape drainage area (A) | 0.95 acres |
| Peak flow, impervious: Q = CiA | 0.85 × 2.1 × 1.15 = **2.05 cfs** |
| Peak flow, landscape: Q = CiA | 0.20 × 2.1 × 0.95 = **0.40 cfs** |
| Total site peak (undetained) | **2.45 cfs** |
| Pre-development target discharge | 0.60 cfs |
| Required detention volume (attenuation to target) | **1.15 ac-ft** |

Trigger check: net new impervious area (50,000 sf) vs. jurisdiction review threshold (confirm locally; commonly 5,000–10,000 sf) — this site exceeds it, full detention review required.
