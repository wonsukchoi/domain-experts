# Playbook

Filled worksheets for method selection, hole spacing, sounding, and yield logging. Numbers are stated heuristics drawn from quarry-supplier technical guidance and craft-association process descriptions unless otherwise noted — recalibrate against a given ledge's own logged history, since rock behavior varies quarry to quarry.

## 1. Split-method selection table

| Situation | Default method | Why |
|---|---|---|
| Rift-aligned line, one-off or irregular block | Hand plug-and-feather | Fastest to set up, no machine mobilization, reliable on rift/grain |
| Grain-aligned line, one-off block | Hand plug-and-feather, tighter spacing (see §2) | Still hand-viable but crack propagates less readily than on rift |
| Hard-way line, any block size | Wire saw or chop saw | Hand-wedged hard-way splits run materially lower clean-split rates than rift/grain on the same rock (this ledge's log: ~60% vs ~95%) |
| Production block program, repeat cuts on the same face geometry | Wire saw or hydraulic guillotine splitter | Setup cost amortizes across repeat cuts; hand method's per-cut labor cost doesn't scale down |
| Large mass separation from the ledge (not yet block-sized) | Jackhammer channeling, or presplit blasting where the quarry uses it | Plug-and-feather is a finishing/block-sizing tool, not a mass-removal tool |
| Slab/sheet cutting off a finished block | Chop saw or gang saw | Different geometry (thin repeat cuts) than block splitting |

## 2. Plug-and-feather hole spacing by rock class

| Rock class | Hole diameter | Spacing (o.c.) | Depth | Notes |
|---|---|---|---|---|
| Fine/medium-grained granite, rift-aligned | 7/8–1 in | 8 in | 5–6 in | Standard production spacing |
| Fine/medium-grained granite, grain-aligned | 7/8–1 in | 6 in | 5–6 in | Tighter than rift to keep the crack on line |
| Coarse-grained or seamy granite, any direction | 7/8–1 in | 4–6 in | 5–6 in | Wider spacing risks the crack stalling between holes |
| Marble | 3/4–7/8 in | 6–8 in | 4–5 in | Marble splits more predictably than granite along bedding but is more prone to unwanted secondary cracks near a seam |
| Slate | 5/8–3/4 in | 4–6 in | 3–4 in | Fine natural cleavage — tighter spacing controls sheet thickness |
| Limestone (bedded) | 7/8–1 in | 8–10 in | 5–6 in | Split along the natural bed wherever the spec allows, matching the stonemason's natural-bed principle at the extraction stage |

## 3. Sounding and grain-reading checklist (run before marking any line)

1. Tap the face at 1–2 ft intervals along the intended split line with a sounding hammer; log any dull/dead return as a probable seam location.
2. Visually trace grain lines across the face in raking light; confirm rift direction against known ledge orientation, not assumed from the previous block.
3. Cross-check any dull-tap zone against visible grain/color banding — a seam frequently, but not always, shows a visible line.
4. If a dull zone falls inside the marked block boundary, re-mark to route around it or reduce block size before drilling, rather than drilling through it and hoping.
5. Record the confirmed rift/grain/hard-way orientation for the face in the block log, so the next crew on this face doesn't re-derive it from scratch.

## 4. Hard-way conflict worksheet (expected-yield comparison)

Use when a block spec's dimensions put any cut face on the hard way and a hand split is the default method being considered.

| Step | Formula | Worked value (Block GB-114 example) |
|---|---|---|
| Raw block volume | L x W x H | 12' x 8' x 5' = 480 cf |
| Raw block weight | volume x density (granite ≈165 lb/cf) | 480 x 165 = 79,200 lb = 39.6 st |
| Logged clean-split rate for this face's rock | from crew face log | rift/grain ≈95%, hard-way ≈60% |
| Expected usable tonnage | raw weight x clean-split rate | 39.6 x 0.60 = 23.76 st (hard-way) |
| Revised spec (trim to keep all faces on rift/grain) | recompute volume/weight | 11' x 8' x 5' = 440 cf = 36.3 st |
| Revised expected usable tonnage | 36.3 x 0.95 | 34.485 st |
| Decision rule | choose the option with higher expected usable tonnage, not higher raw tonnage | revised spec wins by ~10.7 st (≈45%) |

Route the trimmed spec back to the block order desk for sign-off before re-marking — a dimension change against a customer order is not the splitter's call to finalize alone.

## 5. Block yield log format (per block, logged before loadout)

| Field | Example entry |
|---|---|
| Block ID | GB-114 |
| Face / bench | Face 14 |
| As-split dimensions | 11'0" x 8'0" x 5'0" |
| As-split weight | 36.3 st (computed) / confirmed on scale at loadout |
| Split method | Hand plug-and-feather, 7/8" x 6" o.c. x 5" deep |
| Seam/quality note | None on final faces; original 12' spec seam noted and avoided |
| Clean-split (Y/N) | Y |
| Logged by / date | — |

A block that logs "N" on clean-split gets routed to secondary sizing (smaller block, dimension stone rubble, or aggregate feed) rather than passed forward as if it met spec.
