# Red flags

### A single uniform grade rule increment applied across an entire size range without checking for non-uniform size chart increments
- **Usually means:** progressively worse fit at sizes distant from where the uniform increment was actually validated.
- **First question:** does the size chart show the same measurement increment at every size break, or does it vary at certain transitions?
- **Data to pull:** size chart increment table compared against the grade rule increment(s) actually applied.

### Pattern outline grown across sizes while ease allowance distribution is left unchanged from the base size pattern
- **Usually means:** proportional fit is distorted at sizes away from the base.
- **First question:** does ease allowance at the graded extreme sizes actually match the base size's proportional distribution?
- **Data to pull:** ease allowance at the base size compared against ease allowance as actually distributed at graded extreme sizes.

### Internal pattern details left at a fixed position during grading rather than moved proportionally with their related body point
- **Usually means:** these details end up mispositioned relative to the body at sizes away from the base.
- **First question:** does the detail's position at graded sizes actually track the body point it's keyed to?
- **Data to pull:** internal detail position at base size compared against position at graded sizes, relative to the body point each detail is keyed to.

### A new pattern graded across a size range with no fit-test sampling at the smallest and largest sizes
- **Usually means:** fit problems specific to the range extremes go undetected until customer or production feedback.
- **First question:** which sizes in the range were actually sample-made and fit-tested?
- **Data to pull:** fit-test sample records compared against which sizes in the range were actually tested.

### A size chart or brand standard changed without re-deriving grade rules for existing patterns
- **Usually means:** patterns continue being graded to an outdated increment table that no longer matches current target measurements.
- **First question:** does the current size chart match the one the existing grade rules were originally derived from?
- **Data to pull:** current size chart compared against the size chart/increments the existing grade rules were originally derived from.

### A recurring fit complaint pattern for a specific size or size range appears without a traced grading cause
- **Usually means:** a systemic grading issue — uniform-rule-over-non-uniform-range, ease distortion, internal detail mispositioning — rather than isolated one-off variation.
- **First question:** does the fit complaint log correlate with a specific size break or grading methodology?
- **Data to pull:** fit complaint log by size compared against the grade rules/methodology actually used for that pattern.

### A grade rule created for one garment category reused directly for a different category without adjustment
- **Usually means:** the grade rule may not suit the different stretch/ease characteristics of the new category, producing off-fit results even if the size chart numbers are nominally the same.
- **First question:** were grade rules specifically validated for this garment category and fabric type?
- **Data to pull:** garment category and fabric type compared against whether grade rules were specifically validated for that category.
