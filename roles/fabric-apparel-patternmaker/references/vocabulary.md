# Vocabulary

### Grade rule
A specific incremental amount added or subtracted at a defined pattern point when scaling a pattern from one size to the adjacent size, derived from actual body measurement progression rather than a uniform percentage.
**In use:** "Grade rule at this pattern point is 0.5 inches per size step from 10 to 16, but drops to 0.375 inches for the plus-size transition."
**Common misuse:** Applying a single uniform grade rule increment across an entire size range without checking whether the underlying size chart has different increments at certain size breaks.

### Nested grading
A pattern grading method where all sizes in a range are displayed layered or nested around the base size pattern, showing how each pattern point moves outward or inward at each size step.
**In use:** "Pulled up the nested grading file to check how the dart point tracks across all six sizes at once."
**Common misuse:** Assuming nested grading output is automatically correct without verifying it was built from the actual size chart's specific, possibly non-uniform, increments.

### Ease allowance (grading context)
The built-in difference between body measurement and finished garment measurement, which must be preserved consistently in proportion across all graded sizes, not just at the base size.
**In use:** "Check ease at size 20 specifically — growing the outline alone doesn't guarantee the same proportional fit as the base size."
**Common misuse:** Growing only the pattern's outline dimensions during grading while leaving the base size's ease distribution unchanged, distorting proportional fit at sizes away from the base.

### Base size / sample size
The specific size at which a pattern is originally developed and fit-tested, from which all other sizes in the range are graded up or down.
**In use:** "Base size 10 fits correctly — that doesn't confirm the graded size 20 will fit the same way relative to its own body measurements."
**Common misuse:** Assuming a pattern validated as correctly fitting only at the base size will automatically fit correctly at all graded sizes without further verification.

### Size chart
A specification document listing target body measurements for each size in a range, whose increments between adjacent sizes form the actual basis for grade rules.
**In use:** "Size chart's increments compress at the 16-to-18 break — that has to be reflected in the grade rule, not assumed uniform."
**Common misuse:** Assuming a size chart's increments are uniform across the entire range without checking, when many charts have compressed or expanded increments at specific transitions.

### Dart manipulation (grading)
Techniques for repositioning a pattern's darts as it's graded across sizes, keeping them correctly placed relative to the body point they shape, like the bust point.
**In use:** "Dart manipulation moves the bust dart point outward at each size step — it can't stay fixed while the pattern grows around it."
**Common misuse:** Leaving darts at a fixed position during grading rather than moving them proportionally as the surrounding pattern grows or shrinks.

### Pattern point
A specific, defined location on a pattern piece — a seam intersection, a dart tip, a notch — where grade rule increments are applied during grading.
**In use:** "Check the grade rule at every pattern point, not just the side seam — the pocket placement is a pattern point too."
**Common misuse:** Applying grade rule increments only at the pattern's outer perimeter without also correctly grading internal pattern points relative to their related body measurements.

### Fit-test sampling
The practice of producing and physically testing sample garments at specific sizes, particularly the extremes of a graded range, to verify actual fit rather than relying on grading calculations alone.
**In use:** "Fit-test sampling at size 4 and size 20 caught a proportion issue the grading math alone wouldn't have shown."
**Common misuse:** Sampling and fit-testing only the base size, assuming the grading math guarantees correct fit at every other size in the range without physical verification.

### Size break
The specific transition point between two adjacent sizes in a range, where a grade rule increment is applied, which may differ from the increment used at other breaks in the same range.
**In use:** "This size break needs its own grade rule — the size chart's increment here doesn't match the rest of the range."
**Common misuse:** Assuming all size breaks in a range use the same grade rule increment, rather than checking each break against the size chart's actual measurement difference at that specific transition.

### Proportional grading
Grading a pattern so that all its features — outline, ease, internal details — scale in a way consistent with actual body measurement progression, as opposed to a simple percentage-based enlargement of the whole pattern.
**In use:** "This is proportional grading, not a photocopier scale-up — every pattern point moves by its own calculated amount."
**Common misuse:** Treating pattern grading as a uniform scale-up/scale-down operation, like resizing an image, rather than a measurement-driven process where different pattern points move by different, specifically calculated amounts.
