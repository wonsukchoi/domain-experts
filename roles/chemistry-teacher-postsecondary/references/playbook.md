# Playbook

## Lab report grading rubric (fill per report)

| Component | Weight | What earns full credit | What loses points |
|---|---|---|---|
| Methodology / technique | 40% | Correct procedure followed, deviations noted and justified | Undisclosed shortcuts, steps skipped without comment |
| Error analysis / documentation | 30% | Loss/error sources identified, plausible magnitude estimated | No error analysis, or a magnitude inconsistent with the stated cause |
| Numeric result (vs. plausible range, not a single expected value) | 20% | Result consistent with the documented methodology and error sources | Result unexplained by the methodology described, in either direction |
| Purity / corroboration check (melting point, titration, TLC, etc.) | 10% | Independent check run and reported | No corroborating check on a yield claim above ~85% |

**Rule:** a yield above ~85% with no purity/corroboration check is treated as unverified, not as evidence of success — request the check before finalizing the grade.

### Worked scoring example

| Student | Yield | Purity check | Error documented | Score |
|---|---|---|---|---|

## Lab-safety pre-run checklist (complete before any new demo/lab is run live)

| Check | Confirm before running |
|---|---|
| SDS reviewed for every reagent in this specific procedure | Y/N |
| Ventilation/fume hood tested functional in this room, this day | Y/N |
| Quantities scaled to this room's equipment (not copied at textbook scale by default) | Y/N |
| Waste stream identified and disposal method confirmed | Y/N |
| Dry run completed by instructor/TA under this room's actual conditions | Y/N |
| PPE requirements specified and available in sufficient quantity for class size | Y/N |
| Emergency procedure (spill, exposure, fire) reviewed with any TA running the section | Y/N |

**Rule:** if any row is "N," the procedure does not run live that day — reschedule after the gap is closed, regardless of syllabus timing pressure.

## Procedural-freedom gating table

| Student latitude requested | Prerequisite before granting |
|---|---|
| Choice of pre-approved reagent/step variants only | No additional review needed beyond standard pre-lab check |
| Student-proposed modification to a vetted procedure | Instructor reviews the specific modification against the hazard checklist before approval |
| Fully open-ended, student-designed procedure | Full hazard/safety review of the specific proposed procedure, signed off before any reagent is issued |

## Simplified-model-to-limits map (state the limit in the same lecture the model is taught)

| Simplified model taught | Where it breaks down | What replaces it later |
|---|---|---|
| Bohr model of the atom | Cannot explain multi-electron atom spectra or orbital shapes | Quantum mechanical orbital model |
| Octet rule | Fails for expanded octets (e.g., PCl5, SF6) and electron-deficient species (e.g., BF3) | Molecular orbital theory / formal exceptions taught explicitly |
| Ideal gas law | Breaks down at high pressure / low temperature where intermolecular forces matter | Van der Waals / real gas equations of state |
| Single-step reaction mechanism (e.g., simple SN2 diagram) | Many reactions proceed through multi-step mechanisms with intermediates | Full multi-step mechanism with rate-determining step analysis |
| Arrhenius acid-base definition | Cannot explain reactions with no H+/OH- in the medium (e.g., in non-aqueous solvents) | Brønsted-Lowry, then Lewis acid-base theory |

**Rule:** name the model being taught as a simplification the first time it's introduced, not only when a student happens to ask about an exception.
