# Playbook

## Chilling verification worksheet

Fill for the sampling plan's specified carcass checks.

| Field | Value |
|---|---|
| HACCP requirement | 41°F internal deep-tissue within 24 hrs |
| Chiller setpoint | 34°F |
| Chiller air temp at 24-hr check | 34°F (setpoint met) |
| Deep-tissue probe result | 46°F — FAILS requirement despite air temp |
| Carcass position/loading | Position C-14, dense loading, reduced airflow |
| Corrective action | Extended chilling 4 additional hrs |
| Re-check result | 40°F — within spec |
| Position flagged for review? | Y — loading/airflow issue at C-14 |

**Rule:** chiller air temperature reaching setpoint is never sufficient confirmation of chilling requirement compliance — always verify deep-tissue temperature directly per the sampling plan, with extra scrutiny on known problem positions.

## Stunning/evisceration control checklist

1. Confirm stunning equipment is calibrated and functioning per specification.
2. For EVERY animal, verify specific insensibility signs before proceeding to the next step — never rely on equipment status alone.
3. Perform evisceration using the specified technique to minimize intestinal/bung contamination risk as the primary control.
4. Treat post-evisceration inspection as a backup check, not the primary defense — technique compliance is the main control.
5. If contamination is found at inspection, investigate whether it traces to a technique issue (requiring retraining/correction) versus an isolated event.
6. Sanitize equipment/knives per the specified interval or trigger without exception, regardless of production pace pressure.
7. If a contamination event occurs, check whether other carcasses processed on the same equipment since the last sanitation cycle also need investigation.

## Line-speed/inspection-capacity matching guide

1. Identify each station's specific verification task and the time realistically required to perform it thoroughly.
2. Compare current line speed against that required time — confirm adequate margin, not just barely sufficient time.
3. Before increasing line speed, re-verify each affected station's verification task can still be performed thoroughly at the new pace.
4. Monitor defect/contamination detection rates after any line speed change — a drop in detection rate at unchanged actual defect rates suggests inspection capacity has been exceeded.
5. If inspection capacity can't support a desired line speed, address through additional staffing/stations rather than reducing verification rigor.
