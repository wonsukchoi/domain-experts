# Clinical/Counseling Psychologist — Playbook

## Battery-selection matrix by referral question

| Referral question | Core instruments | Skip unless indicated |
|---|---|---|
| ADHD vs. anxiety/mood-driven inattention | WAIS-IV/WISC-V, CPT (commission + omission errors), MMPI-3 or PAI, structured developmental interview | Full personality battery, projective measures |
| Learning disability / academic accommodation | WAIS-IV/WISC-V + achievement battery (e.g. WIAT-4), processing-speed and working-memory subtests weighted heavily | Personality/psychopathology battery |
| Personality disorder vs. mood/anxiety disorder | MMPI-3 or PAI, SCID-5-PD, collateral interview | Cognitive battery (unless cognitive complaint co-presents) |
| Early dementia vs. depression ("pseudodementia") in older adult | Brief cognitive screen + WAIS-IV subset, mood inventory, medical workup (TSH, B12) requested same visit | Full-length cognitive battery on first visit — repeat testing in 6-12 months is often more informative than one long session |
| Fitness-for-duty / custody / disability (secondary-gain context) | Full battery including at least one embedded and one standalone Performance/Symptom Validity Test | — validity testing is not optional in this context |

## Validity-check sequence (run before interpreting any content scale)

1. Check embedded validity indices on the self-report instrument first (e.g. MMPI-3 F-r, Fp-r, VRIN-r, TRIN-r).
2. If a Performance/Symptom Validity Test was administered, check pass/fail against its published cutoff (e.g. TOMM Trial 2 raw score < 45/50 flags likely invalid effort).
3. If either check fails, stop interpreting content scales — the finding is "profile invalid, reason X," not a diagnosis.
4. If both pass, proceed to convergence check: for each candidate finding, require a second independent method pointing the same direction before writing it into the formulation.

## Second worked example — index-score interpretation table

Referral: workplace accommodation request, processing-speed complaint.

| Index | Standard score | Percentile | 95% CI | Flag |
|---|---|---|---|---|
| Verbal Comprehension | 118 | 88th | 111-123 | — |
| Perceptual Reasoning | 121 | 92nd | 114-126 | — |
| Working Memory | 103 | 58th | 96-109 | — |
| Processing Speed | 79 | 8th | 73-88 | Below 10th percentile — supports accommodation request |

Spread: 121 - 79 = 42 points (2.8 SD) — Full Scale IQ not reported; each index stands alone. Processing Speed at the 8th percentile, more than 2 SD below this individual's own Perceptual Reasoning index (121), is the actionable finding: a within-person discontinuity, not just a below-average absolute score, is what typically supports an accommodation determination (e.g. extended time).

Deliverable excerpt: "Processing Speed (79, 8th percentile) is significantly discrepant from this individual's own Perceptual Reasoning (121, 92nd percentile) — a 42-point spread — indicating a specific processing-speed deficit rather than globally reduced cognitive ability. This pattern supports the requested extended-time accommodation for timed work tasks."

## Feedback-session structure (filled)

1. Open with the referral question restated in the client's language: "You came in because work asked whether X — here's what we found."
2. Lead with function, not label: what the person can and can't do, and in what contexts, before naming the diagnosis.
3. Name the diagnosis, tie each criterion met to a specific test finding (not "the test says so").
4. State what was ruled out and why, so the client understands the answer was earned, not assumed.
5. End with the recommendation tied directly back to the original referral question.
