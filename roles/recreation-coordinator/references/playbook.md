# Playbook

Filled templates a recreation coordinator runs directly, not descriptions of them.

## 1. Ratio calculation worksheet

Run this before finalizing any day's staffing, per age band, per activity type — never as a single blended number.

| Age band | Land ratio (ACA) | Water ratio (add lifeguard, keep counselor ratio) | Off-site minimum |
|---|---|---|---|
| 4–5 | 1:6 | 1:6 counselor + 1 lifeguard per pool | 2 staff minimum (two-deep) |
| 6–8 | 1:8 | 1:6 counselor + 1 lifeguard per pool | 2 staff minimum |
| 9–11 | 1:10 | 1:8 counselor + 1 lifeguard per pool | 2 staff minimum |
| 12–14 | 1:12 | 1:10 counselor + 1 lifeguard per pool | 2 staff minimum |
| 15–18 | 1:12 | 1:12 counselor + 1 lifeguard per pool | 2 staff minimum |

**Worked calculation (Explorers group, 14 campers, ages 6–8, land activity):**
```
Required by ratio:     14 ÷ 8 = 1.75 → round up → 2 staff
Required by two-deep:  minimum 2 staff regardless of ratio math
Binding constraint:    max(2, 2) = 2 staff — both floors agree here
```
When the two numbers disagree (e.g., a 9-camper group of 12–14-year-olds: ratio math gives 9÷12 = 0.75 → 1, but two-deep still requires 2), **two-deep always wins** — it is a structural rule, not a ratio.

## 2. Session/activity plan template (filled example)

```
ACTIVITY: Archery skills — Level 2 (progression from Level 1 target basics)
DATE / GROUP: Wed 7/16, Trailblazers (9–11), 16 campers
OBJECTIVE: 80% of returning campers land 3 of 5 arrows inside the blue ring
  (up from white ring in Level 1) — measurable progression, not "have fun."
STAFFING: 2 counselors (meets 1:10 + two-deep) + 1 certified range instructor
MATERIALS: 8 bows, arrows, 4 target lanes, first-aid kit staged at range entry
SAFETY CHECK: range fencing intact, arrow-retrieval whistle system briefed,
  weather — no activity if lightning within 10 miles (30-30 rule)
BACKUP PLAN (rain/thunder): indoor knot-tying and camp-craft rotation
STAFF BRIEF NOTES: camper #14 (D.M.) has a noted attention accommodation —
  pair with lead counselor in first rotation, shorter draw-and-wait intervals
DEBRIEF FIELDS: # campers hitting objective, any near-miss, one change for
  next session
```

## 3. Incident / near-miss report (filled example)

Filed within 24 hours. Facts only — no diagnosis, no blame language, no speculation about cause until reviewed.

```
TYPE: Near-miss (no injury)
DATE/TIME: 7/16, 2:40 PM
LOCATION: Archery range, lane 3
WHO: Camper J.T. (age 10), witnessed by counselors R. Lee and A. Fisk
WHAT HAPPENED (facts only): Camper crossed the retrieval line before the
  "all clear" whistle while another camper still had an arrow nocked.
  No arrow was released. Instructor called "hold" and camper stepped back.
IMMEDIATE ACTION: Camper re-briefed on retrieval-line procedure 1:1 before
  rejoining. Group re-briefed on whistle signals before next rotation.
CONTRIBUTING FACTOR (if evident): Retrieval-line marking faded on lane 3 —
  visibility lower than lanes 1-2.
FOLLOW-UP: Facilities ticket filed to repaint lane 3 marking before Friday.
  Logged in near-miss tracker — 2nd lane-3 line-crossing this month.
ROUTED TO: Program Supervisor (same day), Facilities (ticket #4471)
```

**Rule:** the same near-miss location logged twice in one season triggers a facilities or procedure review before a third occurrence, not after an injury.

## 4. Inclusion accommodation worksheet

```
PARTICIPANT: camper, age 11, uses forearm crutches, full-day program request
DEFAULT: include in standard program with accommodation (ADA Title II) —
  segregate to a separate program only if an individualized assessment
  shows the standard program cannot be made safe/accessible.
ASSESSMENT:
  - Mobility: navigates flat/paved surfaces independently; needs alternate
    route for the trail-hike activity (unpaved, 6% grade).
  - Water activity: swims independently, no accommodation needed at pool.
  - Staffing: no ratio change required — accommodation is route/equipment,
    not 1:1 staffing.
ACCOMMODATION PLAN:
  - Trail hike → paved nature-loop alternate route, same destination point,
    counselor pre-walks the route Monday to confirm surface condition.
  - All other activities → standard program, no modification.
REVIEW DATE: 2 weeks in, confirm plan is working with camper and parent.
```
