# Rental Counter Clerk — Playbook

## Pre-/post-rental inspection checklist (filled example)

Applied identically at pickup and return; the return pass is graded against the pickup pass, not against a fresh assumption of "should be clean."

| Check | Pickup example | Return example |
|---|---|---|
| Exterior/body panels | Photographed 4 sides + top; existing scuff on left rear panel marked on diagram, initialed by customer | Compared against pickup photos; new cracked rear window flagged, not on pickup diagram |
| Fluid/fuel/charge level | Diesel: full (marked on gauge diagram) | Diesel: 3/4 — within full-to-full tolerance (no fee) |
| Controls/attachments function test | Bucket, hydraulics, tracks tested and confirmed functional, initialed | Hydraulic hose fitting cracked — flagged as new damage, not a function-test failure at pickup |
| Hour meter / odometer | Logged: 412.3 hrs | Logged: 418.9 hrs (6.6 hrs used — within expected range for a 3-day rental) |
| Attachments/accessories present | Bucket, thumb, ramps — 3 items checked off | All 3 returned |
| Signatures | Customer + clerk initials on diagram | Customer present for return walk; declines to initial on disputed damage — noted on log, manager notified same day |

Never complete a return inspection without the customer present or a timestamped photo set — an inspection done alone after the customer leaves can't establish when the damage occurred.

## Damage-waiver (LDW) decision matrix

| Situation | Action |
|---|---|
| Customer accepts waiver | Charge 10-15% of rental subtotal (store-set rate); customer's exposure caps at the per-incident deductible ($250-$500 typical, store-set) for accidental damage. |
| Customer declines, current COI on file naming store as additional insured | Valid decline — no waiver charged; customer's own policy carries the risk per the COI terms. |
| Customer declines, no COI on file, item value under store's COI-required threshold (e.g., under $5,000 replacement value) | Valid decline permitted at store discretion — document verbally-declined-no-COI-required in the file. |
| Customer declines, no COI on file, item value over the COI-required threshold | **Invalid decline** — waiver must be charged by default; do not release the item on a bare verbal assurance. |
| Waiver purchased, damage traced to gross negligence, theft, or an excluded use (e.g., off-road use of an on-road rental) | Waiver does not apply — full repair/replacement billed regardless of waiver purchase. |

## Deposit / authorization-hold sizing table (filled example)

| Item class | Replacement/repair exposure | Hold type | Typical amount |
|---|---|---|---|
| Hand tools, pressure washers, small power equipment | Under $1,000 | Credit or debit authorization hold | $50-$150 |
| Mid-size equipment (tillers, generators, mini-excavators) | $1,000-$15,000 | Credit-card authorization hold only (no debit) | $300-$750, or one day's rate + estimated charges, whichever is greater |
| Vehicles (passenger, cargo van, box truck) | Varies by class | Credit-card hold; debit accepted only with proof of return travel (flight itinerary) and a higher hold | $200-$500 (credit) / $400-$600+ (debit) |
| Storage unit rental | N/A (no physical unit handed over) | First month + admin fee collected at signing, not a hold | Store-set; no separate deposit typical |

Hold amounts release 3-5 business days after a clean return inspection is logged — tell the customer this timeframe at checkout so a held charge doesn't read as a billing dispute later.

## Return-condition fee schedule (filled example)

| Condition at return | Fee |
|---|---|
| Fuel/charge below contracted level (e.g., full-to-full) | Flat refuel service fee, e.g., $45, regardless of how much fuel is actually missing — never prorate by eye |
| Excessive dirt/mud requiring more than standard wash | Flat cleaning fee, e.g., $25-$75 by item class |
| Late return, under 2 hours past contracted time | Grace period — no fee, store-set |
| Late return, 2+ hours past contracted time | One additional day's rate charged, applied automatically, not negotiated at the counter |
| Missing accessory/attachment | Replacement cost billed, itemized against the pickup accessory checklist |

## Self-storage lien-sale notice sequence (for storage-account rental clerks)

State statutes vary (30-90 day windows are common); the clerk's job is to trigger the sequence on schedule and route to the manager/attorney for the notice content, not to draft notices independently.

1. Account reaches default (rent unpaid past the store's defined delinquency threshold, commonly 10-30 days).
2. Written default notice sent to the tenant at the address/contact on file — summarizes the amount owed and the cure deadline.
3. If unpaid past the state-mandated cure period, a lien/intent-to-sell notice is sent, stating the amount owed, the sale date, time, and manner.
4. Any secondary lienholders on file (rare, but check the account) receive notice per state statute before the sale proceeds.
5. Sale conducted (commonly public auction); proceeds applied to the debt first, remainder returned to the tenant per state statute — document the full sequence for the file regardless of outcome.

Escalate any account where the tenant disputes the debt or claims they never received notice immediately — do not proceed to sale on a disputed account without manager/legal sign-off.
