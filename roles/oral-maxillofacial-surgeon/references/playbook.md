# Oral and Maxillofacial Surgeon — Playbook

## Pederson Difficulty Index worksheet

| Factor | Options | Points |
|---|---|---|
| Angulation (Winter's) | Mesioangular | 1 |
| | Horizontal / transverse | 2 |
| | Vertical | 3 |
| | Distoangular | 4 |
| Ramus relationship / space (Pell & Gregory Class) | Class I — adequate space | 1 |
| | Class II — ramus partially limits access | 2 |
| | Class III — ramus fully overlaps crown | 3 |
| Depth (Pell & Gregory Position) | Position A — at/above occlusal plane | 1 |
| | Position B — between occlusal plane and cervical line | 2 |
| | Position C — below cervical line | 3 |

**Total score interpretation:** 3–4 = minimally difficult · 5–6 = moderately difficult · 7–10 = very difficult.

Example: mesioangular (1) + Class II (2) + Position B (2) = **5 — moderately difficult**. Book standard chair time (30–45 min), local anesthesia adequate for an ASA I–II patient absent other risk factors.

## Rood's radiographic proximity-sign checklist

| Sign | Present? | Meaning |
|---|---|---|
| Darkening of the root where it crosses the canal | | Root may be grooved by or wrapped around the canal |
| Deflection/deviation of the canal path | | Canal has been displaced by root growth — strong proximity indicator |
| Interruption of the canal's white cortical line | | Cortical boundary between root and canal may be absent |
| Narrowing of the canal at the crossing point | | Canal compressed by adjacent root structure |

**Rule:** 0–1 signs → panoramic-based planning is usually sufficient. ≥2 signs → order CBCT before finalizing the surgical approach; a 2D film cannot resolve buccolingual position.

## Anesthesia-selection matrix

| ASA class | Procedure complexity | Default anesthesia level | Monitoring |
|---|---|---|---|
| I–II | Simple (Pederson 3–4) | Local only | Standard vitals |
| I–II | Moderate (Pederson 5–6) | Local + N2O/O2 | Pulse oximetry |
| I–II | Very difficult (Pederson 7–10) or high anxiety | Local + oral/IV moderate sedation | Pulse oximetry + continuous monitoring |
| III+ | Any | Requires physician clearance first; sedation level set jointly with clearing physician | Per clearance recommendation |

A high Pederson score alone does not justify escalating an ASA I patient past moderate sedation — complexity and medical status are evaluated independently, then the higher requirement governs.

## Post-op complication triage

1. **Pain onset and pattern.** Pain-free interval of 2–4 days followed by new throbbing pain, with a visibly empty socket on exam → alveolar osteitis (dry socket), not infection. Treat symptomatically; this is not typically an antibiotic indication.
2. **Numbness duration.** Beyond the expected duration of the local anesthetic used (roughly 6–8 hours for common agents) → document as a possible nerve injury, test and record sharp/dull and two-point discrimination, schedule early follow-up rather than the routine post-op visit.
3. **Swelling/trismus trajectory.** Expected post-surgical inflammation peaks around day 2–3 and then improves. Worsening after day 3, especially with fever or purulent drainage, points to infection rather than expected healing — re-evaluate, do not simply reassure.
