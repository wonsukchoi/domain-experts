# Playbook

## Shrink rule reference table by metal type (illustrative — always verify with current foundry/alloy-specific data)

| Metal | Typical linear shrinkage |
|---|---|
| Gray cast iron | ~1/8" per foot (~1.0%) |
| Ductile iron | ~1/8"-5/32" per foot (~1.0-1.3%) |
| Aluminum alloys | ~5/32"-3/16" per foot (~1.3-1.6%) |
| Steel (carbon) | ~1/4" per foot (~2.0-2.1%) |
| Bronze/brass | ~3/16"-1/4" per foot (~1.6-2.1%) |

**Rule:** always confirm the specific alloy's characterized shrinkage rate with the foundry — actual rates vary by exact alloy composition and casting geometry/section thickness, and this table is a general starting reference, not a substitute for foundry-specific data.

## Draft angle/parting line design checklist

1. Identify the pattern's withdrawal direction(s) for each mold section (cope, drag, and any additional parts).
2. Apply the minimum draft angle (commonly 1-3° depending on molding method and surface depth) to every surface parallel to the withdrawal direction.
3. Choose the parting line location considering: (a) draft direction compatibility on both sides, (b) core print access, (c) anticipated flash/finishing location.
4. Verify no surface requires withdrawal in a direction incompatible with the chosen parting line without additional pattern sections (loose pieces, split cores).
5. Document draft angles and parting line rationale in the pattern's design record.

## Core print sizing guide

1. Calculate the actual core's weight (volume × core sand density).
2. Size core print bearing surface area to adequately support that weight without excessive unit pressure (consult foundry engineering standards for the specific core sand and casting size).
3. Position core prints to resist expected metal flow forces during pour (accounting for pour direction and turbulence at the core location).
4. For heavier or larger cores, consider additional support features (chaplets) if core print alone is insufficient — but core print design should minimize reliance on chaplets where possible, since they can create their own defects if not managed correctly.
5. Verify core print design via a first-casting trial, checking wall thickness uniformity around the internal cavity to confirm the core didn't shift.
