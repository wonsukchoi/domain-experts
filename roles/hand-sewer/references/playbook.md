# Playbook

## Stitch-selection reference table by application type

| Application | Purpose | Recommended stitch |
|---|---|---|
| Hem finishing (visible garment) | Invisible from outside | Blind hem stitch |
| Closing an opening (lining, stuffed item) | Invisible closure | Slip stitch |
| Raw edge binding/reinforcement | Edge protection, some visibility acceptable | Whip stitch |
| Load-bearing seam repair | Strength primary, some visibility acceptable | Backstitch |
| Buttonhole ends, pocket corners | Stress-concentration reinforcement | Bar tack |
| Fine visible topstitching | Aesthetic, decorative accent | Running stitch or specified decorative stitch |

**Rule:** always identify the element's actual purpose (invisible/aesthetic, load-bearing, or reinforcement) before selecting from this table — don't default to a single familiar stitch regardless of purpose.

## Needle/thread matching guide by fabric weight

| Fabric weight/type | Needle size (approx.) | Thread weight |
|---|---|---|
| Fine wool, silk, delicate weaves | 70/10 or finer | Fine silk or fine polyester |
| Medium-weight cotton, standard garment fabric | 80/12 | Standard-weight polyester or cotton |
| Heavy fabric, denim, canvas | 90/14-100/16 | Heavy-duty polyester or upholstery thread |
| Leather (hand-sewn) | Specialized leather/glover's needle | Waxed thread appropriate to leather weight |

**Rule:** always test needle size on an inconspicuous scrap or seam allowance of the actual fabric before committing to visible stitching — fabric weave tightness varies even within a general weight category, and a needle appropriate for one similar-weight fabric can still leave visible holes in another.

## Reinforcement technique checklist for stress-concentration points

1. Identify all points on the item that will experience concentrated stress in normal use (buttonhole ends, pocket corners, strap attachment points).
2. Confirm the specific reinforcement technique required for each point type (bar tack, box stitch, or equivalent).
3. Apply reinforcement with adequate density/coverage — not a token stitch or two, but the full technique as specified.
4. Test reinforcement strength where practical (a gentle stress test on a repair before considering it complete).
5. Document which points received reinforcement and what technique was used, especially for a repair being handed back to a client with an explanation of what was addressed.
