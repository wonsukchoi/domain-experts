# Red flags

### A single default stitch type used regardless of whether the application needs invisible finishing, load-bearing strength, or reinforcement
- **Usually means:** the stitch's actual purpose wasn't identified before technique selection, risking a visually or functionally wrong result.
- **First question:** does this stitch type match the element's actual requirement (invisible, load-bearing, reinforcing), or was it chosen by habit?
- **Data to pull:** the element's functional/aesthetic requirement, stitch type used.

### Visible puckering or unevenness along a hand-sewn seam
- **Usually means:** inconsistent tension during stitching, a technique control issue rather than a material or thread problem.
- **First question:** does the unevenness correlate with specific sections where tension may have varied, or does it appear uniform?
- **Data to pull:** visual inspection of the seam, comparison of tension consistency across different sections.

### A needle size selected without testing on scrap/inconspicuous fabric for a fine or tightly woven material
- **Usually means:** risk of creating permanent, visible needle holes that can't be undone once made.
- **First question:** was the needle size tested on an inconspicuous area of this specific fabric first, or selected from general habit?
- **Data to pull:** scrap test result if performed, fabric weave/weight characteristics.

### A buttonhole, bar tack, or similar stress-concentration point sewn without the specific reinforcement technique
- **Usually means:** a hidden weak point that will fail under normal use stress, invisible until the item is actually used.
- **First question:** was the specific reinforcement technique for this element applied, or was it stitched as if it were a standard, non-stress-concentrated seam?
- **Data to pull:** the element's specified reinforcement requirement, technique actually used.

### A hand-sewn element serving both a visible and a load-bearing function, with a technique optimized for only one
- **Usually means:** the technique may satisfy appearance while under-delivering strength, or vice versa.
- **First question:** does this element need to be both invisible AND strong, and does the chosen technique actually satisfy both requirements?
- **Data to pull:** the element's dual requirements, technique's known strength/appearance trade-offs.

### A repair or alteration returning with the same failure at the same hand-sewn point
- **Usually means:** the original technique may not have matched the point's actual stress requirement, or reinforcement was inadequate.
- **First question:** was the original stitch type and reinforcement appropriate for this specific point's actual use stress?
- **Data to pull:** the original technique used, the point's function/stress profile.

### Thread weight/type selected without matching to the specific fabric's weight and the seam's functional requirement
- **Usually means:** either insufficient strength for a load-bearing application or an inappropriate, overly heavy appearance for a fine visible finish.
- **First question:** does the thread weight/type match both the fabric and the seam's actual functional requirement?
- **Data to pull:** fabric weight, thread specification used, seam's functional classification.

### A multi-stage hand-sewing project handed off between workers without specifying stitch type/needle/thread used
- **Usually means:** the incoming worker may use a different technique, creating inconsistency across the finished piece.
- **First question:** does the handoff specify the exact stitch type, needle size, and thread used so far?
- **Data to pull:** the handoff record, work completed so far.

### A fine fabric alteration performed with the same needle/thread combination used for a heavier, different fabric on a prior job
- **Usually means:** settings appropriate for one fabric weight carried over without adjustment for the current, different fabric.
- **First question:** does the current fabric's weight/weave match what the needle/thread combination was actually suited for?
- **Data to pull:** current fabric specification, needle/thread combination used.
