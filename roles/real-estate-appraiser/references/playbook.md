## Highest and best use analysis checklist

1. **Legally permissible:** What does current zoning and any pending zoning change allow for this site?
2. **Physically possible:** What does the site's size, shape, topography, and access support?
3. **Financially feasible:** Would a reasonably probable use generate a positive return relative to its development cost?
4. **Maximally productive:** Among the feasible uses, which produces the highest value?

**Use:** Run all four tests before concluding the current use is also the highest and best use — a property can pass "legally permissible" and "physically possible" for a higher use while the current improvement still sits well below that potential value.
