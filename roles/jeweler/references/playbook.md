# Playbook

## Casting shrinkage calculation (filled example)

14k yellow gold ring, target finished inner diameter 22.5mm, shrinkage rate 1.8%.

| Step | Value |
|---|---|
| Target finished diameter | 22.5mm |
| Alloy shrinkage rate | 1.8% |
| Wax pattern diameter | 22.5 × 1.018 = 22.91mm |
| Result if wax cast to naive 22.5mm dimension | 22.5 / 1.018 ≈ 22.10mm — undersized ~0.40mm |
| Ring size impact of 0.40mm undersize | ~0.8mm per half-size, so ~0.40mm ≈ roughly half a US ring size too small |

**Comparison across common karats/alloys (same 22.5mm target):**

| Alloy | Typical shrinkage rate | Wax pattern diameter |
|---|---|---|
| 14k yellow gold | 1.8% | 22.5 × 1.018 = 22.91mm |
| 18k yellow gold | 1.5% | 22.5 × 1.015 = 22.84mm |
| Sterling silver (.925) | 2.0% | 22.5 × 1.020 = 22.95mm |

## Metal reconciliation worksheet (filled example)

Custom ring job, 14k yellow gold.

| Input/output | Weight (grams) |
|---|---|
| Raw stock issued | 8.50g |
| Solder used | 0.15g |
| Findings (none for this job) | 0.00g |
| **Total in** | **8.65g** |
| Finished piece weight | 7.90g |
| Recovered scrap/sweeps | 0.68g |
| **Total accounted for** | **8.58g** |
| Unreconciled difference | 8.65 − 8.58 = 0.07g |
| Shop tolerance | ≤ 0.10g per job |

**Result:** 0.07g unreconciled falls within the shop's 0.10g tolerance — no investigation needed. If the difference exceeded 0.10g, the job would require checking sweep recovery, bench cleanup, and scale calibration before closing it out.

## Gemstone handling-sensitivity reference (filled example)

| Stone | Mohs hardness | Heat sensitivity | Ultrasonic-safe? | Notes |
|---|---|---|---|---|
| Diamond | 10 | Low | Yes | Generally tolerant of standard heat and ultrasonic cleaning |
| Sapphire/Ruby | 9 | Low | Yes | Similar tolerance to diamond for standard bench work |
| Emerald | 7.5–8 | High (fractures common) | **No — avoid** | Often has oil-filled fractures; heat and ultrasonic vibration can worsen or reveal them |
| Opal | 5.5–6.5 | High | **No — avoid** | Can crack from thermal shock or dehydrate from heat; avoid both heat and ultrasonic |
| Pearl | 2.5–4.5 | High | **No — avoid** | Organic material, easily damaged by heat, chemicals, and ultrasonic vibration |

Rule applied: default to the safest handling assumption (avoid heat/ultrasonic) unless the specific stone's profile confirms tolerance — never default to "standard cleaning" without checking the stone first.
