# Red flags

### Press parameters set by garment type or appearance rather than confirmed fiber content
- **Usually means:** an assumption about fabric composition may not match this specific garment's actual fiber content, risking damage.
- **First question:** has the garment's actual fiber content label been checked, or are parameters assumed from garment style?
- **Data to pull:** fiber content label/tag, press parameters used.

### A batch pressed at standard parameters without checking whether this specific style's fiber content matches the assumed standard
- **Usually means:** a style variation (a different blend or fiber for the same garment category) could receive incorrect parameters for the entire batch.
- **First question:** does this specific style/lot's fiber content match what the standard parameters assume?
- **Data to pull:** fiber content for this specific batch, standard parameter assumption.

### Visible shine, glazing, or scorching on a pressed garment, especially dark or synthetic fabric
- **Usually means:** over-pressing — excess heat, pressure, or dwell time for this fabric's actual tolerance.
- **First question:** do the parameters used match what's appropriate for this garment's actual fiber content?
- **Data to pull:** fiber content, parameters used, comparison to the fiber-specific reference chart.

### A garment positioned on the press form without verifying alignment before applying heat/pressure
- **Usually means:** a heat-set crease or pucker could be set in the wrong location, a defect difficult to reverse afterward.
- **First question:** was positioning verified before pressing, or was heat/pressure applied without a pre-press alignment check?
- **Data to pull:** positioning verification record if any, resulting garment inspection.

### An unfamiliar fabric or blend pressed at standard parameters without a test on an inconspicuous area first
- **Usually means:** an untested assumption about this fabric's safe parameters, risking irreversible damage on the full garment.
- **First question:** was a test performed on an inconspicuous area, or was the standard setting assumed safe without verification?
- **Data to pull:** test result if performed, fabric type/blend characteristics.

### Water spotting or visible moisture marks on a finished pressed garment
- **Usually means:** excess steam/moisture for this fabric's actual requirement.
- **First question:** does the steam/moisture setting used match what's appropriate for this fabric type?
- **Data to pull:** moisture/steam setting used, fabric type.

### An insufficiently creased or unfinished-looking press result
- **Usually means:** under-pressing — insufficient heat, pressure, dwell time, or moisture for this fabric to properly set.
- **First question:** were parameters set to the correct window for this fabric, or conservatively low without reaching the actual required set point?
- **Data to pull:** parameters used vs. the fiber-specific reference chart.

### A recurring pressing defect across multiple garments correlating with a specific fabric lot or press station
- **Usually means:** a systematic fiber content mismatch or equipment calibration issue, rather than isolated garment-to-garment variation.
- **First question:** do the affected garments share a common fabric lot, style, or press station?
- **Data to pull:** defect pattern by lot/station, fiber content consistency across the affected garments.

### A press temperature/pressure gauge not recently verified against actual output
- **Usually means:** the displayed setting may not reflect actual delivered heat/pressure, risking either under- or over-pressing despite a "correct" setpoint.
- **First question:** when was this press's temperature/pressure calibration last verified?
- **Data to pull:** equipment calibration log.

### A garment returned/reworked for a pressing defect without re-verifying fiber content and parameters before the rework attempt
- **Usually means:** the same root cause (fiber mismatch, wrong parameters) could produce the same defect again on rework.
- **First question:** was fiber content and correct parameter selection re-confirmed before attempting the rework press?
- **Data to pull:** rework record, fiber content/parameter verification for the rework attempt.
