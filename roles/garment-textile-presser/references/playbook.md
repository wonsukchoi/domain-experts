# Playbook

## Fiber-content-to-press-parameter reference table (illustrative — always use the shop's actual validated chart)

| Fiber content | Temperature | Steam level | Pressure | Dwell time |
|---|---|---|---|---|
| 100% Cotton | 350°F | Heavy | 12-14 psi | 8-10 sec |
| Poly-cotton blend (standard shop default) | 320°F | Medium | 12 psi | 8 sec |
| 100% Polyester | 280°F | Light | 12 psi | 6 sec |
| Wool/wool blend | 300-320°F | Medium-heavy | 10-12 psi | 8-10 sec |
| Delicate synthetic (nylon, some spandex blends) | 250-270°F | Light | 8-10 psi | 4-5 sec |

**Rule:** always verify actual fiber content against the label before selecting parameters — this table shows the direction and magnitude of typical differences, not a substitute for the shop's own validated, fiber-specific chart.

## Pre-press verification checklist

1. Confirm garment fiber content via label/tag before selecting press parameters.
2. Select parameters (temperature, steam, pressure, dwell) matched to the confirmed fiber content.
3. For unfamiliar fabric/blend, test on an inconspicuous inside-seam area first.
4. Position garment on the press form, verifying correct alignment before applying heat/pressure.
5. Run the press cycle at verified parameters.
6. Inspect the pressed result for glazing, scorching, insufficient crease, or water spotting before proceeding to the next garment in the batch.
7. Document fiber content, parameters used, and test results (if performed) in the batch quality record.

## Over/under-pressing defect diagnostic guide

| Symptom | Likely cause | Check |
|---|---|---|
| Shine/glazing | Over-pressing (excess heat/pressure/dwell for this fiber) | Fiber content vs. parameters used |
| Scorching | Significant over-pressing, likely a parameter error for this fiber | Fiber content, temperature setting |
| Insufficient crease/wrinkled appearance | Under-pressing (insufficient heat/pressure/dwell/moisture) | Parameters used vs. fiber-specific requirement |
| Water spotting | Excess steam/moisture for this fabric | Steam/moisture setting vs. fabric type |
| Crease/pucker in wrong location | Positioning misalignment before pressing | Positioning verification record |
