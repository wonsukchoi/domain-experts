# Red flags

### Marker layout accepted based on visual impression without calculating actual fabric utilization percentage
- **Usually means:** unrecognized fabric waste that compounds significantly across a production run.
- **First question:** what is the actual calculated marker efficiency percentage for this layout?
- **Data to pull:** total pattern piece area compared against total fabric area laid, calculated as a percentage.

### Ply height set to the cutting equipment's maximum physical capacity rather than the fabric-specific accuracy limit
- **Usually means:** dimensional distortion at lower layers in the stack, invisible from inspecting only the top layer.
- **First question:** what does the documented accuracy limit for this fabric thickness/equipment combination show against the ply height used?
- **Data to pull:** ply height used compared against the documented accuracy limit for this fabric thickness/equipment combination.

### Fabric cut immediately off the roll without allowing documented relaxation time
- **Usually means:** cut pieces may shrink or distort after cutting once the fabric reaches true dimensional equilibrium.
- **First question:** what was the actual relaxation time versus the fabric's documented requirement?
- **Data to pull:** actual relaxation time observed compared against the fabric's documented relaxation requirement.

### Marker laid for maximum fabric efficiency on directional/napped fabric without verifying consistent piece orientation
- **Usually means:** mismatched panel shading or an inverted directional pattern, a defect invisible until panels are assembled.
- **First question:** does the marker layout maintain consistent orientation across every pattern piece?
- **Data to pull:** piece orientation consistency check compared against the marker layout used, for directional fabric jobs.

### Cutting accuracy verified only by inspecting the top ply layer against the marker outline
- **Usually means:** a dimensional discrepancy at lower layers could go undetected.
- **First question:** what do direct measurements at multiple ply layers, including the bottom, show against specification?
- **Data to pull:** dimensional measurements at multiple ply layers compared against specification.

### A recurring dimensional discrepancy or fabric waste pattern for a specific style/fabric combination appears without a traced cause
- **Usually means:** a systemic process issue — ply height, marker efficiency, relaxation time — rather than isolated one-off run variation.
- **First question:** does the discrepancy/waste log correlate with a specific process parameter?
- **Data to pull:** discrepancy/waste occurrence log compared against process parameter logs (ply height, marker efficiency, relaxation time) over the same period.

### A new fabric type or style introduced without recalculating marker efficiency or reassessing ply height/relaxation requirements
- **Usually means:** parameters validated for a different fabric may not apply, producing unrecognized waste or dimensional issues.
- **First question:** were marker efficiency, ply height, and relaxation time reassessed for this new fabric?
- **Data to pull:** new fabric's characteristics compared against whether marker efficiency/ply height/relaxation were reassessed for it.
