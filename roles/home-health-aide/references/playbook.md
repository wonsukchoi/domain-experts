# Home Health Aide — Playbook

## Pre-visit plan-of-care check (per client, per visit)

| Item | Where to find it | Action if missing/unclear |
|---|---|---|
| Authorized tasks list | CMS-485 section on aide services | Call agency before performing anything not listed |
| Visit frequency/duration | CMS-485 | Flag any change in client need vs. authorized frequency to agency |
| Delegation orders (vitals, glucose checks) | POC addendum, signed by RN/MD | Treat as absent if not in writing — do not perform on a verbal OK |
| Diabetic/cardiac/fall-risk protocol | POC special-instructions section | If a client's condition suggests one should exist but doesn't, report the gap |

## Hypoglycemia symptom response (Rule of 15) — aide-scope version

| Step | Action | Timing |
|---|---|---|
| 1 | Recognize symptoms (shaky, sweaty, confused, irritable) in a client on insulin/sulfonylurea | On observation |
| 2 | Offer 15g fast-acting carbohydrate as ADL/feeding assistance (4oz juice, 3-4 glucose tabs, 1 tbsp honey) | Immediately |
| 3 | Call supervising RN/agency to report time of last dose, time of onset, action taken | Within minutes, before continuing the visit |
| 4 | Recheck symptoms | 15 minutes after carbohydrate given |
| 5 | If unresolved/worsening or client becomes unresponsive | Call 911 immediately, do not give anything further by mouth if unresponsive |
| 6 | If resolved, resume normal meal/medication schedule per RN instruction | As directed |

## Home-safety checklist (walkthrough, first visit and quarterly)

| Area | Hazard checked | Client response if flagged |
|---|---|---|
| Flooring | Loose rugs, exposed cords, clutter in walkways | Recommend removal/taping; document if client declines |
| Bathroom | Grab bars present at toilet/tub, non-slip mat, adequate lighting | Recommend installation via agency/family; document refusal |
| Kitchen | Expired/spoiled food, stove left on, smoke detector present and functional | Report to agency same-day if unsanitary conditions found |
| Stairs | Handrail present and secure, adequate lighting | Recommend repair; note whether client uses stairs regularly |
| Temperature | Home adequately heated/cooled for season | Report immediately if unsafe (e.g., no heat in winter) — self-neglect/utility-shutoff indicator |
| Medication storage | Pillbox matches expected daily count, expired medications present | Report discrepancy same-day, don't assume benign cause |

## Medication-reminder discrepancy log — filled example

```
Client's pillbox (7-day, AM/PM compartments) checked Monday visit:
Expected remaining (per POC, once-daily AM dose): 4 doses (Thu-Sun not yet taken)
Actual remaining: 6 doses
Discrepancy: 2 extra doses present — suggests 2 missed AM doses (Fri, Sat)
Action: reported to agency same visit, not carried to next visit for re-check
```

## Suspected neglect/abuse — reporting escalation (same-day, non-negotiable)

```
Observation: Client has 3 bruises on upper arms, bilateral, in finger-shaped pattern,
             client states "I don't remember," previously denied any falls this week.
Aide action:
  1. Complete the scheduled care tasks as normal — do not confront family present.
  2. Document objectively: location, size, color, pattern, client's exact words.
  3. Call agency supervisor same day (not end-of-week), state observation factually.
  4. Agency routes to Adult Protective Services per state mandatory-reporting statute.
  5. Aide does not independently contact APS unless agency is unreachable and state law
     requires direct reporting — follow agency's specific reporting protocol.
```
