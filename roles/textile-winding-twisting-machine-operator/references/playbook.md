**Rule:** always physically verify twist direction on each package before plying — never trust a label alone, since receiving/labeling errors are a real, catchable risk at this stage.

## Draw ratio consistency reference guide

1. Confirm target draw ratio (average) matches the yarn specification.
2. Monitor for tension/speed fluctuation during the drawing/drafting process, not just the setpoint value.
3. Periodically test yarn evenness (diameter consistency) along sample lengths using an evenness tester, not visual inspection alone.
4. If slubs or thin spots are found, investigate for roller slippage, tension fluctuation, or fiber feed inconsistency as the likely mechanical cause — not assumed as a raw fiber quality issue by default.
5. Document both average draw ratio achieved and evenness test results (variation along length) in the lot's quality record.
