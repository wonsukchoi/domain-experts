# Playbook

## Needle/seam-type selection table by fabric type

| Fabric type | Needle point | Typical needle size | Seam type | Notes |
|---|---|---|---|---|
| Woven, lightweight (e.g., cotton lawn) | Sharp | 65/9-70/10 | Straight lockstitch | Non-stretch; sharp point fine for woven |
| Woven, medium/heavy (e.g., denim, canvas) | Sharp, heavy-duty | 90/14-100/16 | Straight lockstitch, reinforced | Higher stitch density for load-bearing seams |
| Knit, lightweight (e.g., jersey) | Ballpoint | 70/10-80/12 | Overlock/serger | Sharp point risks cutting knit fibers |
| Knit, heavier (e.g., fleece, sweater knit) | Ballpoint, heavier | 80/12-90/14 | Overlock, or coverstitch for hems | Match needle weight to fabric bulk |
| Stretch/spandex blend | Stretch (specialty ballpoint) | 70/10-80/12 | Overlock or specialty stretch stitch | Standard ballpoint may still skip stitches on high-stretch content |

**Rule:** always verify against the current job's actual fabric specification — this table is illustrative, not a substitute for checking the bundle ticket and fabric in hand.

## Stretch test procedure

1. Cut or use a test seam sample representative of the current bundle's fabric and seam construction.
2. Mark a measured length across the seam (e.g., a 4" span).
3. Pull the sample to simulate normal wear stretch — for a side seam, typically 20-30% elongation; consult fabric/garment spec for the actual target.
4. Observe: does the seam hold without thread breakage, popped stitches, or visible seam separation?
5. If the seam fails at or below the target elongation, the seam type/thread/tension setup is inadequate for this fabric — do not proceed to production until corrected and re-tested.
6. Record the test result (pass/fail, elongation achieved) before starting full bundle production.

## Tension diagnostic table

| Symptom | Likely cause | Check |
|---|---|---|
| Loops on underside of fabric | Top tension too tight relative to bobbin, or bobbin too loose | Check bobbin tension directly, don't just loosen top |
| Loops on top of fabric | Bobbin tension too tight relative to top, or top too loose | Check top tension setting and thread path |
| Puckering along seam | Tension too tight overall, or stitch density too high for fabric weight | Check both tensions together; check stitch density against fabric spec |
| Skipped stitches | Often needle issue (wrong type, burred, bent) rather than tension | Inspect needle condition and type before adjusting tension |
| Thread breaking repeatedly | Could be tension, thread weight mismatch, or needle/machine mechanical issue | Check thread weight vs. fabric/needle, tension settings, needle condition in sequence |

## Bundle start checklist

1. Confirm fabric type/weight and seam specification against the bundle ticket.
2. Confirm needle type/size matches the fabric (change if different from prior bundle).
3. Confirm thread weight/type matches the fabric and needle.
4. Run a test seam; check tension balance (both sides) and stitch density.
5. For stretch fabric, run the stretch test procedure before proceeding.
6. Only after all checks pass, begin full bundle production.
7. Periodically re-check needle condition and spot-check stitch quality during the run, not only at bundle start.
