# Nursing Assistant — Playbook

## Shift vitals round — reportable-threshold quick table

| Vital | Normal adult range | Hard-report threshold | Note |
|---|---|---|---|
| Pulse | 60–100 bpm | <50 or >120 | Count for 30 sec x2 if irregular, don't estimate |
| Respirations | 12–20/min | <10 or >24 | Count discreetly — patients alter breathing if they know they're watched |
| SpO2 (room air) | ≥95% | <92% | Re-check probe placement/perfusion before reporting a surprising low |
| Systolic BP | 90–140 | <90 or >180 | Recheck with correct cuff size before reporting an outlier |
| Temp | 97.8–99.1°F | >100.4°F (38°C) or <96.8°F | Route (oral/temporal/axillary) must match the patient's care plan for trend comparison |

## Shift handoff — filled example

```
Room 214-B — [Resident], 84F, POD 3 s/p right hip ORIF, fall-risk care plan
6:00A: 78 / 16 / 128-76 / 97% / 98.2F — no issues, ambulated to bathroom assist-x2
10:00A: 84 / 18 / 122-74 / 96% / 98.6F — no issues, tolerated breakfast well
2:00P: 96 / 20 / 118-70 / 95% / 99.4F — trending, reported to RN [Nurse name] 2:04P,
       patient states "feels a little warm," denies new pain
ADLs: bathing assist-x1 AM, toileting assist-x2 all shift, ate 75% breakfast/50% lunch
Skin: intact, no new findings this shift
Fall risk: bed alarm on, no attempts to self-transfer noted
```

## ADL-assistance level tracking (per patient care plan)

| Task | Independence level today | Yesterday | Flag if |
|---|---|---|---|
| Bathing | Assist-x1 | Assist-x1 | Drops to assist-x2 or total-assist without explanation |
| Toileting | Assist-x2 | Assist-x1 | Any increase in assistance level — report as functional-change finding |
| Transfers | Assist-x2, gait belt | Assist-x2, gait belt | New unsteadiness, buckling, or refusal to bear weight |
| Feeding | Independent, setup only | Independent, setup only | New difficulty swallowing, coughing with liquids — stop feeding, report immediately (aspiration risk) |
| Ambulation | 50 ft with walker, assist-x1 | 100 ft with walker, assist-x1 | Distance drop >25% in one day |

## Fall-risk observation log (excerpt)

```
Patient on fall-risk care plan (bed alarm, hourly rounding, assist-x2 transfers).
0800: Found attempting to stand unassisted, bed alarm sounded, redirected, no fall.
1030: Attempted again, same pattern. Redirected, reinforced call-light use.
1300: Third attempt this shift. Reported pattern (not just individual events) to
      RN — care plan may need reassessment (increased confusion? new urge to void
      not being addressed by toileting schedule?).
```
The third redirection is the reportable event — the *pattern*, not just each individual incident logged in isolation.

## Skin-observation documentation (correct vs. incorrect)

- **Correct (factual, CNA scope):** "2cm x 1.5cm reddened area, non-blanchable, right heel, noted during AM care, patient repositioned, notified RN [name] at 0915."
- **Incorrect (diagnostic, outside scope):** "Stage 2 pressure ulcer forming on right heel."

The correct version gives the nurse everything needed to make the staging determination; the incorrect version substitutes a guess for the assessment.
