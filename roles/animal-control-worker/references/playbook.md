## Rabies-exposure quarantine decision table

| Biting animal | Vaccination status | Prior bite on file | Default quarantine | Notes |
|---|---|---|---|---|
| Dog / cat / ferret | Current | None | Home, 10-day observation | Owner signs quarantine agreement; officer inspects day 0 and day 10 |
| Dog / cat / ferret | Current | 1+ prior bite | Facility, 10-day observation | Elevated-risk track; owner cannot self-secure a repeat-risk animal |
| Dog / cat / ferret | Lapsed/unknown | Any | Facility, 10-day observation | No home-quarantine eligibility without current proof of vaccination |
| Dog / cat / ferret | Unvaccinated, exposed to confirmed-rabid animal | N/A | Euthanize immediately, or strict facility isolation 4 months (dogs/cats) / 6 months (ferrets) if owner declines euthanasia | Not the same protocol as a bite-to-human case; this is animal-to-animal exposure |
| Rabies vector species (raccoon, skunk, fox, bat, coyote) | N/A | N/A | Euthanize and submit head/brain for testing if captured; do not quarantine | No validated quarantine period exists for RVS — quarantine doesn't resolve the exposure question |

Vaccinated dog/cat/ferret exposed (not biting) to a confirmed-rabid animal: booster vaccination plus 45-day observation, per representative state compendium structure — verify against the current jurisdiction's compendium adoption before applying.

## Fee and fine calculation table (representative structure)

| Item | Rate | Notes |
|---|---|---|
| Facility quarantine board | $25–$30/day | Charged to owner unless indigent waiver applies |
| Home quarantine | $0 | Owner absorbs cost of securing animal; still requires signed agreement |
| At-large / bite citation (first documented incident) | $150–$250 | Separate from any dangerous-dog fee |
| At-large / bite citation (repeat, same address, 12 months) | 2× standard citation | Escalation for documented pattern |
| Dangerous-dog registration, annual | $100–$200 | Distinct from base license fee |
| Rabies booster (if lapsed) | $35–$50 | Charged when quarantine reveals lapsed vaccination |
| Liability bond/insurance (post-classification) | $100,000 minimum coverage | Ongoing requirement, not a one-time fee — verify local minimum |
| Secure enclosure inspection (post-classification) | Pass/fail, no fee typically | Must meet local containment spec (commonly 6-ft fence, self-latching gate) before release |

Fees are additive across categories (quarantine + citation + registration) unless the local ordinance states otherwise — do not treat one fee as covering the full exposure.

## Cruelty/neglect scene documentation checklist

1. Photograph the animal's body condition from multiple angles before any handling, timestamped.
2. Score body condition on the 9-point BCS scale (1 = emaciated, 9 = severely obese; 4–5 is ideal for most breeds) and note the score in the report, not just a narrative impression.
3. Photograph and note environmental conditions relevant to neglect (water access, shelter, waste accumulation, visible injury).
4. Document only after immediate medical crisis is addressed — if the animal needs emergency care, secure it first and document en route/at the clinic.
5. Identify and record the responsible party (owner, property manager) before removal, distinct from simply noting the address.
6. If removal occurs, note the specific legal basis (owner consent, exigent circumstances, warrant) in the same report as the BCS and photo documentation — the removal basis and the condition evidence have to travel together for a hearing.

A cruelty case built on photos and a BCS score alone, without a documented removal basis, is vulnerable to the same kind of challenge as a search without articulated authority.
