# Merchandise Displayer — Playbook

## Reset-planning table (by fixture type)

Filled example for a 42-fixture store reset:

| Fixture type | Count | Time each | Subtotal | Notes |
|---|---|---|---|---|
| Standard shelf | 34 | 15 min | 510 min | Restock + facing only |
| Endcap (promo grouping) | 8 | 45 min | 360 min | Multi-SKU + signage swap |
| **Total** | **42** | — | **870 min = 14.5 hrs** | |

Crew/window check:
| Input | Value |
|---|---|
| Closure window | 10:00pm–6:00am (8 hrs) |
| Setup/breakdown buffer | 1 hr |
| Usable execution window | 7 hrs |
| Crew size | 3 |
| Person-hours available | 21 |
| Person-hours needed | 14.5 |
| Slack | 6.5 person-hours (31%) |

Rule of thumb: don't schedule a reset below ~15% slack — deliveries, damaged fixtures, and signage errors routinely eat 1–2 hours.

## Capture-rate test-comparison template

Filled example — window concept A vs. B, 2-week matched test:

| Metric | Concept A (weeks 1–2) | Concept B (weeks 3–4) | Delta |
|---|---|---|---|
| Avg. passing traffic/day | 1,200 | 1,150 | -4.2% |
| Avg. entries/day | 96 | 115 | +19.8% |
| Capture rate | 8.0% | 10.0% | **+25.0% relative** |

Read: raw entry counts (+19.8%) understate the true effect because passing traffic dropped during B's window — capture rate isolates the display's contribution.

Minimum test length: 2 full weeks, to average across weekday/weekend traffic patterns. A test shorter than one full week-cycle risks attributing a single unusual day (e.g. a local event driving traffic) to the display concept.

## Planogram compliance audit checklist

1. Pull the planogram diagram for the zone under audit.
2. Walk the zone, noting every SKU location that doesn't match the diagram.
3. Compute compliance % = (SKUs correctly placed) / (total SKUs in zone).
4. Below 85%: schedule a targeted re-set of that zone within the week — don't wait for the next full reset cycle.
5. Log the drift pattern (which SKUs, which direction) — recurring drift toward the same wrong location usually means the planogram fights how staff naturally restock, and the planogram (not staff behavior) needs revising.

## Cross-merchandising adjacency check

Primary item + complementary item should sit within arm's reach (roughly 3 ft) of each other on the same fixture or an immediately adjacent one. A complementary display more than a few feet away, or on a different aisle, requires the shopper to make a second, independent trip and decision — attachment rate drops sharply past arm's reach.
