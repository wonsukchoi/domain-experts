# Track Maintenance Playbook

Filled worksheets for the three recurring jobs: sequencing a surfacing pass, setting a ballast shoulder/anchor pattern to territory type, and running the pre-release spot-check. Numbers are field-standard targets — confirm against the section's FRA track class and the carrier's own standards before executing.

## 1. Surfacing pass sequencing

Run in this order. Skipping or reordering steps is the single most common cause of a callback within the first month.

| Step | Action | Skip only if |
|---|---|---|
| 1 | Confirm on-track safety (foul time or watchman/lookout, ≥15-sec warning) | Never |
| 2 | Walk the section: ballast fouling (mud/fines at tie ends), anchor pattern vs. territory type, visible tie condition | Never — this is where the anchor-pattern and fouling defects get caught before they're buried under fresh ballast |
| 3 | Undercut / clean ballast if fouling found in step 2 | Ballast is clean and free-draining |
| 4 | Verify design reference (fresh chord/versine reading or recent geometry-car pass) before programming the tamper | Reference confirmed current within the last inspection cycle |
| 5 | Tamp (lift and line to the verified reference) | — |
| 6 | Anchor / box to the territory-required pattern (see table below) | Anchor pattern already verified compliant in step 2 |
| 7 | Ballast regulator pass: shoulder profile and crib fill | — |
| 8 | Dynamic stabilizer pass | Track will see >5,000 tons of traffic before next scheduled inspection and the delay to get a stabilizer isn't worth it — flag a temporary speed restriction instead |
| 9 | Re-measure against the same threshold that triggered the work order | Never |
| 10 | Log as-left condition, deferred items, and any speed-restriction recommendation | Never |

## 2. Anchor pattern by territory type

| Territory | Pattern | Anchors per 1,320 ft (660 ties @ 24") |
|---|---|---|
| Tangent, level, mixed/manifest traffic | Box-anchored every other tie (4/boxed tie) | 1,320 |
| Grade ≥1% descending with unit-train or heavy loaded traffic (braking-force concentration) | Box-anchored every tie (4/tie) | 2,640 |
| Approach to open-deck bridge (2 rail lengths, ~78 ft, each side) | Box-anchored every third tie within the approach zone | ~156 (within the 78-ft zone, both approaches) |
| CWR territory with high rail-temperature-swing exposure | Per carrier CWR plan — commonly every tie or every other tie depending on lay temperature and territory | Per carrier plan |

Field-count check: count anchors present in a 100-tie sample, divide by the required anchors-per-tie for the territory (4 if boxed, 2 if single-anchored one direction only), compare to 100%. Anything under 90% of required is a work-order item before the section is released, not a note for next time.

## 3. Ballast shoulder targets

| Location | Minimum shoulder width | Notes |
|---|---|---|
| Tangent track | ~6 in | Measured from tie end to top of ballast slope break |
| Curve, low (inside) rail shoulder | ~6 in | — |
| Curve, high (outside) rail shoulder | ~12 in | Outside shoulder carries the lateral resistance load on a superelevated curve — undersizing it here is the more consequential miss |
| Crib fill (between ties) | To top of tie, or to within 1 in of rail base, whichever is reached first | Underfilled cribs won't hold a tamp regardless of shoulder width |

## 4. Curve elevation cap check (after a lift)

Before locking in a design lift on superelevated curve work, check the resulting elevation against the class cap, not just the pre-work target:

| Track class | Maximum outside-rail elevation |
|---|---|
| Class 1–2 | 8 in |
| Class 3–5 | 7 in |

**Worked check:** design calls for lifting a Class 4 curve from a settled 5.25 in to a target 6.75 in. 6.75 in is under the 7 in Class 3–5 cap — proceed. If the same design target were 7.25 in on this class, the lift itself would create a new class-cap violation; the correction has to be capped at 7 in and the shortfall handled as a permanent slower-speed condition or an elevation-plan revision, not executed as designed.

## 5. Pre-release spot-check worksheet

Re-measure the same parameter that triggered the work order, at the same location, using the same method:

- Crosslevel: re-check within the 62-ft window at the flagged station. Pass requires under 1.5 in difference where elevation is ≥6 in (or the section's class limit where elevation is lower).
- Profile: re-run the chord/versine check at the flagged midordinate station.
- Alignment: re-run a string line or geometry-car confirmation if the defect was alignment, not surface.
- Gauge: hand-gauge at the flagged tie plus five ties each direction.

Record the re-measured value next to the original exception value in the work log — "corrected" without the number is not a closed item.
