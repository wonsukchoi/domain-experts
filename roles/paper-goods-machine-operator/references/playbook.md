## Web tension-by-roll-diameter reference table (illustrative — always use the machine's specific tension profile)

| Roll diameter (relative to full) | Tension adjustment |
|---|---|
| 100-75% (full to 3/4) | Standard baseline tension |
| 75-50% | Slight taper down, per machine's automated profile or manual schedule |
| 50-25% | Further taper — smaller diameter means same tension = higher relative stress |
| Below 25% (near empty) | Lowest tension setting in profile; monitor closely for wrinkle/misfeed as web nears end |

**Rule:** always use the specific converting line's established tension profile (automated or manual schedule) — this table shows the general direction (taper down as diameter decreases), not universal values.

## Adhesive bond verification checklist

1. Confirm adhesive type, quantity setting, and application temperature match the specification for this paper/adhesive combination.
2. Apply adhesive per the run's standard settings.
3. Pull a sample and perform a peel/pull test at the specified interval (e.g., every X units or per shift) — not a visual check alone.
4. If bond strength fails the test threshold, check quantity and temperature settings before assuming an adhesive material issue.
5. Re-test after any adjustment before resuming full production.
6. Document bond test results (not just "glue applied") in the run's quality record.
