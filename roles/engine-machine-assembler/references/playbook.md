# Playbook

## Torque sequence worksheet

Fill for any multi-bolt joint with a specified staging pattern.

| Field | Value |
|---|---|
| Joint | Compressor Main Bearing Cap |
| Final torque spec | 85 ft-lb |
| Sequence pattern | Alternating cross-pattern |
| Stage 1 (30%) | 25.5 ft-lb, all bolts in pattern order |
| Stage 2 (70%) | 59.5 ft-lb, all bolts in pattern order |
| Stage 3 (100%) | 85 ft-lb, all bolts in pattern order |
| Confirmed each stage completed on all bolts before proceeding? | Y |

**Rule:** never take an individual bolt directly to final torque before the other bolts in the pattern have reached their corresponding stage — this defeats the purpose of staged sequencing.

## Bearing clearance measurement guide

| Field | Value |
|---|---|
| Clearance spec | 0.0015"-0.0025" |
| Measurement method | Plastigauge |
| Measured clearance | 0.0031" (out of spec, 24% over max) |
| Free-rotation check (secondary, not sufficient alone) | Rotated freely — does NOT confirm in-spec clearance |
| Root cause investigation | Shim/shell/bore stack-up |
| Corrective action | Reselect shim/shell combination |
| Re-measured clearance | 0.0021" (within spec) |

**Rule:** free rotation or "parts go together" is never sufficient confirmation of correct clearance — always use the specified direct measurement method.

## Sub-assembly/break-in verification checklist

1. Identify all sub-assemblies that can be independently tested before final integration.
2. Test each sub-assembly per its specification before integrating it into the next level of assembly.
3. Document sub-assembly test results before proceeding to integration.
4. After final assembly, follow the complete specified break-in procedure (duration, load profile) without compression or skipping.
5. Verify performance parameters (temperature, vibration, output) at the end of break-in before releasing the machine for full-load service.
6. Document break-in completion and post-break-in verification results in the machine's quality record.
