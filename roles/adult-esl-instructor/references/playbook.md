# Adult ESL/Basic Education Instructor — Playbook

| EFL | CASAS Reading scaled score range |
|---|---|
| Beginning ESL Literacy | ≤190 |
| Low Beginning ESL | 191–200 |
| High Beginning ESL | 201–210 |
| Low Intermediate ESL | 211–220 |
| High Intermediate ESL | 221–235 |
| Advanced ESL | 236–245 |
| Exiting to ABE | >245 |

Stated reference ranges from published NRS/CASAS technical-assistance crosswalks — confirm against the current program year's guide before reporting, since ranges are periodically revised.

## Standalone module template

| Module | Entry skill required | Real-world task anchor | Re-entry note |
|---|---|---|---|
| Reading a pay stub | None (High Beginning ESL+) | Identify gross pay, deductions, net pay on a real sample stub | Can start mid-term; no prior module needed |
| Calling in sick to work | None (High Beginning ESL+) | Phone script + voicemail practice | Can start mid-term |
| Phonics: consonant blends | Alphabet recognition | Foundational literacy skill, no direct task anchor | Sequential — requires the prior alphabet unit |
| Citizenship interview Q&A | Low Intermediate ESL+ | 10 sample USCIS civics questions | Can start mid-term |
| Reading a lease agreement | High Beginning ESL+ | Identify rent, deposit, notice-period clauses on a real sample lease | Can start mid-term |

Rule of thumb: if a learner could plausibly need the skill in isolation (a single doctor's appointment, a single job interview), build it standalone. Only sequence content where the second unit is mechanically impossible without the first (decoding before fluent reading, not "unit 5 vocabulary before unit 6 vocabulary").

## Attendance check-in thresholds

| Trigger | Action | Script opener |
|---|---|---|
| <60% attendance over rolling 4 weeks | Phone/text check-in within 3 days | "I noticed you've missed a few classes — is everything okay with your schedule?" |
| 3+ consecutive missed sessions | Same-week check-in, offer a standalone re-entry module | "No worries about missing — here's a module you can join right where we are." |
| Learner states goal achieved (e.g., got the job) | Document as a goal-achieved exit, not attrition | "That's great news — want to talk about whether you'd like to keep building your English, or is this a good stopping point?" |

## Post-test readiness check

Before administering a post-test, confirm: (1) learner has logged at least ~40 contact hours since intake per current NRS guidance, (2) the test form differs from the intake form (avoid re-administering the identical form), (3) the testing window falls before the funding-cycle reporting deadline with enough buffer to re-test a learner who was tested too early. Skipping (1) is the single most common cause of an understated program-wide gain rate.
