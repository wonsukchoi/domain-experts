# Orthodontist — Playbook

## Space analysis worksheet (filled example)

| Measurement | Value | Source |
|---|---|---|
| Arch length available (ALA), mandibular | 68.0 mm | Digital model, molar-to-molar arc length |
| Tooth-size sum required (TSA) | 74.0 mm | Mesiodistal width sum, remaining unerupted/erupting teeth |
| Raw discrepancy (ALA − TSA) | −6.0 mm | Calculated |
| Leeway space available (both sides) | +4.0 mm | Moyers average, confirmed by panoramic — primary molars/canines not yet drifted |
| Adjusted discrepancy | −2.0 mm | Calculated |
| Expansion capacity (slow expander, both sides) | +1.5 to 2.0 mm | Appliance-specific estimate |
| IPR capacity (4 mandibular incisor contacts, 0.25mm each) | +1.0 mm | Standard IPR limit per contact |
| Total resolvable space | +2.5 to 3.0 mm | Sum |
| **Verdict** | **Non-extraction feasible** (2.5-3.0mm ≥ 2.0mm needed) | |

Rule: never finalize an extraction decision from raw discrepancy alone. Always check leeway space (mixed dentition), profile analysis, and growth stage first.

## CVM staging reference (cervical vertebral maturation)

| Stage | Growth status | Growth-modification window |
|---|---|---|
| CVM 1 | Pre-pubertal | Window open, growth-modification highly responsive |
| CVM 2 | Pre-peak | Window open, good candidate for growth-modification start |
| CVM 3 | Peak growth | Last strong window — start growth-modification now if indicated |
| CVM 4 | Post-peak, decelerating | Window closing — growth-modification response diminishing |
| CVM 5-6 | Growth complete | Window closed — treatment is camouflage or surgical, not growth-modification |

Read from a lateral cephalometric radiograph, cervical vertebrae C2-C4 morphology. A patient assessed as CVM 4 or later with a skeletal Class II discrepancy is no longer a growth-modification candidate — reset the treatment-planning conversation around camouflage mechanics or a surgical referral, not appliance selection.

## Retention protocol template

| Element | Default | Adjust when |
|---|---|---|
| Retainer type | Vacuum-formed (clear) for aligned arches, Hawley if bite adjustment still settling | Bonded lingual retainer for high-relapse-risk cases (large rotations corrected, extraction cases, adult patients) |
| Wear schedule, months 1-6 post-debond | Full-time except eating/brushing | Reduce to nights-only sooner if compliance excellent and 3-month check shows stability |
| Wear schedule, month 6 onward | Nights only, indefinitely | Never fully discontinue — periodontal ligament remodeling and mesial drift continue lifelong |
| Review interval | 3, 6, 12 months post-debond, then annually | Sooner if any relapse signs noted at review |
| Bonded retainer check | Every recall visit | Replace immediately if any wire segment debonds — even one tooth unsplinted can shift within weeks |

## Anchorage planning example

Case: retracting both maxillary canines the full first-premolar extraction space (≈7mm each side) to close space after extraction.

- Molar-only anchorage: expect 2-3mm of unwanted molar mesial drift per side without reinforcement — a moderate-to-high anchorage-demand case.
- Anchorage plan: place a TAD between the second premolar and first molar on each side; retract canines against the TAD rather than the molar.
- Result: molar position preserved within 0.5mm, full extraction space closes at the canine as planned — anchorage loss essentially eliminated versus the molar-only approach.

Rule: classify anchorage demand (low/moderate/high) at the treatment-planning stage, before mechanics start — not after molar drift is already visible at a progress visit.
