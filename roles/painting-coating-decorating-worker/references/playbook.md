## Flash-time reference table (illustrative — always use the specific material's technical data sheet)

| Condition | Flash time adjustment |
|---|---|
| Within validated range | Standard specified flash time |
| Below validated temperature range | Extend flash time — slower solvent evaporation |
| Above validated humidity range | Extend flash time — slower solvent evaporation, humidity competes with solvent release |
| Above validated temperature range | May need to shorten window slightly, or watch for accelerated skinning — consult material data sheet |

**Rule:** always confirm against the specific coating material's technical data sheet — this table shows the general direction of adjustment, not specific values, which are material-dependent.

## Decorative consistency comparison checklist

1. Confirm a fixed reference sample or documented specification exists for the decorative/cosmetic standard being matched.
2. At the specified sampling interval (e.g., every Nth unit), compare the current unit directly against the fixed reference — not against the immediately preceding unit.
3. If a deviation is found, check whether it's an isolated unit issue or reflects a gradual drift by reviewing several recent units together.
4. If drift is detected, recalibrate technique/materials against the fixed reference before continuing, rather than continuing to compare against recently-drifted units.
5. Document comparison results and any corrective recalibration in the production quality record.
