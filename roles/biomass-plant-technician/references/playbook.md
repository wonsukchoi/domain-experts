# Playbook

## Fuel-moisture-to-combustion-parameter adjustment worksheet

Fill for any new fuel delivery or batch change.

| Field | Value |
|---|---|
| Design basis moisture | 30% |
| Actual tested moisture (this delivery) | 42% |
| Deviation | +12 points |
| Baseline air/fuel ratio | Design-basis setting |
| Adjusted air/fuel ratio | Increased to compensate for wetter fuel |
| Baseline feed rate | Design-basis rate |
| Adjusted feed rate | Reduced to reflect lower effective heating value |
| Projected steam output (unadjusted parameters) | Would risk incomplete combustion, elevated CO |
| Projected steam output (adjusted parameters) | ~46,000 lb/hr (vs. 50,000 rated) |
| CO emissions (unadjusted, projected) | ~180 ppm (exceeds 150 ppm permit limit) |
| CO emissions (adjusted, actual) | ~58 ppm (within limit) |

**Rule:** never assume a new fuel delivery matches the design basis moisture content — sample and test, then adjust combustion parameters proportionally rather than holding fixed settings.

## Bridging/jamming intervention checklist

1. Confirm the interruption is a mechanical flow issue (fuel not moving) rather than a combustion-side problem (check fuel presence/flow sensors first).
2. Identify the specific location of the blockage (hopper, feeder, conveyor transition point).
3. Apply the appropriate physical intervention (mechanical agitation, vibration, or manual clearing per the equipment's designed intervention method) — do not attempt to force flow with increased mechanical force beyond equipment rating.
4. Once cleared, check whether the fuel batch's characteristics (moisture, particle size) suggest an elevated bridging tendency requiring closer monitoring going forward.
5. Document the bridging event, location, intervention used, and any fuel characteristic correlation for pattern tracking.

## Slagging/fouling monitoring guide

1. Monitor steam temperature and heat transfer efficiency trends continuously, not just at scheduled maintenance points.
2. If trends indicate faster-than-expected fouling, increase soot-blowing frequency proactively rather than waiting for the next scheduled cleaning.
3. Correlate fouling rate with specific fuel batches/sources where possible, to identify if certain sources consistently drive more aggressive fouling.
4. Document actual observed fouling rate against the standard schedule's assumption, feeding this back into schedule adjustments over time.
5. Never rely solely on a fixed calendar schedule when trend data indicates a different actual fouling rate for current fuel conditions.
