# Occupational Therapist — Playbook

## ADL/IADL scoring table (filled example)

| Task | Independence level | Assistance type/amount | Notes |
|---|---|---|---|
| Feeding | Independent | None | — |
| Grooming | Independent | None | — |
| Upper-body dressing | Independent | None | — |
| Lower-body dressing | Moderate assistance → Independent | Right-side weakness, balance in standing → resolved with adaptive equipment | Reacher, long-handled shoehorn added session 8 |
| Bathing/shower transfer | Moderate assistance → Independent | Tub-wall clearance, standing tolerance → resolved with equipment | Transfer bench + grab bar, remounted after home-photo review |
| Toileting | Independent | None | — |
| Cooking (stove-to-sink task) | Unsafe (2 missed safety steps) → Independent | Sequencing/task-monitoring deficit → resolved with cueing strategy | Written checklist posted at point of use |
| Medication management | Not yet assessed | — | Scheduled for session 14 |

Score each task from directly observed performance, never from self-report or family report alone — self-report runs optimistic relative to observed performance by a well-documented margin, especially in the first 1–2 weeks post-injury.

## Home-assessment documentation template (filled example)

```
HOME ENVIRONMENT ASSESSMENT — [Patient name], [date]
Source: photos submitted by family (bathroom, bedroom, entry stairs), verified by phone walkthrough.

Entry: 3 steps, no handrail on right (stronger) side — RECOMMEND: temporary rail
  install before discharge, or use of left-hand support with reacher-assisted
  door management.
Bathroom: tub with tiled ledge, ledge height measured at 22" from floor via
  photo scale reference — grab bar spec (34" standard height) does NOT clear
  the tiled ledge; remount at 38" confirmed against photo.
Bedroom: bed height 24", within standard transfer range — no equipment needed.
Kitchen: stove is gas with manual dial (not push-button auto shutoff) — cueing
  strategy (posted checklist) required; NOT a candidate for "the appliance
  will handle it" assumption.

Equipment ordered: transfer bench, wall-mounted grab bar (38" per measured
  spec, NOT the 34" catalog default), reacher, long-handled shoehorn.
Equipment NOT ordered pending in-person confirmation: raised toilet seat
  (verbal report only, no photo — hold until home visit or additional photo).
```

Never finalize a permanent-installation recommendation (grab bar, ramp) from a verbal description alone. A photo with a scale reference (a common object of known size, or the patient's own tape-measure reading) is the minimum bar; an in-person home visit is preferred whenever the discharge timeline allows it.

## Return-to-work capacity comparison table (filled example)

| Job physical demand | Required capacity (per job description) | Patient measured capacity | Gap? |
|---|---|---|---|
| Lifting, floor to waist | Up to 35 lbs, occasional | 25 lbs, occasional (FCE measured) | Yes — 10 lb gap |
| Lifting, waist to shoulder | Up to 20 lbs, occasional | 20 lbs, occasional | No |
| Standing tolerance | 6 hrs/8-hr shift | 4 hrs before fatigue-related form breakdown | Yes — 2 hr gap |
| Fine motor (right hand) | Repetitive, frequent | Full, no deficit | No |
| Stair use | Occasional, no rail required | Independent with rail, unsafe without | Yes — conditional |

A gap in any row is a specific, quantified restriction to state in the return-to-work recommendation (e.g., "lifting restricted to 25 lbs until re-evaluation," not "light duty"). "Light duty" without quantified restrictions is not enforceable by the employer and routinely gets ignored or misapplied.
