# Playbook

## Heat-sensitive grinding checklist (hardened/tempered material)

1. Confirm the material is hardened/tempered and identify its hardness specification before starting.
2. Use light, intermittent contact — brief passes with pauses — rather than sustained continuous grinding.
3. Periodically touch-check the workpiece temperature (too hot to comfortably touch = stop and let cool before continuing).
4. Do not rely on the absence of visible discoloration (straw/blue tempering colors) as confirmation that no heat-affected softening occurred.
5. Complete stock removal to spec using this controlled approach, even if it takes meaningfully longer than a continuous-contact approach.
6. For any hardness-spec-critical surface, flag the part for hardness verification testing before release — don't rely on dimensional/visual pass alone.
7. Document the process used (intermittent contact, temperature checks performed) and hardness verification result.

## Grit progression reference table (illustrative — always use the job's specified sequence)

| Stage | Typical grit | Purpose |
|---|---|---|
| 1 | 120-180 | Remove major stock, initial shaping |
| 2 | 220-320 | Remove coarse-stage scratch pattern |
| 3 | 400-600 | Refine surface, remove stage 2 scratches |
| 4 | 800-1200 | Final pre-polish refinement |
| 5 (if specified) | Polishing compound | Achieve final specified Ra/appearance |

**Rule:** never skip a stage in the sequence — each stage exists specifically to remove the previous stage's scratch pattern. Confirm the job's actual specified sequence rather than assuming this illustrative table applies directly.

## Media dedication tracking guide

| Metal type | Dedicated media required? | Cross-contamination risk if shared |
|---|---|---|
| Carbon/mild steel | Yes — separate from stainless | Iron embedding into stainless → surface rust |
| Stainless steel | Yes — dedicated, never shared with carbon steel | Same as above, reversed direction of concern |
| Aluminum | Yes — separate from steel types | Galvanic/staining issues, embedded steel particles |
| Titanium | Yes — dedicated, avoid any iron contact | Contamination can affect corrosion resistance and, in some applications, fatigue properties |

**Rule:** mark and physically separate media by dedicated metal type at the workstation — don't rely on memory or "it looks clean" as sufficient assurance between different materials.

## Dimensional check frequency guide (tight-tolerance hand finishing)

| Tolerance band | Suggested check frequency during process |
|---|---|
| ±0.001"-0.003" (tight) | Check every few passes, not just at completion |
| ±0.005"-0.010" (moderate) | Check at meaningful process milestones (e.g., halfway, near completion) |
| Looser than ±0.010" | Standard end-of-process check typically sufficient |

**Rule:** the tighter the tolerance, the more frequently dimensional checks should occur during the process — hand-grinding's inherent unevenness has less margin to hide in on a tight-tolerance part.
