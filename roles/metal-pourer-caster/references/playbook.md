# Playbook

## Treatment fade tracking worksheet

Fill for any melt receiving inoculation, degassing, or similar time-sensitive treatment.

| Field | Value |
|---|---|
| Casting | Ductile Iron, Pour #DI-6604 |
| Treatment | In-stream inoculation at tap |
| Specified fade window | 8-10 minutes |
| Treatment timestamp | record actual |
| Actual pour timestamp | record actual |
| Elapsed time | calculate |
| Within fade window? | Y/N |
| Pour temperature at point of pour | 2,580°F (spec min 2,500°F) |
| If fade window exceeded: follow-up testing required | Nodularity/metallographic check |
| Test result | 78% (spec min 85%) |
| Disposition | Hold for engineering review |

**Rule:** track fade window elapsed time independently of temperature — a melt can be at perfect pour temperature while treatment benefit has already faded.

## Pour readiness checklist

1. Confirm ladle/tool is fully dry — verify directly, don't assume based on storage or prior use.
2. Inspect ladle refractory condition — check for cracking, excessive wear, buildup — before this specific use.
3. Verify melt temperature at the point of pour (not just furnace tap), especially after any delay.
4. If a time-sensitive treatment was applied, confirm elapsed time is within the specified fade window.
5. Confirm pour rate/technique matches the specific mold/gating design's intended fill rate.
6. Only after all checks pass, proceed with the pour.
7. Document all readiness checks (dryness, refractory, temperature, treatment timing) before or immediately after the pour.

## Pour rate reference guide by gating design type (illustrative — always use the specific mold's design specification)

| Gating design | Typical pour rate characteristic |
|---|---|
| Simple, open top-pour mold | Faster fill generally tolerated; less turbulence sensitivity |
| Bottom-gated system | Controlled, moderate rate to avoid turbulence at the gate |
| Multiple-gate/complex cavity | Rate matched to fill balance across gates — uneven pouring can cause differential fill timing |
| Thin-wall/intricate casting | Slower, more controlled rate to avoid premature solidification (cold shut) while still avoiding turbulence |

**Rule:** always use the specific mold/gating design's actual specification — this table is illustrative of general tendencies, not a substitute for process-specific pour rate data.
