# Auto Damage Insurance Appraiser — Playbook

## Total-loss formula calculation (filled example)

| Step | Value |
|---|---|
| Initial repair estimate | $6,860 |
| Approved supplement (frame/structural, teardown-documented) | $1,970 |
| **Cumulative repair estimate** | **$8,830** |
| ACV (3-comparable local market analysis) | $9,500 |
| State TLF threshold percentage | 75% |
| Threshold dollar amount (75% × $9,500) | $7,125 |
| **Result** | $8,830 > $7,125 → **Total loss** |

**Use:** Re-run this table every time a supplement changes the cumulative estimate — a vehicle can flip from repairable to total loss mid-repair, and the formula has to be checked again, not assumed to still hold from the first pass.

## ACV comparable-vehicle worksheet (filled example)

| Comparable | Year/Trim | Mileage | Asking/Sold price | Adjustment | Adjusted value |
|---|---|---|---|---|---|
| 1 | 2019 sedan, same trim | 55,000 mi | $9,800 | -$100 (higher trim option not on subject vehicle) | $9,700 |
| 2 | 2019 sedan, same trim | 61,000 mi | $9,200 | +$200 (lower mileage than comparable adjustment) | $9,400 |
| 3 | 2018 sedan, one trim below | 52,000 mi | $9,600 | -$100 (trim difference) | $9,500 |
| **Average adjusted ACV** | | | | | **$9,533 → rounded to $9,500** |

**Use:** Adjust each comparable for material differences (trim, mileage, options) before averaging — using raw asking prices without adjustment overstates or understates the subject vehicle's actual value.

## Supplement evaluation checklist (per line item)

1. Does this line item have teardown or inspection photos showing the specific damage?
2. Is the item's cost consistent with the estimating platform's reference pricing for this part/operation?
3. If labor hours diverge from book time, is there a documented condition (corrosion, prior repair, access difficulty) explaining the difference?
4. Does the supplement, combined with prior approvals, change the total-loss formula result? (Re-run the table above if cumulative total is now closer to or over threshold.)

**Use:** Evaluate and approve/deny each supplement line individually — approving or denying an entire supplement request as a lump sum skips the evidence check that makes the approval defensible later.

## Parts-sourcing decision table

| Vehicle age | Part type | Default sourcing | Notes |
|---|---|---|---|
| Under ~3 years (state-specific cutoff) | Structural/safety-critical | OEM | Many states restrict aftermarket on newer vehicles for these parts |
| Under ~3 years | Cosmetic (fender, bumper cover, non-structural trim) | OEM or aftermarket, per state disclosure rule | Disclosure to insured typically required if non-OEM used |
| Over ~3 years | Structural/safety-critical | OEM preferred; aftermarket if state permits and fit/quality verified | |
| Over ~3 years | Cosmetic | Aftermarket or recycled (LKQ) acceptable | Standard cost-saving choice absent insured objection |

**Use:** Check the specific state's parts-disclosure statute for the exact age cutoff and part-class rules before finalizing sourcing — the ~3-year figure here is a commonly cited reference point, not a universal rule.

## Total-loss determination memo — filled example

> **Total Loss Determination — Claim [ref]**
> Initial estimate: $6,860 | Approved supplement (frame/structural, teardown-documented): $1,970 | Revised total: $8,830
> ACV (3-comparable market analysis): $9,500 | State TLF threshold (75%): $7,125
> $8,830 exceeds $7,125 — **Total loss declared.**
> Salvage value (auction bid comparable): $2,100 | Deductible: $500
> **Payout to insured: $9,000. Net cost to insurer after salvage recovery: $6,900.**
