## SBTi trajectory validation (filled example)

| Component | Value |
|---|---|
| Base year | 2023 |
| Target year | 2030 |
| Period | 7 years |
| Required minimum annual linear reduction (near-term, 1.5°C-aligned) | 4.2%/year |
| Cumulative required reduction: 1 − (1 − 0.042)⁷ | 1 − 0.744 = **25.6%** |
| Proposed target | 20% reduction by 2030 |
| **Result** | 20% < 25.6% required → **target does not meet SBTi trajectory, needs revision** |

**Use:** Always run this calculation before accepting a proposed target as "science-based" — a target's percentage alone doesn't tell you whether it clears the required trajectory for its specific base year and timeline.

## Materiality assessment scoping checklist

1. Identify which reporting framework(s) apply to this disclosure (GRI, SASB, TCFD/ISSB, CSRD/ESRS).
2. Check whether the applicable framework requires single (financial) or double (financial + impact) materiality.
3. If double materiality is required, run both lenses separately and document each topic's rating under both.
4. Prioritize disclosure topics based on the materiality assessment's actual output, not a predetermined list of "typical" ESG topics.

## Emission factor selection checklist

1. Identify the activity being calculated (fuel type, electricity, purchased goods category).
2. Check for the most geographically specific factor available (regional/national grid factor vs. generic global average).
3. Check for the most temporally current factor available (current or most recent published year vs. an older vintage).
4. Document the source and vintage of every emission factor used, for auditability.
