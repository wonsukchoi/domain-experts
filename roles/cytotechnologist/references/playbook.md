# Playbook — filled screening procedures a cytotechnologist actually runs

## 1. Gynecologic adequacy assessment, filled (liquid-based prep)

```
SPECIMEN: ThinPrep Pap, cervical
CELLULARITY CHECK (10-field count-and-extrapolate method):
  Field 1: 42 squamous cells    Field 6:  48 squamous cells
  Field 2: 51 squamous cells    Field 7:  40 squamous cells
  Field 3: 39 squamous cells    Field 8:  47 squamous cells
  Field 4: 44 squamous cells    Field 9:  43 squamous cells
  Field 5: 46 squamous cells    Field 10: 50 squamous cells
  Average per field: 45.0
  Estimated fields across 20mm monolayer prep area: ~180
  ESTIMATED TOTAL: 45 x 180 = 8,100 squamous cells
  THRESHOLD: >= 5,000 (liquid-based) -> ADEQUATE

OBSCURING FACTORS: blood/inflammation obscuring < 25% of epithelial cells -> does not affect adequacy
  (threshold: >75% obscured on liquid-based prep -> unsatisfactory)

ENDOCERVICAL/TRANSFORMATION ZONE (TZ) COMPONENT:
  Cluster identified: 14 well-preserved endocervical cells
  THRESHOLD: >= 10 cells (or 2 groups of >=5) -> TZ COMPONENT PRESENT

RESULT: Satisfactory for evaluation, TZ component present.
```

Conventional smear uses the same structure but the cellularity threshold is 8,000-12,000 squamous cells, and obscuring-factor tolerance is tighter (>50% obscured -> unsatisfactory) because there is no processing step to remove blood/mucus the way liquid-based prep does.

## 2. Bethesda category decision table (gynecologic)

| Finding on the slide | Category | Sign-out authority |
|---|---|---|
| No epithelial abnormality; may include organisms (Trichomonas, Candida, bacterial vaginosis/shift, Actinomyces, HSV) or reactive changes (inflammation, atrophy, IUD) | NILM (Negative for Intraepithelial Lesion or Malignancy), with organism/reactive-change subcategory noted | Cytotechnologist, independent |
| Nuclear enlargement 2.5-3x normal intermediate nucleus, mild contour irregularity, mild-to-moderate hyperchromasia, N:C ratio still low | ASC-US (Atypical Squamous Cells of Undetermined Significance) | Refer to pathologist; reflex HPV per protocol |
| Atypia with features raising concern for HSIL but not fulfilling all criteria | ASC-H (cannot exclude HSIL) | Refer to pathologist |
| Nuclear enlargement 3x+ normal, well-defined perinuclear halos (koilocytosis), binucleation | LSIL (Low-grade Squamous Intraepithelial Lesion) | Refer to pathologist |
| High N:C ratio, coarse irregular chromatin, irregular nuclear contours, smaller/less mature cells than LSIL | HSIL (High-grade Squamous Intraepithelial Lesion) | Refer to pathologist |
| Glandular-cell nuclear enlargement/overlap, feathering, rosette formation without unequivocal malignant features | AGC (Atypical Glandular Cells), specify NOS or favor neoplastic | Refer to pathologist |
| Overtly malignant squamous or glandular features (tumor diathesis, marked pleomorphism) | SCC / Adenocarcinoma / AIS | Refer to pathologist — never signed out by the screener |

Rule for picking the category: use the single most atypical group actually present on the slide, not an averaged impression — a slide that is 99.9% unremarkable and contains one diagnostic-quality HSIL-criteria group is categorized HSIL, not "mostly normal with a few odd cells."

## 4. ASC:SIL ratio audit, filled

```
Month-to-date gynecologic case log:
  Total GYN cases screened:        4,800
  ASC-US calls:                      132
  SIL calls (LSIL + HSIL):            38
  ASC:SIL RATIO = 132 / 38 = 3.47 : 1
  INTERNAL BENCHMARK CEILING:        3.0 : 1   (CAP Q-Probes-informed)

STATUS: Ratio exceeds benchmark by 0.47.

ACTION: Ratio drift does NOT change any individual case's category.
  It triggers: (a) a blinded batch re-review of a sample of this month's
  ASC-US calls by a second screener/pathologist, (b) a check of whether
  the drift is concentrated in one screener or spread across the team,
  (c) documentation of the review outcome in the QA log before the
  following month's cases are audited again.
```

## 5. Non-gynecologic reporting system quick reference

| System | Specimen type | Category tiers |
|---|---|---|
| The Paris System for Reporting Urinary Cytology | Voided urine, bladder washing | Nondiagnostic / Negative for HGUC / Atypical Urothelial Cells (AUC) / Suspicious for HGUC / HGUC / Low-grade urothelial neoplasm |
| The Milan System for Reporting Salivary Gland Cytopathology | Salivary gland FNA | Nondiagnostic / Non-neoplastic / Atypia of Undetermined Significance (AUS) / Neoplasm (benign or salivary gland neoplasm of uncertain malignant potential) / Suspicious for malignancy / Malignant |
| The Bethesda System for Reporting Thyroid Cytopathology (TBSRTC) | Thyroid FNA | I Nondiagnostic (ROM ~5-10%) / II Benign (~0-3%) / III AUS/FLUS (~10-30%) / IV Follicular Neoplasm/Suspicious (~25-40%) / V Suspicious for malignancy (~50-75%) / VI Malignant (~97-99%) |

ROM = risk of malignancy per the published tier, used to set the clinical follow-up pathway (repeat FNA, molecular testing, or surgical referral) — the category itself carries the risk estimate, so getting the tier right is not cosmetic.

## 6. Rapid on-site evaluation (ROSE) adequacy check, filled

```
Target: thyroid nodule FNA, 1.8 cm, right lobe
Pass 1: Diff-Quik air-dried smear reviewed at bedside
  Cellularity: moderate, follicular cell groups present
  Target-specific adequacy: colloid background scant, follicular groups
    present but fewer than 6 groups of 10+ cells across 2 slides (TBSRTC
    adequacy convention) -> NOT YET ADEQUATE for the specific question
    (follicular lesion characterization)
  Feedback to proceduralist: "Follicular cells present, colloid low —
    recommend one additional pass targeting the same site."

Pass 2: repeat Diff-Quik smear
  Cellularity: 8 groups of 10+ well-preserved follicular cells identified
    across the two passes combined -> ADEQUATE per TBSRTC convention
  Feedback to proceduralist: "Adequate for evaluation — no further
    passes needed for this target."
```
