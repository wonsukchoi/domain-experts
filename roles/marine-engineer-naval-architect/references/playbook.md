# Playbook

Filled formulas, coefficient tables, and calculation sequences a marine engineer/naval architect runs against at concept and basic design stage. Verify tabular values and rule editions against the current consolidated IMO/class text before certifying — these are the standard methods, not a substitute for the governing edition.

## 1. Displacement and hull-form coefficients (Archimedes)

**∇ = Cb × L × B × T** (volume of displacement, m³); **Δ = ∇ × ρ** (ρ = 1.025 t/m³ seawater, 1.000 t/m³ fresh water).

*Example: Cb=0.720, L=88.0, B=14.0, T=5.80 → ∇ = 0.720×88.0×14.0×5.80 = 5,144.83 m³; Δ = 5,144.83×1.025 = 5,273.45 t.*

| Ship type | Typical Cb range | Typical Fn (speed-length ratio) |
|---|---|---|
| Fast container/ro-ro | 0.55–0.65 | 0.26–0.32 |
| General cargo/coastal | 0.68–0.76 | 0.18–0.24 |
| Bulk carrier/tanker | 0.78–0.85 | 0.14–0.18 |
| Offshore supply vessel | 0.60–0.70 | 0.20–0.28 |

**Rule:** pick Cb from the target Froude number (Fn = V/√(gL)), not from a sister ship alone — a hull built for a different speed carries the wrong resistance/capacity trade for this mission.

## 2. Weight and KG (vertical center of gravity) estimate

| Item | Weight (t) | VCG (m) | Moment (t·m) |
|---|---|---|---|
| Lightship (steel + outfit + machinery) | W1 | VCG1 | W1×VCG1 |
| Cargo | W2 | VCG2 | W2×VCG2 |
| Fuel/water/stores/crew | W3 | VCG3 | W3×VCG3 |
| **Δ = ΣW** | | | **KG = ΣMoment/Δ** |

**Rule:** ΣW must equal the Archimedes-derived Δ from Section 1 within the design's weight-margin allowance (commonly 2-5% at concept stage, closing to near-zero by the inclining experiment) — if it doesn't, the hull-form sizing and the weight estimate are out of sync and one has to move.

## 3. KB and BM (hydrostatic particulars)

**KB — Morrish's formula:** KB = T × [5/6 − Cb/(3×Cwp)]

**BM — Normand's approximation:** inertia coefficient C = 0.096 + 0.89×Cwp²; I_T = (L×B³/12)×C; BM = I_T/∇

**KM = KB + BM**

*Example: T=5.80, Cb=0.720, Cwp=0.86 → KB = 5.80×[0.8333−0.2791] = 3.215 m. C = 0.096+0.89×0.7396 = 0.7543; I_T = (88×2,744/12)×0.7543 = 15,180.6 m⁴; BM = 15,180.6/5,144.83 = 2.950 m. KM = 6.165 m.*

**Rule:** these are concept-stage approximations. Once the hull lines are faired, pull KB/BM/KM from the hydrostatic curves generated by the fairing software (NAPA/Maxsurf/GHS), not the hand formula — the approximation is for early sizing only.

## 4. Intact stability — IMO IS Code 2008 criteria (Res. MSC.267(85))

| Criterion | Threshold |
|---|---|
| Initial metacentric height, GM₀ (fluid) | ≥ 0.15 m |
| Righting lever GZ at ≥30° heel | ≥ 0.20 m |
| Angle of maximum GZ | ≥ 25° |
| Area under GZ curve, 0°–30° | ≥ 0.055 m·rad |
| Area under GZ curve, 0°–40° (or downflooding angle if less) | ≥ 0.090 m·rad |
| Area under GZ curve, 30°–40° (or downflooding angle if less) | ≥ 0.030 m·rad |

**GM(solid) = KM − KG. GM(fluid) = GM(solid) − FSC** (free surface correction, summed across all slack tanks: FSC = Σ(i_t × ρ_liquid)/Δ, i_t = transverse free-surface moment of inertia of each tank).

**GZ(θ) = KN(θ) − KG×sin(θ)**, where KN(θ) is read from the hull's cross curves of stability (generated by the hydrostatics software for a range of assumed KG, independent of the vessel's actual KG) — GZ itself always depends on the actual KG of the loading condition being checked.

*Example: KM=6.165, KG=5.525, FSC=0.090 → GM(solid)=0.640 m; GM(fluid)=0.550 m. KN(30°)=3.850 → GZ(30°) = 3.850 − 5.525×sin(30°) = 3.850−2.7625 = 1.088 m.*

**Rule:** run every criterion in the table at every condition in the loading manual (departure, arrival, worst-case deck cargo, ballast-only) — a condition that isn't in the approved loading manual isn't a condition the ship is certified to operate in.

## 5. Minimum freeboard — ICLL 1966/1988 Protocol, Annex I

| Step | Regulation | Operation |
|---|---|---|
| 1 | Reg. 28, Table A or B | Tabular freeboard by ship length and type (A = tanker-type, restricted openings; B = all other) |
| 2 | Reg. 30 | If Cb > 0.68: multiply by (Cb+0.68)/1.36 |
| 3 | Reg. 31 | If D (molded depth) > L/15: add (D−L/15)×R mm, R=250 mm/m for L<120 m, R=L/0.48 for L≥120 m |
| 4 | Reg. 37, 38 | Deduct superstructure/trunk length credit and sheer credit per the GA |
| 5 | — | Compare resulting minimum freeboard to (D − T) at the proposed design draft |

*Example: F_tab≈903 mm; Cb correction ×1.0294 → 929.6 mm; depth correction +333.3 mm → 1,262.9 mm; superstructure/sheer credit −180 mm → F_min = 1,082.9 mm. Actual freeboard D−T = 7.20−5.80 = 1,400 mm; margin = 317.1 mm.*

**Rule:** the block-coefficient and depth corrections are not optional line items — omitting either on a full-form (Cb>0.68) or deep-hull (D>L/15) vessel understates the true minimum and can assign an unlawful load line.

## 6. Powering — Admiralty coefficient method

**Ac = Δ^(2/3) × V³ / SHP**, rearranged: **SHP = Δ^(2/3) × V³ / Ac**

| Ship type | Typical Ac range (concept stage) |
|---|---|
| Fast container/ro-ro | 220–300 |
| General cargo/coastal | 400–550 |
| Bulk carrier/tanker (slow speed) | 500–650 |
| Offshore supply vessel | 250–350 |

*Example: Δ=5,273.45 t, V=14.0 kn, Ac=430 → Δ^(2/3)=303.1, V³=2,744 → SHP = 303.1×2,744/430 = 1,934 SHP. Add 15-25% sea margin for the intended route before specifying installed power.*

**Rule:** Ac must come from a reference ship matched on hull form (Cb, L/B) and speed range — an Ac borrowed from a different ship type is a coefficient mismatch, not a shortcut, and routinely produces a 50-100% powering error.

## 7. Trim check (MCT1cm)

**MCT1cm = (Δ × GM_L) / (100 × LBP)** (moment to change trim 1 cm, t·m/cm; GM_L = longitudinal metacentric height, from hydrostatic curves).

**Change of trim (cm) = (weight added × distance from LCF) / MCT1cm**

**Rule:** check resulting trim against propeller-immersion and forward-visibility-over-the-bow requirements at every loading condition, not just that MCT1cm was computed — a mathematically valid trim can still leave the propeller partially exposed in a light-ballast condition.
