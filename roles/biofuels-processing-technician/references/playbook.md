# Playbook

## FFA-to-catalyst-dose calculation worksheet

Fill for any biodiesel batch before processing.

| Field | Value |
|---|---|
| Batch | #BD-4471 |
| Standard catalyst dose (baseline) | 1.0% (calibrated for 0.5% FFA feedstock) |
| Measured feedstock FFA | 2.5% |
| FFA excess above baseline | 2.0 points |
| Adjusted catalyst dose | ~1.8% (compensating for saponification consumption) |
| Glyceride content spec (ASTM D6751) | ≤0.24% total glycerin |
| Post-batch glyceride test result | 0.18% — PASS |
| Projected result if unadjusted (1.0% dose) | ~0.35% — would FAIL |

**Rule:** always titrate feedstock FFA before processing and calculate catalyst dose accordingly — never apply the standard dose without confirming it matches this specific batch's actual FFA content.

## Fermentation monitoring checklist

1. Confirm target temperature and pH range for the specific feedstock/process before starting.
2. Monitor temperature and pH continuously (or at frequent intervals) throughout the fermentation, not just at start and end.
3. If an excursion outside the target range occurs, note the time, duration, and magnitude — don't just correct the condition and continue without logging it.
4. If fermentation progress (measured by specific gravity/sugar consumption) appears to stall, check the monitoring log for a correlating excursion before assuming a random stuck fermentation.
5. Document the full temperature/pH profile and any excursions in the batch record, regardless of whether the final yield appears acceptable.

## Byproduct quality reference table (illustrative — always use the specific market/buyer's actual specification)

| Byproduct | Quality parameter | Higher-value use threshold | Lower-value/disposal threshold |
|---|---|---|---|
| Glycerin (biodiesel) | Purity (%) | ≥95-99% (pharmaceutical/technical grade) | Below threshold — fuel-grade or disposal |
| Distillers grains (ethanol) | Protein/moisture content | Within feed-grade spec | Outside spec — reduced value or disposal |

**Rule:** monitor byproduct quality against its own specification, distinct from primary fuel quality — process conditions optimized only for fuel yield can inadvertently reduce byproduct value even when fuel quality remains acceptable.
