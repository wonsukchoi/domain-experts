# Playbook

## Overhang/load engineering verification worksheet

Fill before cutting any load-bearing or overhang stone element.

| Field | Value |
|---|---|
| Job | Kitchen island countertop, granite |
| Requested overhang | 12 inches, unsupported |
| Cut thickness | 1.25 in (3cm) |
| Stone type engineering max unsupported overhang at this thickness | 10 in |
| Overrun | 2 in (+20% beyond limit) |
| Real-world load scenario | Person leaning/sitting on overhang edge |
| Corrective action | Corbel bracket support added at 10 in mark |
| Client informed/approved before cutting? | Y |
| Fissure check completed in overhang section? | Y — none found |
| Coolant flow verified (thermal + blade protection)? | Y |

**Rule:** never install an unsupported overhang beyond the specific stone type's engineering rating at the actual cut thickness — always add support or increase effective thickness rather than assume general stone strength covers the gap.

## Slab fissure assessment checklist

- [ ] Full slab visually inspected under raking light for fissures/veins before any cut planned.
- [ ] Fissure locations mapped relative to intended cut lines.
- [ ] Cutting orientation adjusted to avoid placing a structural cut line through a fissure.
- [ ] For large/structural pieces, NDT (ultrasonic or equivalent) considered if visual inspection is inconclusive.
- [ ] Minor, non-structural natural variation distinguished from an actual structural weak point — not every visible line is a defect requiring rejection.

## Multi-slab matching guide

| Step | Check |
|---|---|
| 1 | Identify all slabs intended for a single continuous installation (e.g., large island top, memorial). |
| 2 | Lay out slabs side by side in intended orientation before cutting. |
| 3 | Verify vein pattern and color continuity across each intended seam. |
| 4 | Adjust slab orientation/selection if a mismatch is visible at the seam. |
| 5 | Get client sign-off on the matched layout before cutting. |
| 6 | Cut only after layout is confirmed — a mismatch discovered post-installation is uncorrectable without replacing a slab. |

**Rule:** slab matching is verified by laying out the actual intended slabs in the actual intended orientation — never assumed from slabs sharing the same quarry lot or stone type name.
