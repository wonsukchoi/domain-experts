# Playbook

Filled operational documents a choreographer actually produces and works from — count sheets, rehearsal-hour budgets, an audition-combination structure, and a tech-week schedule. Populate these with the real numbers for the piece at hand rather than treating them as prose descriptions.

## 1. Count sheet (per phrase)

The working rehearsal document — not a script, a counted map of formation and step against the music.

| Count | Formation | Step / action | Note |
|---|---|---|---|
| 1–8 | Diagonal, upstage-left to downstage-right | Walk-run into first pose, arrive count 8 | Arrive *on* 8, not before — held pose needs the full count |
| 9–16 | Same diagonal | Contraction phrase, release on 13, extend 14–16 | Common error: releasing on 12; recheck against click track |
| 17–24 | Break to center block, 2 rows of 4 | Traveling turn sequence, unison | Spacing rehearsal required — new formation |
| 25–32 | Center block holds | Partnering lift, counts 27–29 | Lift only on trained pairs; understudy pairing needs separate rehearsal |
| 33–40 | Reverse diagonal, downstage-left to upstage-right | Repeat opening phrase, mirrored | Mirroring error is the #1 note at this point — flag in rehearsal |
| 41–48 | Center, single file | Final unison phrase to closing pose | Arrive final pose on count 48 exactly, hold for blackout cue |

## 2. Rehearsal-hour budget template

Build this before teaching begins; reconcile actuals against it weekly.

| Line item | Formula | Hours (example: 48 eight-counts, moderate-level company) |
|---|---|---|
| Teach + refine | eight-counts ÷ (eight-counts set per hour, 3–5 for moderate level) | 48 ÷ 4 = 12.0 |
| Spacing rehearsals | (formation changes − 1) × 1 hr each | (3 − 1) × 1 = 2.0 |
| Full-out run(s) | 1 minimum before tech, folded into teach hours if budget is tight | 0 (folded into final teach hour — flag as risk) |
| **Total needed** | sum of above | **14.0** |
| **Hours booked** | per contract/schedule | 14.0 |
| **Variance** | booked − needed | 0.0 — no slack; any added note or cast change consumes contingency that doesn't exist |

Rule of thumb for the first line: 3–5 eight-counts/hour for a moderate-level company, 5–8 for a highly trained company, 1–3 for a student or community cast learning technique and choreography simultaneously.

## 3. Audition combination structure

A combination is built to sort dancers on the specific things the piece needs, not to look impressive in isolation.

| Section | Length | Purpose | What it screens for |
|---|---|---|---|
| Warm-up phrase | 16 counts | Settle nerves, establish tempo | Musicality, ability to pick up counts from a single walkthrough |
| Technical phrase | 24 counts | Turns, jumps, or the piece's signature technical demand | Baseline technical level against the role's actual requirement |
| Partnering/lift snippet (if cast requires) | 8 counts | Test with a rotating partner, not a fixed one | Trust, weight-sharing, adaptability — not chemistry with one specific partner |
| Free-movement/improv | 8–16 counts | Unscripted, to a different piece of music | Whether the dancer has movement quality beyond executing set steps |

Cut order when time is short: drop free-movement first (least diagnostic per minute spent), keep the technical phrase and partnering snippet last (highest-cost mistakes to discover after casting).

## 4. Tech-week schedule (Equity "ten-out-of-twelve" framework)

A tech day runs 10 hours of actual work inside a 12-hour call, per standard Equity/AGMA agreements — the 2 non-worked hours are meal breaks (typically one 1-hour and one 30-minute break, plus shorter rest breaks through the day).

| Block | Hours | Focus |
|---|---|---|
| Cue-to-cue | 1–3 | Lighting/sound cue placement only — dancers walk marked positions, no full-out dancing |
| Spacing pass | 2 | Reconcile rehearsal-room blocking against actual stage dimensions, wings, traps, sightlines |
| First full stagger-through | 3 | Full-out, with stops for major fixes only — this is where undiscovered stamina/collision issues surface |
| Notes + reset | 1 | Choreographer and director triage: cast self-correct vs. needs another rehearsal slot |
| Second run | 3 | As close to performance conditions as the day allows; last chance before dress to catch what marking hid |

Fallback order when a tech day loses time to a technical problem (lighting, set): protect the spacing pass and first full stagger-through before protecting a second run — an unspaced piece is a safety problem at cue time; a second run is a polish problem.
