# Travel Agent — Vocabulary

### Minimum Connection Time (MCT)
The airport-published minimum number of minutes a booking system will allow between a passenger's arriving and departing flights at a given connection point.
**In use:** "The GDS will let me sell a 60-minute connection here because that's the posted MCT, but I'm not booking it that tight for a client with checked bags."
**Common misuse:** treating MCT as a safe, workable connection time rather than the bare legal floor the booking system will sell.

### Fare basis code
A short alphanumeric code identifying the specific fare product (cabin, restrictions, refund rules) attached to a ticket.
**In use:** "The fare basis code changed between my first search and now, so I need to re-pull the rules before I quote this price."
**Common misuse:** assuming two tickets in the same cabin class have the same fare basis code, and therefore the same change/refund rules.

### 24-hour rule
US DOT regulation (14 CFR 259.5) requiring airlines to allow free cancellation or a fare hold for 24 hours after booking, for tickets purchased 7+ days before departure, on US-originating itineraries.
**In use:** "We have the 24-hour window to fix this if the client changes their mind, but I still want to get the fare right on the first quote."
**Common misuse:** treating the 24-hour rule as a general safety net that applies to all bookings, all timeframes, or non-US-originating itineraries.

### Global Distribution System (GDS)
The reservation/booking infrastructure (e.g., Sabre, Amadeus, Travelport) that aggregates airline, hotel, and car-rental inventory for travel agents.
**In use:** "I pulled this fare directly in the GDS rather than through the airline's own site because the fare rules display differently."
**Common misuse:** assuming GDS-displayed pricing and rules are always identical to what's available booking direct with the supplier — commission structure and even fare availability can differ.

### Cancel For Any Reason (CFAR)
An optional travel-insurance rider allowing cancellation reimbursement (typically 50–75%, not 100%) for reasons outside the base policy's covered-reasons list.
**In use:** "The base policy wouldn't cover a change-of-mind cancellation, but the CFAR rider would, at a reduced reimbursement rate."
**Common misuse:** assuming CFAR pays out 100% of trip cost, or that it can be added any time before departure rather than within the insurer's purchase-deadline window (commonly 14–21 days after initial trip deposit).

### Fare class / booking class
The single-letter code (e.g., Y, B, M, K) representing a specific price/restriction tier within a cabin (economy, business, etc.).
**In use:** "This is a K-class fare — cheapest available, but non-refundable and no same-day changes."
**Common misuse:** conflating fare class with cabin class; multiple fare classes can exist within the same physical cabin, each with different rules.

### Commission override / net rate
A negotiated commission rate (or wholesale net rate) an agency receives from a supplier, which can differ from the publicly advertised rate.
**In use:** "Our net rate with this tour operator is better than their published commission, which is part of why I'm recommending them — worth disclosing."
**Common misuse:** assuming all agencies receive the same commission from a given supplier; rates are negotiated per agency relationship and volume.

### Layover vs. connection
A layover is a scheduled stop of any duration on a single ticket; a connection specifically implies a change of aircraft (and often airline) requiring the passenger to make their own way to the next gate.
**In use:** "This isn't just a layover — it's a tight connection with a terminal change, so I'm building in extra buffer."
**Common misuse:** using the terms interchangeably when the operational risk (missing the next flight) only applies to true connections, not through-flights with a scheduled stop.

### PNR (Passenger Name Record)
The unique reservation record in a GDS containing all booking details for one or more travelers on a trip.
**In use:** "I'll pull up the PNR to confirm both travelers' passport numbers are attached correctly before ticketing."
**Common misuse:** assuming a single PNR means all travelers on it share identical entry requirements or fare rules — they can differ by nationality even within one booking.

### Consolidator fare
A discounted airfare purchased in bulk by a wholesale consolidator and resold through travel agencies, often with different (sometimes stricter) change/refund rules than the airline's own published fares.
**In use:** "This fare is cheaper because it's a consolidator ticket, but I want to flag that the change fee is higher than a standard published fare."
**Common misuse:** assuming a consolidator fare's rules mirror the airline's standard published fare rules for the same route and cabin.

### Deposit refund schedule
A supplier's (typically cruise or tour operator's) tiered schedule showing what percentage of a deposit is refundable at various points before departure.
**In use:** "At 90 days out this deposit is fully refundable, but it drops to 50% at 60 days and zero at 30 days — that's the window that matters for the insurance conversation."
**Common misuse:** treating "refundable deposit" as a fixed yes/no rather than a schedule that changes the closer the trip gets.
