# Podiatrist — Playbook

## Wound classification: Wagner grade

| Grade | Description |
|---|---|
| 0 | Intact skin, at-risk foot (deformity, prior ulcer, callus) — no open lesion |
| 1 | Superficial ulcer, full skin thickness, no deeper structure involvement |
| 2 | Ulcer extends to tendon, capsule, or bone (no abscess/osteomyelitis) |
| 3 | Deep ulcer with abscess, osteomyelitis, or joint sepsis |
| 4 | Gangrene localized to forefoot or heel |
| 5 | Gangrene involving the whole foot |

## University of Texas classification (depth × infection/ischemia)

| Stage \ Grade | 0 (pre/post-ulcer) | I (superficial) | II (to tendon/capsule) | III (to bone/joint) |
|---|---|---|---|---|
| A — clean | Lowest risk | | | |
| B — infected | | | | |
| C — ischemic | | | | |
| D — infected + ischemic | | | | Highest risk, urgent multidisciplinary |

Use both systems together, not one alone: Wagner gives a fast depth read; UT stage adds the infection/ischemia dimension Wagner misses. A UT stage-D wound at any grade is a same-day multidisciplinary case regardless of depth.

## Offloading selection by location and severity

| Ulcer location | First-line offloading | Escalation if not healing |
|---|---|---|
| Plantar forefoot (metatarsal heads) | Total contact cast or removable cast walker | Custom molded orthotic + activity restriction; surgical metatarsal head resection if recurrent |
| Plantar heel | Heel-relief cast/boot with heel cutout | Non-weight-bearing until closure if location prevents effective offloading |
| Dorsal (toe, over deformity) | Extra-depth shoe, toe spacer/padding | Deformity correction (hammertoe/bunion surgery) if recurrent under an uncorrected deformity |
| Midfoot (Charcot deformity) | Total contact cast, non-weight-bearing | Surgical reconstruction if instability persists after acute Charcot phase resolves |

## ABI interpretation and debridement gating

| ABI | Interpretation | Debridement decision |
|---|---|---|
| 1.0–1.4 | Normal | Proceed with debridement as clinically indicated |
| 0.9–1.0 | Borderline | Proceed; monitor healing trajectory closely |
| 0.5–0.9 | Mild-moderate PAD | Proceed if healing trajectory tracks on schedule; low threshold for vascular referral if it stalls |
| <0.5 | Severe PAD | Hold elective debridement — urgent vascular referral first, unless active infection requires source control regardless |
| >1.4 | Non-compressible (calcified vessels, common in long-standing diabetes) | ABI unreliable — use toe-brachial index or other perfusion study instead |


## Healing-trajectory checkpoint (4-week benchmark)

```
Initial area: 1.6 cm^2
4-week area: 0.68 cm^2
% reduction: (1.6 - 0.68) / 1.6 = 57.5%
Benchmark: >=50% reduction at 4 weeks predicts on-track healing by ~12 weeks
Result: ON TRACK — continue current plan
```

If percent reduction falls below 50% at the 4-week checkpoint, escalate: imaging (MRI or bone scan) for occult osteomyelitis, repeat vascular workup, or surgical/wound-care-center referral — do not simply continue the same plan for another interval without a reason to expect it will now work.
