# Playbook

Filled procedures a CSA actually runs, with structure, checkpoints, and example numbers. Adapt to the surgeon's stated preference card and the facility's protocol — these are the shape of the process, not a substitute for either.

## 1. Count protocol (No Thing Left Behind method)

Two-person, called-aloud, at fixed checkpoints — never a single-person tally, never skipped for a "quick" case (a large share of retained-item events happen in cases judged low-risk beforehand).

**Checkpoints, every case:**
1. **Baseline count** before incision — sponges, sharps, instruments, and any small items (needles, blades) counted by scrub and circulator together, assistant confirms verbally.
2. **First closing count** — before closing a body cavity (peritoneum, pleura, joint capsule). Assistant does not touch closure until this count is reconciled.
3. **Second closing count** — before fascia closure.
4. **Final count** — before skin closure/final dressing.
5. **Any relief/shift-change count** — full recount whenever scrub or circulator hands off mid-case.

**On a discrepancy:**
```
1. STOP. State the discrepancy aloud to the whole team.
2. Visually search the field, the floor, the drapes, and the trash/linen — in that order.
3. If unresolved: intraoperative X-ray before the patient leaves the OR.
   (Do not accept "we'll X-ray after if there's a problem" as a substitute.)
4. Document the discrepancy, the search, and the resolution regardless of outcome.
```
Assistant's role: independently verify the count out loud, not just listen — and be the second voice requesting the X-ray if there's any hesitation to get one.

## 2. Camera-driving setup (laparoscopic/robotic assist)

**Before insufflation:**
- White-balance the scope against a white sponge/gauze, not the drape (color temperature differs).
- Confirm 0° vs. 30° scope choice matches the planned approach (30° for angled structures — e.g., posterior gallbladder dissection, pelvic anatomy).
- Anti-fog solution applied; confirm light cable connection isn't kinked (heat buildup risk at the tip).

**During the case:**
- Horizon level relative to the patient, re-leveled any time the surgeon changes working angle — never let the horizon drift to match the instrument angle instead.
- Working instrument tip kept centered, roughly one-third to one-half of the frame from the leading edge, so the surgeon sees where the tip is heading, not just where it is.
- **Before any energy activation, confirm the active tip is fully in frame.** If the surgeon calls for energy and the tip is at the frame edge or obscured by smoke/fluid, reposition the camera before the device fires, not after.
- Smoke evacuation cued proactively during extended cautery/ultrasonic use rather than waiting until visibility is already degraded.

## 3. Retractor selection by exposure type

| Exposure need | Typical retractor choice | Assistant's active task |
|---|---|---|
| Upper abdominal (subcostal, epigastric) | Bookwalter or Omni-Tract with fixed ring | Select blade sizes for liver/costal margin retraction; reposition blades if resistance increases rather than adding tension |
| Deep pelvic | Deaver, malleable, or St. Mark's retractor hand-held | Continuous manual retraction — rotate hand position/angle periodically on long cases rather than holding one fixed vector throughout |
| Long midline laparotomy, prolonged case | Self-retaining ring system | Full release at the 30-minute mark, reassessing blade position and padding on reset, checking for blanching or excessive pressure at contact points |
| Laparoscopic/robotic | Graspers via assist port | Provide countertraction opposite the surgeon's working instrument, not traction on the same tissue plane the surgeon is dissecting |

**Rule:** if a retractor needs more force than the last position to hold the same exposure, that is a repositioning problem, not a tension problem. See `references/red-flags.md` #7 for the 30-minute release interval and its evidence.

## 4. No-touch saphenous vein harvest (per surgeon's protocol)

1. **Incise skin over the marked vein course**, preserving the perivascular fat pedicle — dissect around the pedicle, not directly on the vein wall.
2. **Ligate side branches** ~1 cm off the vein wall (not flush), using fine ties, to avoid narrowing the parent vessel or leaving a stub that thromboses.
3. **Avoid high-pressure saline distension** — no-touch technique specifically skips this step; if patency check is needed, use gentle manual distension or Doppler rather than pressurized saline, which the surgeon's protocol should specify.
4. **Compute harvest length before cutting the distal end** — anastomosis-to-anastomosis path length + ~1.25 cm trim per anastomosis + ~15% for post-harvest elastic recoil + ~2 cm handling margin (see SKILL.md worked example for the full arithmetic).
5. **Mark orientation** (proximal/distal) per the surgeon's stated convention before handoff — a reversed vein graft depends on valve orientation being correct.
6. **Verbal handoff report** at the moment of transfer: length, technique used, any branches ligated and how, any wall irregularities, orientation marking. Never hand off silently.

## 5. Closure sequence (layered abdominal closure, example)

```
1. Peritoneum   — running absorbable suture (e.g., 2-0 polyglactin),
                  assistant provides countertraction opposite the needle
                  path, not tension on the suture line itself.
2. Fascia       — surgeon's specified suture (often looped or running
                  permanent/slow-absorbing), assistant maintains even
                  tissue apposition and clears intervening fat/omentum
                  from the bite before each pass.
3. Subcutaneous — interrupted absorbable, dead-space closure if wound
                  depth warrants it (obesity, prior wound complications).
4. Skin         — per surgeon preference (subcuticular running,
                  staples, adhesive strips) — confirm before starting;
                  do not default to habit from a different surgeon's
                  preference card.
```
At each layer boundary, confirm the relevant count checkpoint (see #1) is reconciled before contributing suture tension to that layer.
