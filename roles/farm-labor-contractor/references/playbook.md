# Farm Labor Contractor — Playbook

## Piece-rate true-up calculation (filled example)

| Component | Value |
|---|---|
| Hours worked | 8.0 |
| Pounds picked | 210 |
| Piece rate | $0.45/lb |
| Piece-rate earnings (210 × $0.45) | $94.50 |
| Effective hourly rate ($94.50 ÷ 8) | $11.81/hr |
| Applicable minimum wage | $15.00/hr |
| Minimum wage floor for the day (8 × $15.00) | $120.00 |
| **True-up owed** ($120.00 − $94.50) | **$25.50** |
| **Total pay for the day** | **$120.00** |

**Use:** Recalculate this table every pay period for every piece-rate worker — a strong day cannot be assumed to offset a weak one across a period without checking each period against the applicable true-up rule.

## H-2A three-quarter guarantee calculation (illustrative example)

| Component | Value |
|---|---|
| Total workdays in contract period | 40 |
| Three-quarter guarantee (75% × 40) | 30 workdays |
| Actual workdays provided (season ended early — weather) | 24 workdays |
| Shortfall workdays (30 − 24) | 6 workdays |
| Contract daily pay rate (AEWR $16.25/hr × 8 hrs) | $130.00/day |
| **Shortfall payment owed** (6 × $130.00) | **$780.00** |

**Use:** Calculate this whenever a season ends early or work volume falls short of the job order's promised schedule — the three-quarter guarantee is a specific payable obligation, not a discretionary adjustment.

## Pre-season compliance checklist

1. Confirm federal MSPA registration and any applicable state FLC license are current (not expired, not pending renewal).
2. Prepare written wage/job disclosure in the language(s) recruited workers understand; confirm delivery process happens before any travel commitment.
3. If housing will be provided, schedule and complete a pre-occupancy inspection against the applicable temporary labor camp standard.
4. If transportation will be provided, verify vehicle insurance, safety inspection status, and driver licensing (including CDL where required) before the season starts.
5. If piece-rate pay will be used, set up a per-pay-period true-up calculation process before the first pay period runs.
6. If working with a grower client, document the actual division of supervisory control to assess joint-employer exposure realistically.
7. If subcontracting any portion of the work, verify the subcontractor's license/registration status before engagement.
