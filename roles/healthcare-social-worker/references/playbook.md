# Healthcare Social Worker — Playbook

## LACE index scoring table

| Component | Value | Points |
|---|---|---|
| **L** — Length of stay | <1 day | 0 |
| | 1 day | 1 |
| | 2 days | 2 |
| | 3 days | 3 |
| | 4–6 days | 4 |
| | 7–13 days | 5 |
| | ≥14 days | 7 |
| **A** — Acute/emergent admission | Yes | 3 |
| | No (elective) | 0 |
| **C** — Charlson comorbidity index | 0 | 0 |
| | 1 | 1 |
| | 2 | 2 |
| | 3 | 3 |
| | ≥4 | 5 |
| **E** — ED visits, prior 6 months | 0 | 0 |
| | 1 | 1 |
| | 2 | 2 |
| | 3 | 3 |
| | ≥4 | 4 |

**Total LACE = L + A + C + E.** Range 0–19. Score ≥10 = high risk of 30-day readmission or death — trigger high-intensity discharge planning. Score 6–9 = moderate risk — standard follow-up plus one targeted intervention. Score <6 = low risk — standard discharge process.

## Psychosocial barrier screen (use when 2+ admissions/ED visits in prior 6 months)

| Domain | Screening question | Positive finding → action |
|---|---|---|
| Medication affordability | Can you afford your prescribed medications at the pharmacy? | Refer to hospital financial counselor / manufacturer assistance program before discharge, not after. |
| Transportation | Do you have a confirmed way to get to your follow-up appointment? | Book NEMT or telehealth follow-up before discharge — "will figure it out" is not a confirmed plan. |
| Caregiver capacity | Who will help you at home, and can they confirm specific hours and physical tasks? | Verify directly with the caregiver, not through the patient's report of the caregiver's willingness. |
| Housing stability | Is your home safe and accessible for your current mobility level? | PT home-safety evaluation or temporary SNF stay if stairs/access is a barrier. |
| Health literacy | Can you walk me through how you'll take your new medications? (teach-back) | If teach-back fails, arrange a follow-up call within 48 hours, not just written instructions. |

## Discharge-destination decision table

| Functional status | Payer authorization likelihood | Recommended destination |
|---|---|---|
| Independent ADLs, stable vitals, confirmed caregiver capacity | N/A | Home, standard follow-up |
| 1–2 ADL deficits, confirmed caregiver capacity for those specific deficits | Home health typically authorizes with documented ADL deficits | Home + home health (skilled nursing/PT) |
| 3+ ADL deficits or no confirmed caregiver capacity | SNF requires documented skilled-need + functional deficit, not general debility | SNF — document specific deficits (e.g., "unable to transfer bed-to-chair without two-person assist," not "weak") |
| Terminal diagnosis, goals-of-care conversation completed | Hospice election requires physician certification of ≤6-month prognosis | Hospice referral, coordinate with attending for certification |

**Rule:** never write "generalized weakness" or "poor functional status" as the sole justification for SNF — payer utilization review requires a specific, numbered deficit (e.g., "requires max assist x2 for transfers, failed 3 of 5 physical therapy mobility milestones"). Vague documentation is the most common reason a clinically appropriate SNF placement stalls in authorization.

## Warm handoff script (high-risk transfer)

1. Call the receiving facility/agency's intake nurse directly — do not rely on the fax alone.
2. State the LACE score or equivalent risk flag and the specific reason (e.g., "LACE 14, CKD stage 3 with diuretic-dosing sensitivity, confirmed caregiver transportation gap").
3. Confirm the receiving party has the medication reconciliation list and the follow-up appointment date/time.
4. Document the handoff: who you spoke with, when, and what was confirmed.
