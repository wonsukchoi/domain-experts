# Physical therapist aide — playbook

Filled examples an aide can execute or populate as-is. Load this when a specific task, modality, or scheduling conflict needs the actual parameters, not just the principle.

## Task classification table

| Task | Category | Supervision required |
|---|---|---|
| Clean/disinfect treatment table, restock linens/gel | Non-patient-related | None — do independently |
| Schedule appointments, pull charts, file insurance paperwork | Non-patient-related | None — do independently |
| Set up hydrocollator/paraffin unit, check unit temperature log | Non-patient-related | None — do independently |
| Transport patient to/from treatment area (wheelchair, walker) | Patient-related (many states carve this out as physical support only) | PT/PTA on-site; verify against your state's practice act |
| Apply a hot/cold pack or paraffin dip the PT already parameterized | Patient-related | PT/PTA on-site and immediately available |
| Assist a gait-belt transfer or ambulation the PT already ordered | Patient-related | PT/PTA on-site and immediately available |
| Progress exercise resistance, reps, or range | Clinical judgment | Not delegable at any supervision level — PT/PTA performs or explicitly re-orders it |
| Interpret a patient's response to treatment ("that means your inflammation is down") | Clinical judgment | Not delegable — relay the observation, not an interpretation |
| Answer a patient's question about diagnosis or prognosis | Clinical judgment | Not delegable — redirect to PT |

## Modality parameter tables

### Hydrocollator (moist heat) pack

| Parameter | Standard value | Adjust when |
|---|---|---|
| Unit temperature | 160–166°F | Never exceed manufacturer spec even if a patient requests it hotter |
| Toweling layers | 6–8 layers between pack and skin | Use 8 (top of range) for sensory-impaired, diabetic, or pediatric/geriatric patients |
| First skin check | 5 minutes in | Move to 3 minutes for any patient with neuropathy, PVD, or prior burn history |
| Standard treatment duration | 15–20 minutes | Pull immediately, regardless of elapsed time, on any abort signal (below) |
| Abort signals | Numbness, tingling, sharp pain, skin color change (pink/mottled/white) at pack border | Remove pack, notify PT before doing anything else |

### Cold pack / cryotherapy

| Parameter | Standard value | Adjust when |
|---|---|---|
| Duration | 10–15 minutes direct application | Reduce to 10 min for thin-tissue areas or impaired circulation |
| Barrier | Thin towel or pillowcase between pack and skin — never bare skin | Add a second layer for prolonged (>15 min) applications |
| Check interval | Every 5 minutes | Every 3 minutes for sensory-impaired patients |
| Abort signals | Burning pain (distinct from expected cold ache), blanching beyond the pack's footprint, patient reports "sharp" not "cold" | Remove immediately, notify PT |

### Paraffin bath (dip and wrap)

| Parameter | Standard value | Adjust when |
|---|---|---|
| Bath temperature | ~118–126°F (per unit spec — verify against the manufacturer's posted range before each use) | Never use if the log shows the unit outside its posted range |
| Dips before wrap | 8–12 dips to build a sealed wax coat | Fewer dips = incomplete seal = uneven heat, not a shortcut |
| Wrap | Plastic bag or wrap, then a towel, over the coated area | — |
| Duration | 20–30 minutes or until wax fully cools | — |
| Contraindications (do not apply — hold and notify PT) | Open wound, active skin infection, recent thin scar (<6 months), impaired thermal sensation, known peripheral vascular disease | Any one present is a stop, not a "monitor closely" |

## Room-turnover checklist (target: inside the clinic's scheduled unit, commonly 15–20 minutes)

1. Clear and disinfect the treatment table and any surfaces the prior patient contacted (per the clinic's EPA-registered disinfectant contact-time spec).
2. Log the cleaning in the equipment/room log — an undocumented clean is treated as not done if anyone asks later.
3. Reset linens, positioning wedges/pillows, and any single-use supplies.
4. Re-check hydrocollator/paraffin unit temperature against the log if the modality is queued for the next patient.
5. Pull the next patient's chart and confirm that day's ordered parameters before the PT enters the room.
6. Stage the modality (pack heating, table height, equipment positioned) so the PT's first minute in the room is spent on the patient, not setup.

If any step is skipped under time pressure, skip step 3 (restock cosmetics like extra pillows) before ever skipping steps 1–2 (infection control) or step 5 (parameter verification) — the ordering reflects consequence severity, not convenience.

## Fall-risk transfer sequencing (when two transfers are needed and one aide is available)

1. Confirm the PT-assigned fall-risk tier for each patient (high / moderate / low) — this is documented, not eyeballed.
2. High-risk transfers go first and always get two people; if a second person isn't available within the time the transfer is needed, the transfer waits and the PT is notified — it does not proceed solo.
3. Moderate-risk transfers with a gait belt can proceed solo only if the aide has been trained and cleared for solo gait-belt assistance at this clinic; a gait belt supports balance, it does not substitute for a lift and cannot arrest a fall in progress.
4. Low-risk/independent-ambulation patients are sequenced last, since the consequence of a short wait is inconvenience, not injury.
5. Any transfer where the patient's condition doesn't match the assigned tier that day (e.g., a "low-risk" patient reporting new dizziness) is treated as the higher tier until the PT re-assesses — the aide upgrades, never downgrades, on ambiguous signals.
