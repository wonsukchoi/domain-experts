# Embalmer — Playbook

## Case-condition dilution-adjustment table

| Case condition | Index adjustment | Volume adjustment | Notes |
|---|---|---|---|
| Standard (no complicating factors) | Baseline (e.g. 24-index) | Baseline (1 gal / 45 lb) | Single-point carotid injection typical |
| Mild edema (localized) | +1 index-equivalent | −10% volume | Local dilution from interstitial fluid; targeted massage to aid distribution |
| Severe/generalized edema | +2 index-equivalent | −15 to −20% volume | Consider multi-site injection; cavity aspiration first if gas/fluid distension present |
| Jaundice | +1 index-equivalent, add bleaching agent to fluid | Baseline volume | Discoloration requires cosmetic correction in addition to chemical treatment |
| Emaciation | Baseline or −1 index-equivalent | −20 to −30% volume | Reduced tissue mass; over-injection distorts features |
| Obesity | Baseline index | +15 to +25% volume | Higher fat content requires more fluid for full distribution; monitor for uneven distribution |
| Autopsied case | +1 index-equivalent | Split: arterial + cavity treatment | Six-point or per-facility multi-site injection; standard single-point technique insufficient |
| Early decomposition (1–2 indicators) | +2 index-equivalent, cavity pretreatment | Standard to +10% | Cavity treatment before arterial injection, not after |
| Advanced decomposition (3+ indicators) | Consult facility protocol — may require non-standard preservation method | — | May exceed standard embalming scope; consult supervising embalmer |

## Decomposition-stage assessment checklist

Assess and document each on arrival, cross-checked against reported time-since-death:

1. **Rigor mortis** — present/resolving/absent. Typically onset 2–6h, peak 12h, resolution 24–48h (ambient-temperature-dependent).
2. **Livor mortis** — fixed or blanching. Fixes approximately 8–12h post-mortem.
3. **Skin slippage** — none / early (small areas) / advanced (large sheets). Indicates decomposition has begun at the cellular level.
4. **Marbling** — vascular discoloration pattern from bacterial hemolysis. Presence indicates decomposition is underway regardless of reported TSD.
5. **Gas distension** — abdominal/facial swelling from bacterial gas production. Requires cavity aspiration before or alongside arterial injection.
6. **Odor** — none / mild / strong. Strong odor independent of other indicators warrants immediate cavity treatment.

**Mismatch rule:** if two or more indicators suggest a decomposition stage more advanced than the reported time-since-death implies, treat the case per the more advanced stage — ambient temperature and cause of death (e.g., sepsis, prolonged discovery interval) can accelerate decomposition well beyond what elapsed time alone would predict.

## Restorative-art workflow

1. Photograph the affected area before beginning any correction (reference for family/mortician sign-off and for the embalmer's own before/after record).
2. Compare against any family-supplied reference photograph for expected appearance.
3. Build up wax/tissue-builder in thin layers — check color and form match after each layer, not only at completion.
4. Apply cosmetics last, matched to the deceased's natural skin tone using the facility's cosmetic reference set, not a generic shade.
5. Final check against the reference photograph and, where facility protocol allows, a second embalmer's sign-off before the family views.
