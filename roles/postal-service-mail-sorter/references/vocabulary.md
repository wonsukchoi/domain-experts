# Vocabulary

### DBCS (Delivery Bar Code Sorter)
A letter-sorting machine that reads the barcode or OCR-derived address and sorts letter-size mail into delivery sequence.
**In use:** "DBCS-06 is running First-Class letters this shift, rated at 35,000 pieces an hour."
**Common misuse:** Using it interchangeably with FSM; a DBCS handles letters only — flats require a physically different machine.

### FSM (Flat Sorting Machine)
Sorting equipment built for flat-size mail — magazines, catalogs, large envelopes — that letter sorters can't process.
**In use:** "That catalog batch needs to go to FSM-2, not the DBCS line."
**Common misuse:** Assuming any sorting machine can handle any mail shape; letters and flats are processed on entirely separate equipment.

### AFCS (Advanced Facer Canceler System)
Equipment that orients ("faces") letters so address and postage align, then cancels the postage, as a stage before sorting.
**In use:** "Batch came through AFCS already faced and canceled — should induct clean on the DBCS."
**Common misuse:** Confusing facing/canceling with sorting itself; AFCS is an earlier, separate processing stage, not the sortation step.

### REC (Remote Encoding Center)
A facility where staff key in addresses from images of mail pieces a sorting machine's OCR couldn't confidently read.
**In use:** "That reject batch is headed to REC — OCR confidence was too low to auto-sort."
**Common misuse:** Assuming unreadable mail is simply delayed or lost; in most cases it's routed to REC for manual resolution rather than dropped.

### Reject rate
The percentage of pieces in a run a sorter can't confidently read or sort automatically, routed instead to REC or manual handling.
**In use:** "Reject rate's 9.4% today against a 6% baseline for this mail type — that's worth pulling a sample on."
**Common misuse:** Comparing reject rate only against the contractual ceiling; the mail-type's own baseline, usually well under that ceiling, is what actually flags a new problem.

### Culling
Separating mail by class or size — manually or mechanically — before it reaches a sorting machine.
**In use:** "These trays weren't culled right — flats mixed in with the letter batch."
**Common misuse:** Treating culling as the same step as sorting; culling is the precondition that makes automated sorting possible, not the automated step itself.

### Nonmachinable mail
Mail that fails automated processing criteria — size, rigidity, thickness — and must be hand-processed instead.
**In use:** "That rigid mailer's nonmachinable — routes to manual, and the surcharge should already be applied."
**Common misuse:** Treating "nonmachinable" as just a shape description; it's a defined cost-driving category with a surcharge policy specifically because hand processing costs materially more per piece.

### Facing
Orienting mail pieces so the address and postage face the same direction before canceling or sorting.
**In use:** "AFCS handles the facing — by the time it reaches the DBCS, orientation's already fixed."
**Common misuse:** Assuming facing happens automatically at every stage; it's a discrete step some equipment performs and later equipment assumes is already done.

### Delivery bar code / Intelligent Mail barcode (IMb)
The machine-readable code printed on a mail piece encoding its delivery-point information, which the sorter actually reads.
**In use:** "The ZIP+4 printed fine, but the IMb didn't scan clean — that's why it kicked to reject."
**Common misuse:** Assuming the printed ZIP+4 alone drives the sort; the machine sorts against the encoded barcode, which can diverge from the visible address if printed incorrectly.

### Run
A scheduled batch of a specific mail class processed through a specific machine during a shift.
**In use:** "Run 4471 was First-Class letters on DBCS-06 — the flats run is scheduled separately this afternoon."
**Common misuse:** Treating "run" and "shift" as synonyms; a single shift commonly includes multiple runs of different mail classes or machines.
