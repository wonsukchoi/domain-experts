# Playbook

Filled data sheets, fading protocols, and referral checklists a paraeducator runs day to day. Adapt the numbers to the actual student's plan — the structures are the reusable part.

## 1. Prompting hierarchy fading protocol (system of least prompts)

Five rungs, used in strict order, moved one rung at a time:

| Rung | Prompt | Wait before delivering |
|---|---|---|
| 0 | Independent attempt | 3–5 sec of visible processing, no prompt |
| 1 | Gesture (point, tap the material) | 3–5 sec after the gesture |
| 2 | Verbal (a hint or restated direction, not the answer) | 3–5 sec after the prompt |
| 3 | Model (demonstrate the step, student repeats it) | immediate re-attempt |
| 4 | Physical (hand-over-hand or physical guidance) | last resort for this hierarchy |

**Movement rule:** 2 consecutive misses at a rung → move up one rung, not to the top. 3 consecutive independent successes at a rung → move down one rung and re-test. Never skip a rung in either direction — a jump from independent straight to physical loses the data point of exactly where the breakdown happened.

**Filled example (task: "write today's date on worksheet"), 8 trials:**

| Trial | Rung used | Result |
|---|---|---|
| 1 | 2 (verbal) | correct |
| 2 | 2 | correct |
| 3 | 2 | correct → drop to rung 1 next trial |
| 4 | 1 (gesture) | correct |
| 5 | 1 | incorrect |
| 6 | 1 | incorrect → move up to rung 2 next trial |
| 7 | 2 | correct |
| 8 | 2 | correct |

Read this sheet as: rung 1 (gesture) isn't reliable yet for this step, rung 2 (verbal) is — the plan for next session starts at rung 2, not back at rung 4.

## 2. Data-collection method selection and filled sheets

**Selection table:**

| Behavior type | Method | Why |
|---|---|---|
| Discrete, countable (hand raise, call-out, correct response) | Frequency/tally count | Each instance is a clean, separate event |
| Defined by how long it lasts (tantrum, off-task, out-of-seat) | Duration recording | A count doesn't capture severity; a 2-minute vs. 20-minute tantrum are very different data points |
| Continuous or very rapid (stereotypy, humming, rocking) | Partial-interval recording | Too fast/continuous to count discretely; interval sampling approximates rate |
| Function unknown | ABC narrative | Need to see antecedent and consequence before any other method makes sense |

**Filled frequency sheet** (target: hand-raising instead of calling out, goal ≥5/period):

```
Student: J.T.        Date: 10/2        Period: Reading (45 min)
Hand raises (tally): //// //  = 7        Call-outs (tally): ///  = 3
Notes: call-outs clustered in first 10 min before meds effect; none after 10:15.
```

**Filled duration sheet** (target: reduce out-of-seat time):

```
Student: A.M.         Date: 10/2        Period: Math (40 min)
Instance 1: 9:05–9:08 (3 min)     Instance 2: 9:22–9:24 (2 min)
Total out-of-seat: 5 min / 40 min period (12.5%)
Prior week average: 9.5 min/period → this session down 47%.
```

**Filled partial-interval sheet** (target: reduce stereotypy, 10-sec intervals over a 10-min sample = 60 intervals):

```
Student: R.K.         Date: 10/2        Activity: independent work
Intervals with behavior present: 22 / 60 = 36.7%
Prior baseline: 41/60 = 68.3% → down ~46% from baseline.
```

**Filled ABC narrative:**

```
Student: D.L.    Date: 10/2    Time: 10:40am
Antecedent: Teacher announced transition to math worksheets.
Behavior: D.L. swept materials off desk, said "no."
Consequence: Was given a 2-min break in the calm corner, then re-approached with a choice of two worksheets.
Likely function: Escape from non-preferred task (3rd occurrence this week at a transition into written math).
```

## 3. Task analysis with prompt-level tracking

Break a multi-step routine into scorable steps and record the rung used per step, not a single pass/fail for the whole task — a whole-task score hides which specific step is actually the bottleneck.

**Filled example — "wash hands independently" (6 steps), one trial:**

| Step | Rung used |
|---|---|
| 1. Turn on water | 0 (independent) |
| 2. Wet hands | 0 |
| 3. Apply soap | 1 (gesture) |
| 4. Scrub 20 sec | 2 (verbal — needed a count-along prompt) |
| 5. Rinse | 0 |
| 6. Dry hands with towel | 0 |

Reading this sheet: steps 1, 2, 5, 6 are independent — stop prompting those even reflexively. Steps 3 and 4 are the actual instructional targets; that's where fading effort belongs next, not a blanket re-teach of the whole routine.

## 4. FBA referral trigger checklist

Escalate to the case manager for a functional behavior assessment referral when **any one** of these is true, not only when all are:

- [ ] Same behavior occurred 3+ times in a rolling 5-school-day window
- [ ] Behavior resulted in removal from instruction (office referral, timeout, suspension) at least once
- [ ] ABC data across 3+ occurrences shows no consistent antecedent or function yet identifiable
- [ ] Behavior involved any risk to student or others' physical safety, regardless of frequency
- [ ] Current behavior plan strategies (if one exists) have been implemented with fidelity for 2+ weeks with no downward trend

Bring to the referral conversation: the ABC log for all occurrences, frequency/duration data if collected, and what's already been tried — a referral with no data attached goes back to "collect more data first," costing another two weeks.

## 5. Same-day data note template (what goes to the case manager)

```
[STUDENT] — [DATE] data check
Baseline: [N sessions, date range]: avg [X] [unit] ([range]).
Intervention: [strategy name, date range]: [period-by-period averages].
Change from baseline: [%] — goal is [threshold].
Today's individual data points: [list].
Recommend: [continue / adjust / escalate], because [one line of reasoning].
```
