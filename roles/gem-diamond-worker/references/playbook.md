# Playbook

## Inclusion-mapping-to-cutting-plan worksheet

Fill before finalizing any cutting plan.

| Field | Value |
|---|---|
| Rough stone | 3.80 ct diamond |
| Inclusion location (max-yield orientation) | ~1.2mm from projected girdle, crown-visible area |
| Crystal structure identified | Y — grain direction confirmed before planning |
| Max-yield cutting plan (ignoring inclusion) | 1.85 ct, inclusion crown-visible, projected clarity I1 |
| Clarity-optimized cutting plan (inclusion-aware) | 1.65 ct, inclusion under girdle, projected clarity VS1 |
| Yield sacrificed | 0.20 ct (~11%) |
| Estimated value: max-yield plan | 1.85 ct × ~$3,000/ct = $5,550 |
| Estimated value: clarity-optimized plan | 1.65 ct × ~$8,500/ct = $14,025 |
| Value difference | +$8,475 (+153%) favoring clarity-optimized plan |
| Client approved plan before cutting? | Y |

**Rule:** always calculate both a max-yield and a clarity-optimized scenario before committing — the "obvious" larger-carat option is not automatically the higher-value option once clarity grade differences are factored into per-carat pricing.

## Yield-vs-clarity trade-off table

| Scenario | Yield priority | Clarity priority | When appropriate |
|---|---|---|---|
| Inclusion small, easily hidden at minimal yield cost | Balanced — minor yield sacrifice for clarity gain | High | Most common favorable case |
| Inclusion large/central, unavoidable without major yield loss | May need to accept moderate clarity impact | Balanced | When clarity-optimization would sacrifice too much yield to be worthwhile |
| No significant inclusions present | Maximize yield | N/A — no trade-off needed | Clean rough material |
| Multiple inclusions requiring different optimal orientations | Requires prioritization decision | Requires prioritization decision | Complex case — model multiple scenarios, present trade-offs to client |

## Standardized grading criteria reference (illustrative — always use the specific applicable grading standard, e.g., GIA)

| Clarity grade | General criteria |
|---|---|
| FL/IF (Flawless/Internally Flawless) | No inclusions visible under 10x magnification |
| VVS1/VVS2 | Inclusions extremely difficult to see under 10x magnification |
| VS1/VS2 | Inclusions minor, difficult to see under 10x magnification |
| SI1/SI2 | Inclusions noticeable under 10x magnification |
| I1/I2/I3 | Inclusions visible to the naked eye, affecting transparency/brilliance |

**Rule:** always grade against the full, official standard's detailed criteria (this table is a simplified illustration) — never assign a grade based on general impression alone.
