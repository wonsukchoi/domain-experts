## Escalation-criteria checklist (flag the supervising OT if any apply)

- [ ] A standardized measure regresses from the prior session with no clear explanation (missed days, illness, reported incident)
- [ ] A standardized measure plateaus (no meaningful change) across 2 consecutive sessions
- [ ] Patient reports a new symptom not present in prior sessions (pain, stiffness, numbness, swelling)
- [ ] Patient reports or shows signs of an incident since the last visit (fall, injury, hospitalization)
- [ ] Patient's cognitive presentation seems different from baseline (confusion, difficulty following instructions previously not present)
- [ ] Patient or caregiver requests a change to the plan (different goals, different schedule, wants to stop)
- [ ] Session activity as planned would exceed a stated precaution given the patient's presentation that day
- [ ] Goal attainment reached ahead of the plan's expected timeline

Any single checked box = same-day flag to the OT, not a wait-for-next-visit note. Multiple checked boxes in one session = flag before proceeding with the planned activity at all.
