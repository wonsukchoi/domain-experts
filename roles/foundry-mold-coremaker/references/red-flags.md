# Red flags

### Pattern built to the part's nominal print dimension with no alloy-specific shrinkage allowance applied
- **Usually means:** the finished casting will come out undersized by roughly the shrink percentage for that alloy.
- **First question:** what shrink rule or percentage was actually applied for this specific alloy?
- **Data to pull:** the pattern build sheet compared against the alloy's standard shrink-rule table.

### Riser sized by visual habit rather than a modulus/solidification-time calculation
- **Usually means:** either wasted yield from an oversized riser, or shrinkage porosity in the casting from an undersized one.
- **First question:** does the riser's modulus actually exceed the modulus of the casting section it's meant to feed?
- **Data to pull:** the riser's modulus calculation compared against the casting section's modulus.

### Casting shows gas-related surface defects (blowholes, pinholes), and the entire sand mix is changed in response without isolating the cause
- **Usually means:** the actual root cause — moisture, core binder gassing, or vent inadequacy — goes unaddressed while unrelated variables get changed instead.
- **First question:** have sand moisture, core gassing, and vent path each been checked independently?
- **Data to pull:** sand test results (moisture, permeability) and core binder specification compared against the actual defect location.

### A gating system is reused from a different alloy's standard design without checking fill velocity/turbulence tolerance for the current alloy
- **Usually means:** turbulent-fill defects (oxide entrainment, air pockets) for an alloy more sensitive to turbulence than the one the gating was originally designed for.
- **First question:** does the sprue:runner:gate area ratio match this alloy's recommended fill-velocity range?
- **Data to pull:** the gating ratio compared against the specific alloy's recommended fill velocity.

### Casting yield percentage isn't calculated when evaluating a process or tooling change
- **Usually means:** cost comparisons are being made on scrap count or piece count alone, missing the actual cost-per-good-casting picture.
- **First question:** what's the actual good-casting weight versus total poured weight, including gates, risers, and scrap, for each process being compared?
- **Data to pull:** good casting weight and total poured weight for each process under comparison.

### Shrinkage porosity found in a casting section whose feeding riser theoretically solidifies after it, per the modulus calculation
- **Usually means:** the calculation's assumptions (pour temperature, mold material) didn't match real conditions, or the riser's feeding path was blocked.
- **First question:** did the actual pour temperature and timing match what the modulus calculation assumed?
- **Data to pull:** the actual pour temperature/timing log compared against the modulus calculation's assumptions.

### Dimensional deviation appears consistently across multiple castings from the same pattern, in the same direction
- **Usually means:** the shrink rule applied doesn't match this alloy or foundry's actual observed shrinkage, not random process variation.
- **First question:** how does the dimensional data across multiple castings compare to the shrink rule originally applied?
- **Data to pull:** dimensional inspection data across multiple castings compared against the shrink rule used to build the pattern.
