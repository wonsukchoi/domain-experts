# Clinical Nurse Specialist — playbook

Filled worksheets for the recurring processes a CNS runs: sphere triage, evidence appraisal, PDSA test-of-change, and hardwiring a win into structure.

## 1. Sphere-of-influence triage table

Use on intake of any problem, before designing an intervention. Route to the sphere the root cause actually lives in — most problems present in the patient sphere but originate in the system sphere.

| Symptom presented | Sphere it looks like | Sphere it usually is | Diagnostic question | Typical fix |
|---|---|---|---|---|
| One patient falls twice in a shift | Patient | Nurse practice (assessment gap) | Was a validated fall-risk score (e.g., Morse ≥45) documented and acted on? | Reassess + coach the assessing nurse |
| CLABSI rate 2x benchmark, multiple patients | Patient | System (supply/workflow) | Is the bundle step physically available at the bedside every shift? | Par-level/workflow redesign |
| New wound-care protocol charted but wounds not improving | Nurse practice | System (competency gap not remediated) | Was competency validated by return demonstration, or just an in-service attendance sheet? | Competency validation, not another training |
| Nurses say "we don't have time" for a new step | System | Nurse practice OR system (test whether time is real or a proxy for a different objection) | Time-motion audit: does the step add >2 min, or is compliance already <80% for an unrelated reason? | Redesign the step, or find the real objection |
| CNS asked to consult on individual patient repeatedly for the same issue | Patient | Nurse practice (skill not transferring) | Is this the 1st or the 6th consult on this issue this month? | If ≥3 in a month, escalate to a nurse-sphere education/competency fix |

**Threshold:** 3+ recurring instances of the same problem in a month is the line between "handle it as direct care" and "escalate to nurse- or system-sphere intervention."

## 2. Evidence appraisal quick-reference

Grade before spending credibility changing practice. Default action by grade:

| Evidence level | Example source type | Default action |
|---|---|---|
| I — Meta-analysis / systematic review of RCTs | Cochrane review, AACN/AORN evidence-based guideline | Adopt, adapted only for local resource constraints |
| II — Single well-designed RCT | Published multi-site trial | Adopt with a monitoring plan; check population match |
| III — Non-randomized, cohort/case-control | Multi-site QI collaborative data | Pilot before adopting; verify denominator match to local unit |
| IV — Single-center case series / QI report | One hospital's published outcomes | Pilot only, small scale; treat as hypothesis-generating |
| V — Expert opinion / vendor white paper / consensus statement without cited data | Manufacturer in-service material | Do not adopt on this basis alone; find the underlying study or downgrade to "watch" |

**Population-match check (always, regardless of level):** compare the source study's inclusion criteria (acuity, age range, device dwell time, comorbidity burden) against the local unit's actual patient mix before adopting. A Level I guideline built on a general med-surg population does not automatically transfer to a burn ICU or a pediatric oncology unit.

## 3. PDSA test-of-change worksheet (filled example)

**Problem:** CHG hub-scrub compliance observed at 60% (18/30), below the 80% threshold; charted compliance falsely showed 92%.

| Field | Entry |
|---|---|
| **Plan** — change to test | Increase CHG-cap par level from 2 boxes/shift to match measured consumption (~70 caps/day across 3 shifts → ~24/shift); relocate cart to bedside reach |
| **Plan** — scale | 1 pod (12 of 24 beds), 4 weeks |
| **Plan** — go/no-go threshold, stated before starting | ≥80% observed compliance on re-audit of 30+ episodes |
| **Do** — data collected | 33 direct observations of line-access episodes over 4 weeks |
| **Study** — result | 30/33 compliant = 91%, clears threshold |
| **Study** — outcome metric alongside process metric | Pod CLABSI count during test: 0 (n too small to be conclusive alone — track alongside compliance, not instead of it) |
| **Act** — decision | Scale to full unit; hardwire (see §4) rather than repeat the pilot |

**Rule of thumb for cycle length:** 2–4 weeks or ~30 observable episodes, whichever gives a large enough sample to trust the compliance percentage — a 5-observation pilot is noise, not a result.

## 4. Hardwiring / sustainment checklist, in preference order

Once a pilot clears its threshold, embed it in the structure most likely to survive turnover — try each in order and use as many as apply; a champion alone is never sufficient on its own.

1. **Standing order / order set change** — the change is now the default, not an opt-in. (Used in the CLABSI example: par level became a standing supply order, not a verbal agreement with the charge nurse.)
2. **Documentation flowsheet field** — the step has a required field, making non-compliance visible in the chart rather than invisible.
3. **Unit-based council ownership** — the council, not the CNS, owns the monthly audit and reports out; survives the CNS moving to the next project.
4. **Competency checklist / onboarding inclusion** — new hires are validated on the new standard from day one, not retrofitted.
5. **Staffing/supply par-level policy change** — the physical or staffing environment defaults to supporting the change.
6. **Named champion + recurring reminder** *(lowest-durability option — use only as a bridge to one of the above, never as the final state)*.

**Test before declaring a change hardwired:** would compliance survive the CNS taking a 4-week leave and the original champion transferring to another unit? If either answer is no, it isn't hardwired yet.

## 5. Quarterly outcome report table (filled example)

| Metric | Baseline (6 mo prior) | Post-intervention (this quarter) | Benchmark | Cost avoided |
|---|---|---|---|---|
| CLABSI rate (medical ICU) | 2.0 / 1,000 line-days (4 events / 2,002 line-days) | 0 / 1,000 line-days (0 events / 1,001 line-days) | 0.6–1.4 / 1,000 (NHSN adult ICU) | ~$91,628 (2 expected events avoided × $45,814) |
| Bundle compliance (direct observation) | 60% (18/30) | 91% (30/33) | ≥80% (internal threshold) | n/a — process metric |

Report this table, not a narrative, to the sponsoring nurse manager/CNO each quarter — a table survives being forwarded without the CNS in the room to explain it.
