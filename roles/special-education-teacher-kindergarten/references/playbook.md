# Playbook

## IFSP-to-IEP goal-translation worksheet

Fill for every goal carried over from an IFSP or a prior developmental plan — never copy a developmental goal directly into the IEP.

| Field | Example entry |
|---|---|
| IFSP goal (developmental language) | "Will improve fine motor coordination" |
| Underlying kindergarten-referenced skill | Pencil grip and letter formation for daily writing tasks |
| Baseline (observed/measured) | Traces 2 of 10 uppercase letters legibly (week 2 classroom sample) |
| Rewritten IEP goal (measurable, standards-referenced) | "Given a model, student will independently trace and copy 8 of 10 uppercase letters legibly by [date]" |
| Progress-monitoring method | Weekly writing-sample rubric score, 1-4 scale |
| Data collector | Special education teacher, during daily writing block |
| Mastery criterion | 8/10 legible across 3 consecutive weekly samples |

**Rule:** if a goal cannot be restated as an observable kindergarten task with a baseline number and a target number, it is not ready to enter the IEP — send it back for another present-levels data point.

## Skill-vs-performance-deficit probe protocol

Run before writing any goal that addresses a "won't do it" classroom report.

| Step | Action | Decision rule |
|---|---|---|
| 1 | Pull the whole-group/classroom accuracy or participation rate | If already at or near mastery threshold in whole-group setting, stop — no deficit, adjust expectations instead |
| 2 | Run the same task 1:1, no audience, with a model provided, across 2-3 sessions over 1-2 weeks | Record raw accuracy each session |
| 3 | Compare whole-group rate to 1:1 rate | Gap of 30+ percentage points favoring 1:1 → performance deficit is present (may co-occur with a smaller skill gap) |
| 4 | If 1:1 rate is still below the grade-level mastery threshold | Residual gap to threshold is the genuine skill target — write a skill goal for that portion only |
| 5 | If 1:1 rate meets or exceeds mastery threshold | Write a participation/self-regulation goal only — reteaching content already mastered wastes instructional time |
| 6 | Document which condition (group size, novelty, adult proximity, sensory load) most reduced performance | Use as the intervention lever in the participation goal (e.g., reduced-group pull-out, proximity seating, movement break before the task) |

**Rule:** never write a single combined "skill and behavior" goal from this protocol — the two goals have different progress-monitoring cadences and different intervention owners, and combining them hides which lever actually moved the number.

## LRE placement decision table

| Present-levels evidence | Placement default | Support level | Reassess trigger |
|---|---|---|---|
| Accesses core instruction with accommodations only (extended time, visual supports, seating) | General-education classroom, no pull-out | Consultative (monthly check-in) | Grades/progress data drop 2 consecutive reporting periods |
| Accesses core instruction with accommodations + targeted small-group support | General-education classroom + pull-out or push-in | Direct service, 20-60 min/day | 1:1 probe shows skill closing faster/slower than modeled |
| Cannot access core instruction even with accommodations and small-group support, confirmed across 2+ data points | Resource room for the affected content area(s) only | Direct service, majority of that subject block | Skill gap closes to within 1 grade-level band of peers |
| Cannot access any core instruction across all content areas, confirmed across 2+ data points and 2+ observers | Self-contained classroom | Full-day specialized instruction | Any single content area shows sustained gains toward general-education access |

**Rule:** always start from the least-restrictive row the present-levels data supports, document the specific evidence for that row, and set an explicit reassess trigger — a first placement made "to be safe" tends to persist through elementary school even after the child no longer needs it.
