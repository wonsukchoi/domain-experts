# Playbook

## Coldest-point lethality verification worksheet

Fill for any cook cycle with a food-safety lethality requirement.

| Field | Value |
|---|---|
| Product | Ready-to-Eat Chicken |
| Lethality requirement | 165°F held ≥15 sec at coldest point |
| Probe location | Center of thickest/largest piece in batch |
| Time coldest point reached 165°F | t = 42 min |
| Time coldest point held ≥165°F | t = 42-45 min (3 min hold) |
| Meets ≥15-sec requirement? | Y (far exceeds minimum) |
| Surface spot-check (secondary) | 168-172°F, 4 points |
| Sensor calibration status | Current (verified 2026-07-10) |
| Batch disposition | Release — lethality confirmed via data log |

**Rule:** always verify via coldest-point data logging, not a surface spot-check — a spot-check confirms a single moment/location, not the sustained coldest-point profile the requirement is based on.

## Oil condition monitoring table (illustrative — always use the specific oil/product's validated thresholds)

| Indicator | Acceptable range | Action if exceeded |
|---|---|---|
| Free fatty acid (%) | Below product-specific threshold (commonly <1-2%) | Replace oil |
| Polar compounds (%) | Below product-specific threshold (commonly <24-27%) | Replace oil |
| Smoke point drop | Significant drop from fresh oil baseline | Investigate, consider replacement |
| Color (visual, secondary indicator only) | Darkening trend | Use as early-warning signal, confirm with actual test before action |

**Rule:** use actual chemical/physical test indicators as the basis for oil replacement decisions — elapsed time or color alone are secondary/early-warning signals, not the deciding factor.

## Batch parameter adjustment guide

1. Check incoming batch characteristics (moisture content, viscosity, particle size) against the process's validated baseline.
2. If characteristics differ meaningfully from baseline, adjust cook time/temperature/mixing parameters accordingly rather than running the standard process unchanged.
3. Document the adjustment made and the batch characteristic that triggered it.
4. Verify the adjusted process still meets both lethality (if applicable) and quality requirements via testing, not assumption.
5. Feed batch characteristic and adjustment data back to process engineering if a pattern develops (e.g., a specific supplier's material consistently requiring adjustment) to consider updating the standard process baseline.
