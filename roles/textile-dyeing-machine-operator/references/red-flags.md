# Red flags

### Dye recipe scaled to a different liquor ratio using the same absolute dye quantity without recalculating actual bath concentration
- **Usually means:** an off-shade or uneven result from altered exhaustion kinetics, even though the %owf figure is technically correct.
- **First question:** what's the actual bath concentration at this liquor ratio versus the validated recipe's original concentration?
- **Data to pull:** liquor ratio used compared against validated liquor ratio, and the resulting bath concentration comparison.

### Shade drift in a multi-dye recipe corrected by broadly adjusting the whole recipe rather than diagnosing which specific dye component is over/under-exhausting
- **Usually means:** the correction doesn't actually address the root cause and may introduce a new imbalance.
- **First question:** what do the individual dye component exhaustion readings show against the standard?
- **Data to pull:** individual dye component exhaustion readings compared against the standard, not just overall shade comparison.

### Uneven dye uptake or shade defect investigated as a dyeing-process issue without first checking whether bleaching/preparation met its spec
- **Usually means:** the root cause traces back to an earlier process stage that dye-recipe adjustments can't fix.
- **First question:** what did the bleaching stage's whiteness/absorbency test actually show for this batch?
- **Data to pull:** bleaching stage whiteness/absorbency test results compared against spec, for the affected batch.

### A previously successful dye recipe is reproduced with matching percentages but no verification that water quality or temperature profile matches the original run
- **Usually means:** an off-shade result despite an apparently identical recipe.
- **First question:** does water quality (hardness, pH) and temperature ramp/hold match the original successful batch's documented conditions?
- **Data to pull:** water quality and temperature profile compared against the original successful batch's documented conditions.

### Exhaustion rate reads lower than expected for a given dye/temperature/time combination, and the dye lot is assumed defective without checking process variables first
- **Usually means:** a process variable — concentration, electrolyte, pH — is the actual cause, not the dye itself.
- **First question:** do liquor ratio and auxiliary chemical levels actually match process specification?
- **Data to pull:** liquor ratio and auxiliary chemical levels compared against process specification, before the dye lot is flagged.

### A dye batch record doesn't specify actual liquor ratio, %owf, and temperature profile used, only the target recipe
- **Usually means:** a later shade-reproducibility investigation lacks the information needed to identify what actually changed between batches.
- **First question:** what fields does the batch record actually capture versus the standard required documentation?
- **Data to pull:** batch record completeness compared against the standard required documentation fields.

### A recurring shade or evenness defect pattern appears across multiple batches on a specific machine without being traced to a specific process cause
- **Usually means:** a systemic equipment or process issue — liquor ratio consistency, temperature uniformity, water supply — rather than isolated batch-to-batch variation.
- **First question:** does the defect occurrence log correlate with a specific machine or process parameter trend?
- **Data to pull:** defect occurrence log by machine compared against process parameter logs over the same period.
