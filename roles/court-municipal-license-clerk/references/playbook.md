# Court, Municipal, and License Clerk — Playbook

## Fee-schedule worksheet (small-claims example, filled)

| Line item | Applies? | Amount |
|---|---|---|
| Base filing fee | Always | $75.00 |
| Sheriff service-of-process fee | If sheriff service requested | $40.00 |
| Case-history record-search fee | If requested | $10.00 |
| **Total due** | — | **Sum of applicable lines** |

The total is never a flat number memorized from habit — recompute from the specific case's checked boxes every time. A filer requesting self-service and no record search owes $75.00, not $115.00.

## Deficiency-notice checklist (complete-in-one-pass)

Before sending any deficiency notice, verify the following are ALL checked in the same review pass — never send a second notice for something that should have been caught in the first:

1. Required signature(s) present on every page requiring one.
2. All required attachments present (proof of service, supporting exhibits, prior order references).
3. Fee paid matches the itemized total for the specific boxes checked.
4. Case caption/parties match the case number if this is a filing into an existing case.
5. Form version is current (some jurisdictions reject outdated form revisions).

## Fee-waiver eligibility checklist

| Path | Threshold | Documentation required |
|---|---|---|
| Income-based | Household income ≤ jurisdiction's stated % of Federal Poverty Guideline (verify current-year figure and local multiplier) | Pay stubs or tax return, household size affidavit |
| Categorical (public-benefits) | Enrollled in a qualifying program (e.g., SNAP, Medicaid, TANF — verify local qualifying-program list) | Current enrollment letter/card |

Either path independently qualifies — do not require both. Deny only if neither path's documentation is provided or verifiable.

## Public-access screening checklist (before releasing any record)

1. Confirm the case's specific confidentiality/seal flag in the case-management system — do not infer from case type.
2. If the case type is one that's *sometimes* sealed by statute (juvenile, certain family-law, expunged records), confirm the flag explicitly even if it doesn't appear sealed on the surface.
3. If a record is partially sealed (e.g., a financial affidavit within an otherwise public divorce file), release only the unsealed portions — never the full file by default.
4. Log every records release request and outcome for audit purposes.

## License/permit eligibility checklist template

| Item | New application | Renewal (within grace period) | Renewal (lapsed past grace period) |
|---|---|---|---|
| Application form | Required | Required | Required |
| Fee | Full fee | Renewal fee (lower) | Full fee (treated as new) |
| Supporting credentials | All required, current | Proof of continued validity | All required, current — re-qualify from scratch |
| Background check | Required | Not required if within grace period | Required |

A lapsed renewal moves to the "new application" column regardless of the applicant's prior history — the checklist track is determined by elapsed time against the stated grace period, not by applicant intent or prior good standing.
