# Nurse Practitioner — Playbook

## Clinical decision rule: McIsaac-modified Centor score (pharyngitis)

| Criterion | Points if present |
|---|---|
| Tonsillar exudate | +1 |
| Tender anterior cervical adenopathy | +1 |
| Fever >38°C by history or exam | +1 |
| Absence of cough | +1 |
| Age 3-14 | +1 |
| Age 15-44 | 0 |
| Age ≥45 | -1 |

| Score | Estimated strep probability | Action |
|---|---|---|
| ≤1 | 1-10% | No testing, no antibiotics |
| 2-3 | 11-35% | RADT (rapid antigen test); treat if positive |
| 4+ | 38-63% | RADT indicated; some protocols support empiric treatment pending culture if RADT unavailable |

## Referral-vs-manage triage worksheet

| Presentation | Manage in-scope if | Refer/escalate if |
|---|---|---|
| Chest pain | Reassuring exam, low-risk score (e.g., HEART score ≤3), no red-flag findings | Any cardiac-risk-factor cluster, abnormal EKG, or elevated troponin |
| New murmur | Asymptomatic, known benign pattern (e.g., physiologic flow murmur in a child) | New murmur in adult, symptomatic, or grade ≥3/6 |
| Diabetic foot lesion | Superficial, no signs of infection, adequate perfusion (palpable pulses) | Any sign of infection, absent pulses, or ulcer present >2 weeks without improvement |
| Single joint pain/swelling | No fever, no recent trauma, low suspicion for septic joint | Fever, inability to bear weight, or high suspicion for septic arthritis (requires same-day arthrocentesis referral) |
| New neurological deficit | — (default: none manage in-scope without imaging) | Any focal deficit, sudden onset, or suspected stroke — immediate ED referral |

## Medication reconciliation checklist

1. Pull current pharmacy-fill or e-prescribing history if available — do not rely on patient self-report alone as the primary source.
2. Ask directly about OTC medications, supplements, and as-needed prescriptions from other providers — these are the most commonly under-reported.
3. Cross-check new prescription against the reconciled list for interactions (drug-drug, drug-disease).
4. Confirm allergy list is current and specific (reaction type, not just "allergic") before prescribing.
5. Document reconciliation explicitly in the note — "med list reconciled against pharmacy record" — not implied by the medication list being present.
