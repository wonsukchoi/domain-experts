# Red flags

### A last used for an extended period without a wear inspection
- **Usually means:** a developing dimensional deviation could be producing a systematic sizing/fit defect across every unit made on it.
- **First question:** when was this specific last last inspected for wear, and how many production cycles has it run since?
- **Data to pull:** last wear inspection log, cycle count since last inspection.

### Sole bond quality confirmed only by visual inspection at end of batch
- **Usually means:** actual bond strength hasn't been verified, and a visually normal-looking sole can still have inadequate pull strength.
- **First question:** was pull/peel strength testing performed at intervals during the run, or only a visual check at completion?
- **Data to pull:** the sampling plan's specified interval, actual testing performed.

### Lasting tension assumed correct because the machine is cycling normally
- **Usually means:** tension hasn't actually been verified against the specified range for the current material.
- **First question:** does current lasting tension match the specified range for this specific material (leather vs. synthetic have different requirements)?
- **Data to pull:** lasting tension setting/measurement, material-specific specification.

### A defect discovered at end-of-batch inspection with no in-process sampling data to narrow down when it began
- **Usually means:** the full batch quantity may need investigation, when in-process sampling could have narrowed the affected range.
- **First question:** does any in-process sampling data exist to identify roughly when the defect began, or is the full batch the only reference point?
- **Data to pull:** in-process sampling records if any exist, defect characteristics.

### A machine parameter (temperature, pressure, tension) not re-verified after a material or style change
- **Usually means:** settings validated for a different material may not be appropriate for the new one.
- **First question:** were parameters re-verified for this specific material/style combination, or carried over from the prior run?
- **Data to pull:** current material/style specification, parameter verification record for this run.

### A containment scope set to the full batch when in-process sampling data could narrow it to a smaller affected range
- **Usually means:** unnecessary inspection/rework effort is being applied broadly when the actual affected quantity is smaller and identifiable.
- **First question:** does sampling data show when the defect actually began, allowing containment to be scoped to units produced since that point?
- **Data to pull:** in-process sampling timeline, defect onset estimate.

### A recurring fit complaint pattern across multiple pairs from the same last/size
- **Usually means:** a systematic last wear or dimensional issue rather than isolated manufacturing variation.
- **First question:** do the complaints correlate with a specific last/size, or are they distributed randomly?
- **Data to pull:** complaint pattern by last/size, last wear inspection history for the implicated tooling.

### A heater, pressure system, or other process control component not recently verified against its calibration/specification
- **Usually means:** actual process parameters may be drifting from setpoint without detection until a quality issue appears.
- **First question:** when was this component last verified against its actual output (not just its displayed setpoint)?
- **Data to pull:** calibration/verification log for the specific process control component.

### A new last or tooling put into production without a first-article dimensional check
- **Usually means:** a manufacturing defect in the new tooling itself could go undetected, producing a systematic issue from the very first unit.
- **First question:** was a first-article part/pair produced and dimensionally verified against the last's specification before full production began?
- **Data to pull:** first-article inspection record for the new tooling.

### A quality issue addressed by correcting the process going forward without checking already-produced units for the same defect
- **Usually means:** units made before the correction may still carry the defect and haven't been identified for containment.
- **First question:** have units produced since the defect's likely onset (not just going forward) been flagged and checked?
- **Data to pull:** production sequence log, containment scope defined for the correction.
