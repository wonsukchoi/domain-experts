# Dental Assistant — Playbook

## Four-handed instrument-transfer sequence (Class II composite restoration, right-handed operator)

| Step | Procedure moment | Instrument transferred | Transfer zone |
|---|---|---|---|
| 1 | Anesthesia | Topical, then syringe | Transfer zone, syringe capped/recapped one-handed |
| 2 | Isolation | Rubber dam clamp + forceps, or retraction cord | Transfer zone |
| 3 | Preparation | High-speed handpiece | Transfer zone, pre-positioned before dentist reaches |
| 4 | Prep check | Explorer | Transfer zone |
| 5 | Matrix placement | Matrix band + retainer, wedge | Transfer zone, wedge pre-selected by gap size |
| 6 | Bonding | Etch syringe → rinse/suction → bonding brush → curing light | Transfer zone, curing light staged before etch starts (working-time clock begins at etch) |
| 7 | Restoration | Composite syringe (shade pre-selected) → placement instrument → curing light | Transfer zone |
| 8 | Finishing | Finishing burs, then articulating paper/forceps | Transfer zone |

Rule of thumb: anything the dentist will need in the next 15-20 seconds should already be in hand or staged at the transfer zone — waiting for a verbal request adds a stall on every step.

## Radiograph retake decision table

| Error signature | What it looks like | Likely cause | Fix before retake |
|---|---|---|---|
| Elongation | Image stretched, roots appear too long | Vertical angulation too shallow (flat) | Increase vertical angulation |
| Foreshortening | Image compressed, roots appear too short/stubby | Vertical angulation too steep | Decrease vertical angulation |
| Cone-cut | Clear or dark unexposed band across part of image | PID (cone) misaligned with sensor/film | Re-align PID directly over sensor before re-shooting |
| Overlap | Adjacent teeth's proximal surfaces superimposed | Horizontal angulation error | Correct horizontal angle to be perpendicular to the arch at that point |
| Blurring | Image soft/unclear | Patient or sensor movement during exposure | Re-stabilize sensor, re-instruct patient to hold still |

Only retake after identifying which row applies — retaking on a guess repeats the same error at a second dose roughly half the time.

## Chairside material working-time reference (illustrative — confirm against current manufacturer data sheet)

| Material | Typical working time from mix start | Note |
|---|---|---|
| Alginate impression material | ~1.5–2 minutes | Faster in warm water, slower in cold — adjust water temp deliberately, don't just "go faster" |
| Temporary cement (zinc oxide eugenol) | ~2–3 minutes | Humidity/temperature-sensitive |
| Glass ionomer luting cement | ~1.5–2.5 minutes | Working time and setting time are different clocks — don't confuse them |
| Bonding agent (light-cured) | No working-time clock until cured — cure promptly per manufacturer's light-exposure time | Uncured bonding agent left exposed to ambient light too long can partially set |

Never extend a chemical-cure material's stated working time by adding water/liquid — this changes the material's set properties, not just the clock.
