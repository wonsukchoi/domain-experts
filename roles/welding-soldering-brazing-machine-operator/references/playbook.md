# Playbook

## Electrode wear tracking worksheet

Fill and update continuously for any resistance welding application.

| Field | Value |
|---|---|
| Application | Job #RSW-2291, 2-sheet 0.040"+0.040" steel |
| Weld schedule | 9,000A / 12 cyc / 450 lb |
| Nugget diameter spec | ≥0.180" |
| Initial tip diameter | 0.200" |
| Dressing interval (weld count) | Every 2,500 welds |
| Weld count since last dressing | track continuously |
| Peel test sampling interval | Every 500 welds |
| Last peel test result | record nugget diameter |
| Trend (last 3 tests) | record for drift detection |

**Rule:** dress/replace electrodes on the weld-count schedule regardless of visual tip appearance — visible wear lags behind current-density-affecting wear.

## Weld schedule re-qualification checklist

1. Identify any change from the previously validated configuration: material type, thickness, coating, stack-up/layer count.
2. If any of these changed, treat the schedule as unqualified for the new configuration.
3. Run a qualification test series: produce sample welds at proposed schedule parameters.
4. Perform destructive testing (peel/chisel) on qualification samples to confirm nugget diameter/strength meets spec.
5. Only after qualification samples pass, release the schedule for production use with this configuration.
6. Document the qualification test results and the specific configuration the schedule is now validated for.

## Destructive/non-destructive testing sampling table (illustrative — always use the job's specific quality plan)

| Joint criticality | Testing method | Typical sampling rate |
|---|---|---|
| Non-critical/cosmetic | Visual inspection | 100% visual, periodic destructive spot-check |
| Structural, non-safety-critical | Peel/chisel test | Periodic (e.g., every 500 welds or per shift) |
| Safety-critical or pressure-critical | NDT (ultrasonic/radiographic) | 100% inspection, or per applicable code requirement |

**Rule:** always confirm the actual sampling rate and method against the specific job's quality plan or applicable industry code — this table shows general risk-based tiering, not universal values.
