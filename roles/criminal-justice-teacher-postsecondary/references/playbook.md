# Criminal Justice and Law Enforcement Teacher, Postsecondary — Playbook

## Crime-rate assignment rubric (filled example — burglary trend, two precincts)

| Element | Requirement | Why it's scored separately |
|---|---|---|
| Raw counts reported | Year 1: 1,200; Year 4: 1,350 (Precinct A only) | Baseline data, not yet a claim |
| Population for each year | Year 1: 150,000; Year 4: 162,000 | Missing this is the single most common omission |
| Rate per 100,000 computed | Year 1: 1,200/150,000×100,000 = 800; Year 4: 1,350/162,000×100,000 = 833.3 | Required before any trend claim — 3 pts |
| % change stated on rate, not count | (833.3−800)/800 = 4.2%, not the raw-count (1350−1200)/1200 = 12.5% | Scored 0 if the memo leads with the count-based %; the count-based number isn't wrong data, it's the wrong comparison |
| NIBRS-transition note if series crosses Jan 1, 2021 | Explicit sentence flagging the hierarchy-rule change | Automatic point loss if the series crosses the cutover with no note |
| Comparison jurisdiction or baseline trend | At least one, or an explicit scope limitation ("no causal claim, single jurisdiction") | Missing both + a causal verb ("caused," "proves") = revise-and-resubmit |

## IRB Subpart C applicability addendum (filled example — parolee reentry-services survey)

1. **Subject population, stated precisely**: "Adults within 12 months of release from state prison, currently under parole supervision, recruited through a reentry services nonprofit — not currently confined."
2. **Subpart C technical applicability**: "45 CFR 46.303(c) defines 'prisoner' as involuntarily confined or detained in a penal institution. Subjects are not confined, so Subpart C's formal requirements do not strictly apply."
3. **Heightened-scrutiny flag raised anyway**: "Subjects remain under state supervision, and participation or non-participation must not be visible to, or reportable to, parole officers. Consent process explicitly states the researcher has no reporting relationship with the parole system and no incentive/coercion tied to supervision status."
4. **Data security specific to this population**: "No participant name is linked to parole case number in the dataset; recruitment nonprofit holds the crosswalk, not the research team."
5. **IRB determination requested**: Expedited review with the above heightened-scrutiny measures documented, rather than exempt — flagging it up front rather than letting the board discover the supervision status and kick it to full board after submission.

## Ride-along / corrections-practicum placement packet (filled example)

| Document | Content | Who signs |
|---|---|---|
| Background-check clearance | State/FBI fingerprint-based check, dated within 90 days of placement start | Student; confirmed in writing by host agency |
| Liability waiver | University's standard field-placement waiver, naming the specific agency and date range | Student, university risk-management office |
| Agency rules of engagement | Agency-specific: what the student may observe, whether recording/photos are permitted, no discussion of case details with anyone outside the ride, no social-media posting about the shift | Student, host agency supervisor |
| Confidentiality briefing acknowledgment | Signed statement acknowledging juvenile-record and victim-identifying-information confidentiality rules specific to the state | Student |
| Faculty point of contact | Named instructor and phone number the agency can reach same-day for any incident | Faculty (on file with agency) |

Every element gets confirmed *before* the term starts — a placement roster with any student missing a dated clearance is a hold, not a note for later.

## ACJS certification curriculum-mapping table (filled example — partial, BA program)

| ACJS standard element | Course(s) mapped | Credit hours | Gap? |
|---|---|---|---|
| Criminal justice systems and processes | CJ 101 Intro to CJ, CJ 210 Courts | 6 | No |
| Criminology and theories of crime | CJ 220 Criminology | 3 | No |
| Law enforcement | CJ 230 Policing in America | 3 | No |
| Corrections | CJ 240 Corrections | 3 | No |
| Research methods and statistics | CJ 310 Research Methods, STAT 200 | 6 | No |
| Ethics | CJ 480 Ethics in CJ | 3 | No |
| Practicum / applied experience | CJ 495 Practicum | 3 | Flagged — practicum learning outcomes not yet mapped to a rubric ACJS reviewers can score; add before next self-study cycle |

Run this crosswalk annually, not only at self-study time — ACJS certification is voluntary, but a gap found by the program is a curriculum-committee item; the same gap found by an external reviewer during the certification review reads as a program that doesn't track its own standards.

## POST lesson-plan hour crosswalk (filled example — community-college basic academy, partial)

| State POST-mandated topic | Minimum hours (per state training standards) | Hours in course calendar | Gap? |
|---|---|---|---|
| Use of force / defensive tactics | 60 | 60 | No |
| Firearms | 80 | 80 | No |
| Criminal law and procedure | 40 | 42 | No (2 hrs surplus, acceptable) |
| Cultural diversity / bias-based policing | 16 | 12 | Yes — 4-hour shortfall; add before next cohort's state audit |
| Emergency vehicle operations | 40 | 40 | No |
| Report writing | 24 | 24 | No |

A shortfall here isn't a syllabus preference question — mandated hours are a floor set by the state training standards body, and a cohort that graduates short of them can have its academy certification, and the cohort's POST eligibility, questioned on audit.
