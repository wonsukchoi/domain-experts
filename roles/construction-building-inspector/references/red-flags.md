### A dimension or measurement is within ~5% of the code minimum
- **Usually means:** Could be a genuine pass or fail depending on measurement precision — worth re-measuring with the actual instrument used (tape measure, not visual estimate) before recording either way.
- **First question:** Was this measured directly with a tape measure or similar tool, and is the exact figure recorded, or is "looks about right" being treated as a measurement?
- **Data to pull:** The measurement itself with tool used, the exact code section and its numeric threshold, a photo with the measuring tool in frame.

### A trade crew is scheduled on-site the same day or next day as a hold-point inspection
- **Usually means:** Legitimate schedule pressure on the contractor, not new information about the state of the work — the inspection standard doesn't change because a crew is waiting.
- **First question:** Does this specific finding meet code as installed, independent of what happens to the schedule if it fails?
- **Data to pull:** The inspection checklist for this hold point, independent of any statement about tomorrow's crew.

### A contractor cites a prior inspector's approval of a similar condition as the basis for passing this one
- **Usually means:** Either a genuinely settled interpretation worth checking against the current adopted code, or a precedent from a different code cycle, a different inspector's error, or a condition that isn't actually identical.
- **First question:** Is the current adopted code edition (with local amendments) the same one that governed the prior approval, and is the condition actually identical, not just similar?
- **Data to pull:** The prior inspection record if available, the code edition in force at the time of that approval versus now, the specific dimensions/conditions of both cases.

### A hold-point inspection is requested out of sequence (e.g., insulation requested before framing has passed)
- **Usually means:** Either a scheduling error, or an attempt to compress the schedule by combining inspections that exist specifically to catch different things at different points.
- **First question:** Has the prerequisite hold-point inspection for this phase actually passed, and is there a dated record of that approval?
- **Data to pull:** Inspection log/permit software record showing pass/fail history for all prior hold points on this permit.

### A life-safety item (egress, smoke/CO detector, GFCI/AFCI, fire separation) fails by what looks like a small margin
- **Usually means:** Unlike cosmetic or convenience items, a small-looking margin on a life-safety item is not automatically low-severity — a 10–15% shortfall on egress net clear area is still a meaningful reduction in an occupant's ability to escape a fire.
- **First question:** Does this item fall into the zero-tolerance life-safety category, and if so, has it been measured and documented with the same rigor as a large failure would get?
- **Data to pull:** Exact measurement, the applicable code section's numeric minimum, classification of the item as life-safety versus standard.

### The as-built condition doesn't match the stamped plan set
- **Usually means:** Either a harmless field adjustment (dimension shift with no structural or life-safety effect) or a substantive deviation (header size, room use, egress path) that needs an engineering or plan-review determination, not a field call.
- **First question:** Does resolving this discrepancy require a structural or code-intent judgment, or is it a purely factual as-built note with no such effect?
- **Data to pull:** The stamped plan set, the as-built measurement or condition, prior field-change approvals on this permit if any exist.

### A correction notice is written without a specific code section citation
- **Usually means:** The finding is being recorded as a general impression rather than a documented violation, which is unlikely to survive if the contractor appeals it.
- **First question:** What is the exact section of the adopted code (with local amendment number if applicable) that this finding violates?
- **Data to pull:** The adopted code edition and local amendments for this jurisdiction, the specific section text.

### A facility or contractor requests a stop-work order be lifted before the corrective item is reverified
- **Usually means:** Schedule pressure to resume work based on a verbal assurance rather than a completed re-inspection.
- **First question:** Has the specific corrected item actually been re-inspected and passed, with a dated record, or is the request based on the contractor's own statement that it's fixed?
- **Data to pull:** Original stop-work order and cited violation, re-inspection request date and result if one has occurred.

### A re-inspection fee or schedule delay is raised as a reason to reconsider a failed finding
- **Usually means:** The contractor's cost pressure is real but has no bearing on whether the installed work meets code — it's a business fact about the contractor's project, not evidence about the work.
- **First question:** Does the finding change if the fee schedule and calendar are set aside entirely?
- **Data to pull:** The original finding's measurement/condition and code citation, independent of the fee schedule or delay being cited.
