# Red flags

### Multiple stains of different chemistry on the same item
- **Usually means:** treatments will need to be sequenced and neutralized between steps to avoid a chemical interaction or a residue-driven set of the second stain.
- **First question:** what chemistry category is each stain (protein, tannin, oil-based, dye/ink), and does the treatment order avoid mixing incompatible residues?
- **Data to pull:** stain identification notes from intake, treatment product compatibility guidance for each chemistry involved.

### A garment with dark or saturated color and no established processing history at this facility
- **Usually means:** colorfastness hasn't been verified, and dye bleed risk is unknown.
- **First question:** has a spot/bleed test been run on an inconspicuous area of this specific item?
- **Data to pull:** intake notes on colorfastness testing, any prior processing history for this exact item or a comparable one from the same source.

### A blended-fiber item treated with a stain remover formulated for a single-fiber garment
- **Usually means:** the treatment concentration or chemistry wasn't adjusted for the more sensitive fiber present in the blend.
- **First question:** what is the item's actual fiber composition, and does the treatment product have blended-fiber guidance?
- **Data to pull:** garment fiber content (from label or fabric test), treatment product's fiber-compatibility guidance.

### Heat (pressing, hot drying) about to be applied to an item with any stain not confirmed fully removed
- **Usually means:** the stain-treatment stage hasn't been formally closed out before the item moved to finishing.
- **First question:** has this item been inspected (including under UV light if protein staining is a possibility) to confirm no residual staining before heat is applied?
- **Data to pull:** treatment log for this item, finishing-stage sign-off status.

### Existing damage or wear noticed during processing with no matching intake documentation
- **Usually means:** the pre-processing inspection either wasn't done or wasn't documented for this item.
- **First question:** does the intake record note this specific damage, or is this the first time it's been observed?
- **Data to pull:** intake documentation/photos for this item, processing log for when the damage was first noticed.

### A care label instruction that conflicts with the item's actual visible trims or embellishments
- **Usually means:** the label reflects the base fabric's manufacturer testing and doesn't separately address an added component (beading, appliqué, adhesive trim).
- **First question:** is the trim/embellishment rated for the same process the base fabric's label specifies, or does it need separate consideration?
- **Data to pull:** care label instructions, visual inspection notes on trims/embellishments.

### A solvent or wash method selected by garment category default rather than the item's specific construction
- **Usually means:** the facility's standard process for "this type of garment" was applied without checking this specific item's fiber, finish, or construction details.
- **First question:** does this item's actual fiber/finish/construction match what the default process assumes, or does it differ in a way that matters?
- **Data to pull:** item's fiber content and construction notes, the facility's default process assumptions for this garment category.

### A customer complaint about a defect discovered only after pickup
- **Usually means:** either a genuine processing-caused defect, or a pre-existing condition not documented at intake — the distinction depends entirely on whether intake documentation exists.
- **First question:** does the intake record note this area's condition before processing?
- **Data to pull:** intake documentation/photos, processing log for the specific area in question.

### An unfamiliar or unusual fabric/construction with no clear precedent at this facility
- **Usually means:** standard protocols may not directly apply, and extra caution (testing, conservative treatment) is warranted before proceeding.
- **First question:** has this exact fabric/construction type been processed successfully before, here or with documented external guidance?
- **Data to pull:** any manufacturer or fiber-content documentation available, professional reference guidance for this fabric type.

### A stain that reappears or "ghosts" after initial treatment and drying
- **Usually means:** the stain wasn't fully removed at the molecular level even though it was no longer visible when wet — a common signature of an incompletely treated oil-based or protein stain.
- **First question:** was the stain confirmed removed while still wet, or just assumed removed because it wasn't visible?
- **Data to pull:** treatment log, post-treatment inspection notes before drying/pressing.
