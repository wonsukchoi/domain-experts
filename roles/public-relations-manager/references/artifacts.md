# Crisis communication artifacts

Filled templates, not descriptions of templates — populate with real numbers as-is.

## 1. Disclosure cost-comparison model (filled — see Worked example in SKILL.md for full narrative)

| | Proactive disclosure | Delayed/minimizing then forced confirmation |
|---|---|---|
| Direct recall cost (fixed either way) | $4,200,000 | $4,200,000 |
| Estimated reputational impact | 3-5% short-term stock impact, recovers within a quarter | 15-20% stock impact, slower recovery |
| Additional legal/settlement exposure | Minimal beyond recall itself | Estimated $8M-$15M (concealment treated more punitively) |
| **Total estimated exposure** | **~$4.2M + modest recoverable reputational cost** | **~$12.2M-$19.2M** |

**Rule applied:** the recall cost is identical either way — the decision that actually changes total exposure is the communication timing and honesty, not the underlying facts of the defect itself.

## 2. Message house (filled example)

| Element | Content |
|---|---|
| Core facts (identical across all audiences) | 1,360 of 340,000 units affected by overheating defect; 12 reported minor injuries, no deaths; recall and free replacement in progress; investigation began 3 weeks ago |
| Customer-facing emphasis | How to check if their unit is affected, how to get a free replacement, safety steps in the meantime |
| Investor-facing emphasis | Estimated financial impact ($4.2M recall cost), no material effect on broader operations, remediation timeline |
| Employee-facing emphasis | What customer-facing staff should say, escalation path for customer questions, internal timeline |
| Regulator-facing emphasis | Full technical detail on the defect, root cause investigation status, corrective action plan |

**Rule applied:** every version states the same 1,360-unit, 12-injury, no-death facts — only emphasis and level of technical detail differ by audience.

## 3. Post-incident review template (filled example)

> **What happened:** Journalist inquiry with a 6-hour deadline; internal fact-finding was 3 weeks in progress but not yet publicly disclosed.
> **What worked:** Substantiated facts were ready enough to support an honest, specific statement within the deadline rather than a generic holding statement.
> **What didn't work:** No standing crisis communication plan existed for product-safety issues specifically — the approval chain (legal, product safety, CEO's office) was assembled ad hoc under time pressure, adding roughly 90 minutes of avoidable delay.
> **Update to standing plan:** Add a pre-approved escalation tree specifically for product-safety issues, with named approvers, so the next comparable situation doesn't lose time assembling sign-off under deadline pressure.

## 4. Statement red-team checklist (filled example)

| Check | Question | This statement |
|---|---|---|
| Front-page test | Does this survive an unfavorable front-page framing? | Yes — states the number affected and injury count directly rather than minimizing |
| Adversarial read | What's the most damaging alternative interpretation? | Reviewed: could be read as "took 3 weeks to disclose" — addressed by explicitly stating the investigation timeline and what triggered disclosure now |
| Commitment check | Are any promised fixes/timelines confirmed achievable internally? | Yes — recall timeline confirmed with operations before publication |
| Cross-audience consistency | Do all audience versions state the same core facts? | Yes — verified against message house |
