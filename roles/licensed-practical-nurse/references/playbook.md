# Playbook — early-warning check, scope crosswalk, and escalation

Filled processes, not descriptions. Use the Stop & Watch table to trigger a check, the scope crosswalk to decide who can act, and the SBAR-NH templates to escalate.

## 1. Stop & Watch Early Warning Tool

Run this whenever a CNA, family member, or your own pass flags "something's different." Any **single** checked item is sufficient to notify the RN supervisor or physician same-shift — don't wait for more than one.

| Letter | Sign | This shift? |
|---|---|---|
| S | Seems different than usual | |
| T | Talks or communicates less | |
| O | Overall needs more help with daily activities | |
| P | Participates less in activities | |
| A | Ate less at last two meals | |
| N | No bowel movement in 3+ days, or diarrhea | |
| D | Drank less fluid than usual | |
| W | Weight change (up or down) | |
| A | Agitated or nervous more than usual | |
| T | Tired, weak, confused, or drowsy | |
| C | Change in skin color or condition | |
| H | Help with walking, transferring, or toileting more than usual | |

**Rule that gets missed:** this is a same-shift notification tool, not a wait-and-see checklist — one checked box, notify now, and document which box(es) triggered the call.

## 2. Scope crosswalk — state task list vs. facility policy

Before doing (or delegating) a task, run it through both columns. If either is "No" or "Silent," the task is out of scope right now.

| Task | State LPN/LVN task list | Facility delegation policy | Action |
|---|---|---|---|
| Peripheral IV insertion | Yes (many states) | Yes | Proceed if both authorize |
| IV push medication | Varies — check state specifically | Often restricted to RN | Escalate to RN unless both explicitly say yes |
| Blood product initiation | No in most states | N/A | Always escalate to RN |
| Foley catheter insertion | Yes (most states) | Yes | Proceed if both authorize |
| Clean-catch UA / specimen collection | Yes | Yes | Proceed |
| Telephone/verbal order intake | Yes, with read-back | Yes, with cosignature process | Proceed, complete read-back and cosign steps |
| Comprehensive/admission assessment | No (RN-reserved) | N/A | Escalate — RN completes, LPN contributes focused data |
| Central line dressing change | Varies by state and specialty certification | Often facility-specific certification required | Verify both before proceeding |

**Rule that gets missed:** a state permitting a task doesn't override a stricter facility policy, and a permissive facility policy doesn't override a state that restricts the task — the narrower of the two always governs.

## 3. SBAR-NH escalation templates by urgency

**Routine notification (no action needed this call):**
> "[On-call RN/MD name], this is [LPN] at [facility], calling about [resident] in room [#]. Background: [one line]. [Finding], which I wanted to flag for the morning — no action needed tonight, just wanted you aware."

**Urgent (needs a decision this call):**
> "[Name], this is [LPN] at [facility], calling about [resident] in room [#]. He's/she's had a change in the last [X] hours and I need direction now. Background: [one line — admission reason, relevant history]. Assessment: at [baseline time] he/she was [baseline vitals]; right now he/she is [current vitals and trend]. My concern is [named possibility]. Recommendation: I'd like [specific ask — order, med, recheck interval] — can you give me parameters for when to call back versus transfer?"

**Emergent (transfer or 911 imminent):**
> "[Name], this is [LPN] at [facility], [resident] in room [#] — he's/she's acutely worse in the last [X] minutes. [Vitals/trend]. I'm concerned about [named risk] and I think this needs to go to the ED now unless you want to try [specific intervention] first. Which do you want?"

Always end with a specific ask and the exact time of the call; document the read-back for any resulting verbal order before moving to the next task.

## 4. Delegation checklist — applied upward and downward

**Upward (accepting a task delegated to you by an RN):**
1. Is this task on my state's authorized list and my facility's policy?
2. Is this resident's circumstance stable enough that the task doesn't require RN-level judgment mid-task?
3. Do I personally have current training/competency for this specific task?
4. Did the RN give me a specific abnormal-value threshold to call back on, not "let me know if anything changes"?
5. Am I clear on who re-evaluates the outcome — me, or does it go back to the RN?

**Downward (delegating a task to a CNA/UAP):**
1. Right task — routine data collection (vitals, ADLs, intake/output), not assessment or interpretation.
2. Right circumstance — is this specific resident stable enough tonight, or did they just trigger Stop & Watch?
3. Right person — does this CNA have current certification for the specific task (e.g., glucometer)?
4. Right direction — did you give a specific numeric/behavioral callback threshold?
5. Right supervision — will you personally verify the result before acting on it, especially on a resident with an active trigger?

## 5. Post-event documentation sequence

After any Stop & Watch trigger, escalation, or transfer: (1) the timestamped vital/status trend from the last documented baseline through the event, (2) the SBAR call time, who was called, and the order received with read-back confirmation, (3) interventions given with times, (4) the follow-up recheck and its result against the physician's stated parameters, (5) responsible-party notification time, (6) handoff note for the oncoming shift on what to keep watching. This sequence — not a narrative paragraph — is what a supervising RN, medical director, or state surveyor reads first.
